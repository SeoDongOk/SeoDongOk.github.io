`〜んです`는 일본어 회화에서 매우 자주 나오지만, 번역하기 어려운 표현 중 하나다. 문법적으로는 `〜のです`가 정식이고 `〜んです`는 그 회화체다.

핵심 기능은 **설명·강조·이유 묻기**다.

---

## 기본 구조

```
동사·형용사 보통형 + んです
명사 / な형용사: 〜なんです (だ → な)
```

| 형태     | 例                            |
| -------- | ----------------------------- |
| 동사     | 行くんです / 食べたんです     |
| い형용사 | 高いんです / 難しかったんです |
| な형용사 | 好きなんです / 元気なんです   |
| 명사     | 学生なんです / 日本人なんです |

---

## 주요 용법

### 1. 설명·이유 제시

단순 사실 진술이 아니라, **배경·이유를 설명**할 때.

- A: どうしたんですか。 (어떻게 됐어요? / 왜 그래요?)  
  B: 頭が痛い**んです**。 (두통이 있어요. — 그래서 이상하게 보이는 거예요)

- 明日、試験がある**んです**。だから、勉強しています。  
  (내일 시험이 있거든요. 그래서 공부하고 있어요.)

> 단순히 "頭が痛いです"라고 하면 사실 전달, "頭が痛い**んです**"라고 하면 이유·배경 설명.

### 2. 이유 묻기: 〜んですか

상대방의 행동·상태에 대해 이유나 배경을 물을 때.

- どうして来なかった**んですか**。 (왜 안 왔어요?)
- 何を探している**んですか**。 (뭘 찾고 있는 거예요?)
- どこに行く**んですか**。 (어디 가는 거예요?)

> 〜んですか는 보통 "〜ですか"보다 더 관심·파악하려는 뉘앙스.

### 3. 강조·공감

자신의 감정이나 상황을 강조할 때.

- この映画、本当によかった**んです**！ (이 영화 정말 좋았어요!)
- 実は、日本語が好きな**んです**。 (사실은 일본어를 좋아하거든요.)

### 4. 권유·확인 + 이유 설명

- 一緒に行きましょう。楽しい**んです**よ。  
  (같이 가요. 재미있거든요.)
- ちょっと待ってください。急い**でるんです**。  
  (잠깐만요. 급하거든요.)

---

## 〜んです vs 〜です 비교

|        | 〜です       | 〜んです       |
| ------ | ------------ | -------------- |
| 기능   | 사실 전달    | 설명·배경·강조 |
| 뉘앙스 | 중립         | 이유·맥락 있음 |
| 예     | 頭が痛いです | 頭が痛いんです |

> **회화 팁:** 〜んです가 너무 많으면 어색하고, 너무 없으면 딱딱하다. 설명이 필요한 상황에서만 쓰자.

---

## 〜んですが — 전치 표현

용건·부탁을 꺼내기 전에 상황 설명을 먼저 할 때.

- 実は、お願いがある**んですが**...。  
  (사실 부탁이 있는데요...)
- 道に迷ってしまった**んですが**、〜はどこですか。  
  (길을 잃었는데요, 〜는 어디예요?)
- 明日、休みたい**んですが**、よろしいですか。  
  (내일 쉬고 싶은데요, 괜찮으세요?)

---

## 〜んですか로 이유 묻는 패턴

| 상황   | 표현                 |
| ------ | -------------------- |
| 왜     | どうして〜んですか   |
| 무엇   | 何を〜んですか       |
| 언제   | いつ〜んですか       |
| 어디   | どこへ/に〜んですか  |
| 어떻게 | どうやって〜んですか |

---

## 핵심 요약

1. **설명·이유:** 이게 핵심. "〜이기 때문에", 배경 있는 발화.
2. **이유 묻기:** 〜んですか = "왜 〜하는 거예요?"
3. **접속:** な형용사·명사 → 〜**な**んです (だ 대신 な)
4. **회화:** 〜んです → 설명, 〜ですか → 단순 질문
