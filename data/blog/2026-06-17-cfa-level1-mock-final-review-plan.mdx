> CFA Level 1 준비의 후반부는 “얼마나 더 읽을까”가 아니라 **실전에서 점수를 어떻게 안정화할까**의 문제다. 마지막 8주는 Mock Exam과 오답 제거가 중심이 된다.

## 마지막 8주의 목표

마지막 8주의 목표는 세 가지다.

1. 시간 안에 180문항을 끝내는 체력 만들기
2. 과목별 약점 확인과 보완
3. 시험 당일 실수를 줄이는 루틴 고정

새로운 자료를 계속 추가하는 시기가 아니다. 이미 본 내용과 틀린 문제를 압축해서 점수로 바꾸는 시기다.

## Mock Exam 일정

가정: 2027년 2월 시험 응시.

| 시기        | 목표                       |
| ----------- | -------------------------- |
| 시험 8주 전 | Half Mock 또는 과목별 진단 |
| 시험 6주 전 | Full Mock 1회              |
| 시험 4주 전 | Full Mock 2회              |
| 시험 2주 전 | 약점 과목 중심 Mini Mock   |
| 시험 1주 전 | 오답 재풀이와 Ethics 유지  |

Full Mock은 가능한 실제 시험처럼 본다.

```text
세션 1: 135분
짧은 휴식
세션 2: 135분
당일 채점
다음날 오답 분석
```

채점과 분석을 같은 날 모두 끝내려 하면 지칠 수 있다. Mock 당일에는 채점과 큰 약점만 보고, 세부 오답 분석은 다음날 해도 된다.

## Mock 분석표

Mock을 본 뒤에는 점수만 적지 않는다. 과목별로 다음을 기록한다.

| 과목                 | 정답률 | 오답 수 | 오답 원인 | 다음 조치 |
| -------------------- | -----: | ------: | --------- | --------- |
| Ethics               |        |         |           |           |
| Quant                |        |         |           |           |
| Economics            |        |         |           |           |
| FSA                  |        |         |           |           |
| Corporate Issuers    |        |         |           |           |
| Equity               |        |         |           |           |
| Fixed Income         |        |         |           |           |
| Derivatives          |        |         |           |           |
| Alternatives         |        |         |           |           |
| Portfolio Management |        |         |           |           |

중요한 것은 전체 점수보다 **반복해서 틀리는 과목과 유형**이다.

## 오답 원인 분류

오답은 다음 네 가지로 분류한다.

| 유형      | 설명                           | 대응             |
| --------- | ------------------------------ | ---------------- |
| 개념 부족 | 개념 자체를 모름               | 해당 모듈 재학습 |
| 적용 실패 | 개념은 아는데 문제에 적용 못함 | 유사 문제 반복   |
| 계산 실수 | 공식 또는 계산기 입력 실수     | 풀이 루틴 고정   |
| 문장 오독 | 질문 조건을 잘못 읽음          | 키워드 표시 훈련 |

이 분류를 하지 않으면 같은 시간을 써도 개선 속도가 느리다.

## 마지막 8주 공부 비율

| 활동             | 비중 |
| ---------------- | ---: |
| Mock 및 QBank    |  45% |
| 오답 분석        |  30% |
| 약점 개념 재학습 |  15% |
| Ethics 유지      |  10% |

후반에는 “읽는 공부”보다 “틀린 것을 다시 맞히는 공부”가 우선이다.

## 시험 직전 1주 계획

시험 직전 1주는 새로운 어려운 내용을 넣지 않는다.

- Ethics 매일 20~30분
- 공식 카드 반복
- FSA, Fixed Income 핵심 오답 재풀이
- 계산기 조작 점검
- 여권과 시험 예약 정보 확인
- 수면 시간 고정

특히 시험 전날에는 불안해서 새로운 Mock을 보면 안 된다. 틀린 문제가 나오면 멘탈만 흔들릴 수 있다.

## 시험 당일 전략

시험 당일에는 세 가지를 지킨다.

1. 모르는 문제에 오래 붙잡히지 않는다.
2. 모든 문제에 답을 표시한다.
3. 계산 문제에서 단위를 확인한다.

오답 감점이 없기 때문에 빈칸은 남기지 않는다. 확신이 없더라도 가장 가능성 높은 선택지를 고르고 넘어간다.

## 합격권을 위한 개인 기준

공식 합격선은 공개된 고정 점수가 아니므로, 개인적으로는 Mock에서 다음 기준을 목표로 잡는다.

| 기준              |                       목표 |
| ----------------- | -------------------------: |
| 전체 Mock 정답률  | 70% 이상을 안정적으로 목표 |
| Ethics            |              75% 이상 목표 |
| FSA, Fixed Income |              70% 이상 목표 |
| 최저 과목         |            60% 미만 없애기 |

이 숫자는 내 공부 관리를 위한 내부 기준이다. 실제 시험 결과를 보장하는 기준은 아니다.

## 결론

마지막 8주는 공부량을 늘리는 시기가 아니라, 점수 변동성을 줄이는 시기다. Mock을 통해 약점을 드러내고, 오답을 유형별로 제거하고, 시험 당일 루틴을 고정한다.

2027년 2월 시험 전까지 이 글을 기준으로 Mock 기록을 계속 업데이트할 예정이다.
