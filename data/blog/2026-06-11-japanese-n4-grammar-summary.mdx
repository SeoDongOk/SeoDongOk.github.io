## N4 문법 카테고리별 총정리

---

### 1. 가정형 4가지

| 형태   | 핵심 용도      | 예                   |
| ------ | -------------- | -------------------- |
| 〜ば   | 조건·법칙      | 春になれば暖かい     |
| 〜たら | 순서·완료 후   | 帰ったら電話して     |
| 〜と   | 자연 결과·기계 | 右に曲がると駅がある |
| 〜なら | 상대방 말 전제 | 日本に行くなら京都へ |

**시험 포인트:**

- と + 의지·명령 ❌
- ば + 의지·명령 ❌ (후회 문형 제외)
- たら + 의지·명령 ⭕
- なら + 의지 ⭕

---

### 2. 수동형 〜られる

| 그룹  | 만들기      | 예                |
| ----- | ----------- | ----------------- |
| 2그룹 | る → られる | 食べる→食べられる |
| 1그룹 | あ단 + れる | 書く→書かれる     |
| 3그룹 | 불규칙      | する→される       |

- 직접 수동: 先生に褒められた
- 피해 수동: 雨に降られた
- 물건 주어: 〜年前に建てられた

---

### 3. 사역형 〜させる

| 그룹  | 만들기      | 예                |
| ----- | ----------- | ----------------- |
| 2그룹 | る → させる | 食べる→食べさせる |
| 1그룹 | あ단 + せる | 書く→書かせる     |
| 3그룹 | 불규칙      | する→させる       |

- 허가: 〜させてください
- 강제: 〜させた
- 사역수동: 〜させられる (억지로 시켜졌다)

---

### 4. 수수동사

| 동사   | 방향         | 공손형     |
| ------ | ------------ | ---------- |
| あげる | 나→남        | さしあげる |
| もらう | 남→나 (받다) | いただく   |
| くれる | 남→나        | くださる   |

- 〜てあげる / 〜てもらう / 〜てくれる

---

### 5. 추량·전문 표현

| 표현           | 접속   | 뉘앙스           |
| -------------- | ------ | ---------------- |
| 〜らしい       | 보통형 | 간접 정보 추측   |
| 〜そうだ(전문) | 보통형 | 정보 그대로 전달 |
| 〜そうだ(양태) | 어간   | 직접 보고 느낌   |
| 〜ようだ       | 보통형 | 근거 있는 추측   |
| 〜みたいだ     | 보통형 | ようだ의 회화체  |

**시험 포인트:** 양태そうだ만 어간에 접속!

---

### 6. 목적·변화 표현

| 표현         | 의미                 |
| ------------ | -------------------- |
| 〜ために     | 〜을 위해 (목적)     |
| 〜ように     | 〜하도록             |
| 〜ようになる | 〜하게 됐다 (변화)   |
| 〜ようにする | 〜하도록 하다 (노력) |

---

### 7. 복합동사

| 표현     | 의미                |
| -------- | ------------------- |
| 〜始める | 〜하기 시작하다     |
| 〜続ける | 계속 〜하다         |
| 〜終わる | 〜가 끝나다/다 하다 |
| 〜出す   | 갑자기 시작하다     |
| 〜やすい | 하기 쉽다           |
| 〜にくい | 하기 어렵다         |
| 〜すぎる | 너무 〜하다         |

---

### 8. 기타 N4 핵심 문법

| 표현         | 의미           | 예             |
| ------------ | -------------- | -------------- |
| 〜はずだ     | 당연히 〜일 것 | 来るはずだ     |
| 〜はずがない | 〜일 리 없다   | 来るはずがない |
| 〜んです     | 설명·강조      | 頭が痛いんです |
| 〜のに       | 역접·불만      | 勉強したのに   |
| 〜ながら     | 〜하면서       | 歩きながら     |
| 〜てしまう   | 해버리다       | 食べてしまった |
| 〜ておく     | 미리 해두다    | 準備しておく   |
| 〜てみる     | 해보다         | やってみる     |
| 〜てある     | 〜해져 있다    | 書いてある     |
| 〜てほしい   | 〜해줬으면     | 来てほしい     |

---

### 9. 접속사·부사

| 표현     | 의미               |
| -------- | ------------------ |
| それで   | 그래서             |
| だから   | 그러니까           |
| でも     | 그래도/그런데      |
| しかし   | 그러나 (격식)      |
| それに   | 게다가             |
| ところで | 그런데 (화제 전환) |
| そのうえ | 게다가 (추가)      |

---

### 10. N4 시험 자주 나오는 패턴

**빈칸 채우기 함정:**

| 함정             | 정답                      |
| ---------------- | ------------------------- |
| 양태そうだ 접속  | 어간 + そう (보통형 X)    |
| ために vs ように | 의지 vs 목표              |
| あげる vs くれる | 방향 확인                 |
| ば vs たら       | 뒤에 명령형이면 たら      |
| はずだ vs だろう | 논리적 근거 있으면 はずだ |

---

## 시험 전날 체크리스트

- [ ] 가정형 4가지 (ば/たら/と/なら) 구분 가능?
- [ ] 수동형 만들기 빠르게 가능?
- [ ] 사역형 + 사역수동 구분?
- [ ] あげる/もらう/くれる 방향 확실?
- [ ] そうだ 전문/양태 접속 차이?
- [ ] ために vs ように 구분?
- [ ] ようになる vs ようにする 차이?
- [ ] はずだ vs はずがない 상황 구분?
- [ ] んです 언제 쓰는지?
- [ ] 복합동사 7가지?

N4 합격 기준은 **60%** 이상. 문법 문제에서 확실히 점수 따고, 독해·청해로 버티는 전략이 유효하다.
