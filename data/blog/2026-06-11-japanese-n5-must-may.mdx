"해야 해요", "해도 돼요", "하면 안 돼요", "안 해도 돼요" — 이 네 가지 표현은 규칙, 부탁, 허가를 말할 때 매우 자주 쓴다.

모두 て형과 ない형에서 파생되므로 활용 규칙을 알면 바로 응용할 수 있다.

---

## 1. 〜なければならない — 해야 한다 (의무)

```
동사 ない형 의 ない → なければならない
```

| 동사   | ない형   | 의무형               |
| ------ | -------- | -------------------- |
| 行く   | 行かない | 行かなければならない |
| 食べる | 食べない | 食べなければならない |
| する   | しない   | しなければならない   |
| 来る   | こない   | こなければならない   |

**예문:**

- 明日早く起き**なければなりません**。 (내일 일찍 일어나야 합니다.)
- 薬（くすり ）を毎日飲ま**なければなりません**。 (약을 매일 먹어야 합니다.)
- 宿題をやら**なければなりません**。 (숙제를 해야 합니다.)

### 회화체 줄임말

격식 없는 일상 대화에서는 자주 줄여 쓴다.

| 정중 표현            | 회화체 줄임      |
| -------------------- | ---------------- |
| 〜なければならない   | 〜なきゃ         |
| 〜なければなりません | 〜なきゃいけない |

- 行か**なきゃ**！ (가야 해!)
- 勉強し**なきゃいけない**。 (공부해야 해.)

---

## 2. 〜てもいい — 해도 된다 (허가)

```
동사 て형 + もいい
```

- ここで写真を撮っ**てもいいですか**。 (여기서 사진을 찍어도 돼요?)
- 窓を開け**てもいいですか**。 (창문을 열어도 돼요?)
- もう帰っ**てもいいですよ**。 (이제 가도 돼요.)
- このペンを使っ**てもいい**ですか。 (이 펜 써도 돼요?)

**대답:**

- はい、どうぞ。 (네, 물론이죠.)
- はい、もちろん。 (네, 당연하죠.)
- すみません、ちょっと... (죄송한데, 좀...)

---

## 3. 〜てはいけない — 하면 안 된다 (금지)

```
동사 て형 + はいけない
```

- ここでタバコを吸っ**てはいけません**。 (여기서 담배를 피우면 안 됩니다.)
- 図書館で大きい声で話し**てはいけません**。  
  (도서관에서 큰 소리로 말하면 안 됩니다.)
- 他人（たにん）のものを勝手（かって）に触っ**てはいけません**。  
  (남의 것을 함부로 만지면 안 됩니다.)

### 금지 표현 공손도 비교

| 표현             | 공손도 | 상황        |
| ---------------- | ------ | ----------- |
| 〜てはいけません | 높음   | 공식, 안내  |
| 〜てはだめです   | 중간   | 일상        |
| 〜ないでください | 높음   | 정중한 요청 |
| 〜ちゃだめ       | 낮음   | 친구 사이   |

---

## 4. 〜なくてもいい — 안 해도 된다 (불필요)

```
동사 ない형 의 ない → なくてもいい
```

- 今日は来**なくてもいいです**よ。 (오늘은 안 와도 돼요.)
- 全部食べ**なくてもいいです**。 (다 먹지 않아도 돼요.)
- 急が**なくてもいい**ですよ。 (서두르지 않아도 돼요.)

---

## 4가지 표현 비교표

| 표현               | 의미         | 형태                      |
| ------------------ | ------------ | ------------------------- |
| 〜なければならない | 해야 한다    | ない형 + なければならない |
| 〜てもいい         | 해도 된다    | て형 + もいい             |
| 〜てはいけない     | 하면 안 된다 | て형 + はいけない         |
| 〜なくてもいい     | 안 해도 된다 | ない형 + なくてもいい     |

---

## 실전 대화 예

**선생님과 학생 상황:**

> 先生: 教科書（きょうかしょ）を持って来**なければなりません**。  
> (선생님: 교과서를 가져와야 합니다.)
>
> 学生: ノートも持って来**なければなりませんか**。  
> (학생: 노트도 가져와야 합니까?)
>
> 先生: ノートは持って来**なくてもいいです**。  
> (선생님: 노트는 가져오지 않아도 됩니다.)
>
> 学生: 辞書（じしょ）を使っ**てもいいですか**。  
> (학생: 사전을 사용해도 됩니까?)
>
> 先生: 試験中は使っ**てはいけません**。  
> (선생님: 시험 중에는 사용하면 안 됩니다.)
