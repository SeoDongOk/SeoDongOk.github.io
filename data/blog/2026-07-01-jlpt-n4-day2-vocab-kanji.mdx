JLPT N4 어휘·한자 파트는 문항 수가 가장 많지만 한 문제당 배점은 낮다.
그래서 모르는 문제를 붙잡기보다, 자주 나오는 것을 빠르게 정리해 두는 편이 효율적이다.

이 글은 시험 4일 전 2일차 공부용으로, 어휘·한자 빈출 내용을 모았다.
(한자 읽기에서 자주 틀리는 함정은 별도 글에서 자세히 다뤘으니 함께 보면 좋다.)

## 어휘·한자 문제 유형 5가지

| 유형      | 무엇을 묻나          | 공략 팁                   |
| --------- | -------------------- | ------------------------- |
| 한자 읽기 | 한자 → 히라가나 읽기 | 음독/훈독, 탁음·장음 주의 |
| 표기      | 히라가나 → 맞는 한자 | 모양 비슷한 한자 함정     |
| 빈칸 어휘 | 문맥에 맞는 단어     | 앞뒤 문맥으로 소거        |
| 유의어    | 비슷한 뜻 단어       | 세트로 암기               |
| 단어 쓰임 | 올바른 용법 문장     | 예문째로 외우기           |

## 빈출 유의어 세트

유의어 문제는 짝으로 외워 두면 시험장에서 바로 떠오른다.

- 大切（たいせつ）= 大事（だいじ） : 소중하다
- だんだん = 少（すこ）しずつ : 조금씩
- ときどき = たまに : 가끔
- ぜんぶ = みんな : 전부
- もうすぐ = まもなく : 곧
- ちょうど = きっかり : 딱, 정확히

## 표기 함정: 모양이 비슷한 한자

표기 문제는 비슷하게 생긴 한자를 보기에 섞어 놓는다.

- 待（기다리다, ま-つ） / 持（들다, も-つ）
- 体（몸, からだ） / 休（쉬다, やす-む）
- 大（크다） / 太（굵다, ふと-い）
- 売（팔다, う-る） / 買（사다, か-う）

이런 한자는 모양만 보지 말고 뜻과 함께 묶어서 구분해 두자.

## 자주 나오는 부사

빈칸 어휘 문제에 부사가 자주 나온다.

- きっと : 분명히, 틀림없이
- もし : 만약
- ぜひ : 꼭, 부디
- やっと : 드디어, 겨우
- なかなか : 좀처럼 (+ 부정) / 꽤
- ずっと : 쭉, 계속

## 자주 나오는 가타카나(외래어)

가타카나 단어도 빈출이니 읽기를 정리해 두자.

- アルバイト : 아르바이트
- パソコン : 컴퓨터(PC)
- エアコン : 에어컨
- スーパー : 슈퍼마켓
- レストラン : 레스토랑
- アパート : 아파트(연립주택)

## 자동사·타동사 짝

N4에서는 자동사/타동사 구분도 중요하다. 짝으로 외우면 좋다.

| 자동사         | 타동사         | 뜻                  |
| -------------- | -------------- | ------------------- |
| 開（あ）く     | 開（あ）ける   | 열리다 / 열다       |
| 閉（し）まる   | 閉（し）める   | 닫히다 / 닫다       |
| 始（はじ）まる | 始（はじ）める | 시작되다 / 시작하다 |
| 入（はい）る   | 入（い）れる   | 들어가다 / 넣다     |

## 결론

어휘·한자는 양이 많지만 한 문제 배점은 낮다.
유의어 세트, 표기 함정, 빈출 부사·가타카나를 빠르게 훑고,
모르는 문제는 시험장에서 과감히 찍고 넘기는 전략이 점수에 유리하다.
