> Time Value of Money는 Quant에서 끝나는 개념이 아니다. Fixed Income, Equity, Corporate Issuers까지 이어지는 CFA의 핵심 언어다.

## 공식 범위 확인

2027 Quant의 Time Value of Money 모듈은 기대 미래 현금흐름을 기반으로 fixed-income과 equity instrument의 현재가치를 계산·해석하는 내용을 포함한다.

## 돈의 시간가치

오늘의 100만 원과 1년 뒤의 100만 원은 같지 않다. 오늘의 돈은 투자되어 이자를 만들 수 있기 때문이다. 이 차이를 돈의 시간가치라고 한다.

핵심 변수는 다섯 가지다.

| 변수 | 의미               |
| ---- | ------------------ |
| PV   | 현재가치           |
| FV   | 미래가치           |
| r    | 할인율 또는 이자율 |
| N    | 기간 수            |
| PMT  | 기간별 현금흐름    |

## 미래가치

현재 금액을 미래로 굴리면 미래가치가 된다.

```text
FV = PV × (1 + r)^N
```

예를 들어 100을 연 5%로 3년 투자하면:

```text
100 × 1.05^3 = 115.76
```

## 현재가치

미래 금액을 현재로 할인하면 현재가치가 된다.

```text
PV = FV / (1 + r)^N
```

3년 뒤 115.76을 할인율 5%로 현재가치화하면:

```text
115.76 / 1.05^3 = 100
```

현재가치와 미래가치는 같은 관계를 반대 방향으로 보는 것이다.

## 할인율의 의미

할인율은 단순한 계산 입력값이 아니다. 요구수익률, 기회비용, 위험 보상을 반영한다.

할인율이 높아질수록 현재가치는 낮아진다.

```text
할인율 상승 → 미래 현금흐름의 현재가치 하락
할인율 하락 → 미래 현금흐름의 현재가치 상승
```

이 직관은 채권 가격과 수익률 관계를 이해할 때 매우 중요하다.

## 순현재가치

여러 현금흐름이 있을 때는 각 현금흐름을 할인해서 더한다.

```text
NPV = CF0 + CF1/(1+r) + CF2/(1+r)^2 + ... + CFn/(1+r)^n
```

투자안 평가에서는 NPV가 양수이면 요구수익률을 초과하는 가치를 만든다고 해석할 수 있다.

## 계산기 습관

CFA Level I에서는 승인 계산기를 사용한다. TVM 문제는 계산기 입력 습관이 중요하다.

- 현금 유출은 음수, 유입은 양수로 입력한다.
- 기간 수 `N`과 이자율 `I/Y`의 단위를 맞춘다.
- 월복리면 연이율을 월이율로 바꾸거나 복리 횟수를 반영한다.
- 문제를 풀고 나면 부호가 경제적으로 맞는지 확인한다.

## 오답 포인트

| 오답 포인트               | 체크 방법                                 |
| ------------------------- | ----------------------------------------- |
| 할인과 복리를 반대로 적용 | 현재로 오면 나누고, 미래로 가면 곱한다    |
| 기간과 이자율 단위 불일치 | 월/분기/년 단위 확인                      |
| 부호 입력 실수            | 유출과 유입을 반대로 넣지 않기            |
| 할인율 의미 무시          | 위험이 높으면 요구수익률도 높아질 수 있음 |

## 체크리스트

- [ ] 할인율 상승이 현재가치를 낮추는 이유를 설명할 수 있다.
- [ ] 여러 현금흐름의 현재가치를 더해 NPV를 계산할 수 있다.
- [ ] TVM 계산기 입력 시 부호와 기간 단위를 확인한다.
