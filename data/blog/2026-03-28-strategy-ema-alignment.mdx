## 전략 개요

**EMA Alignment** 전략은 3개의 EMA(단기/중기/장기)의 배열 상태가 바뀌는 순간을
추세 전환 시그널로 사용하는 전략이다.

주식, 선물, 코인 트레이더들 사이에서 오랫동안 사용해온 기법으로,
"정배열 = 상승 추세", "역배열 = 하락 추세"라는 단순하지만 강력한 원칙에 기반한다.

---

## 정배열 vs 역배열

```
정배열 (Bullish Alignment):
  EMA(8) > EMA(21) > EMA(50)
  단기선이 중기선 위, 중기선이 장기선 위
  → 가격이 모든 평균보다 빠르게 오르고 있음 = 강한 상승 추세

역배열 (Bearish Alignment):
  EMA(8) < EMA(21) < EMA(50)
  단기선이 모두 아래
  → 강한 하락 추세
```

이 전략은 단순히 "정배열이면 산다"가 아니라,
**역배열 → 정배열로 전환되는 첫 순간**에 진입한다.
추세가 완전히 굳어진 뒤가 아니라, 전환점에서 타는 것이 핵심이다.

---

## 진입 로직

### LONG 조건

```python
# 1. 현재 봉: 정배열 완성
bullish_align = ema_fast > ema_mid > ema_slow

# 2. 직전 봉: 역배열 상태였음
prev_bearish = ema_fast_prev < ema_slow_prev

# 3. 거래량 급증 (가짜 전환 필터)
vol_ok = volume >= volume_ma(20) * 1.2
```

세 조건 모두 충족 시 LONG 진입.

### SHORT 조건

```python
# 1. 현재 봉: 역배열 완성
bearish_align = ema_fast < ema_mid < ema_slow

# 2. 직전 봉: 정배열 상태였음
prev_bullish = ema_fast_prev > ema_slow_prev

# 3. 거래량 급증
vol_ok = volume >= volume_ma(20) * 1.2
```

---

## 거래량 필터가 중요한 이유

EMA 전환은 **whipsaw(가짜 전환)** 가 자주 발생한다.
횡보 구간에서 EMA들이 수렴하면 정배열/역배열이 빠르게 왔다갔다 한다.

거래량이 평균의 1.2배 이상일 때만 진입하면:

- 기관/세력의 실제 방향성 있는 매매 확인
- 단순 노이즈에 의한 전환 차단
- 신호 빈도는 줄지만 신뢰도는 올라감

---

## 신호 강도 계산

```python
# EMA fast와 slow 간격을 slow 대비 비율로 강도 계산
gap_pct = abs(ema_fast - ema_slow) / ema_slow * 100
strength = min(gap_pct / strength_scale, 1.0)  # strength_scale=0.5
```

배열이 많이 벌어질수록 추세가 강하다는 의미 → 포지션 크기에 반영.

## 파라미터 설명

| 파라미터         | 기본값 | 설명                                             |
| ---------------- | ------ | ------------------------------------------------ |
| `ema_fast`       | 8      | 단기 EMA 기간. 작을수록 민감하게 반응            |
| `ema_mid`        | 21     | 중기 EMA 기간                                    |
| `ema_slow`       | 50     | 장기 EMA 기간. 추세 방향 기준선                  |
| `volume_period`  | 20     | 거래량 이동평균 기간                             |
| `volume_mult`    | 1.2    | 거래량 필터 배수. 높일수록 신호 감소/신뢰도 증가 |
| `strength_scale` | 0.5    | 신호 강도 스케일. 낮출수록 강도 값이 커짐        |

### 파라미터 조정 가이드

```env
# 단기 민감 세팅 (빠른 반응, 신호 많음)
EMA_FAST=5, EMA_MID=13, EMA_SLOW=34

# 중기 안정 세팅 (기본값, 15m~1h 적합)
EMA_FAST=8, EMA_MID=21, EMA_SLOW=50

# 장기 보수 세팅 (신호 드물지만 신뢰도 높음)
EMA_FAST=20, EMA_MID=60, EMA_SLOW=120
```

## 백테스트 결과

### SOLUSDT 15m, 30x 레버리지

| 기간  | 수익률       | 거래수 | 승률  | MDD       | Sharpe    |
| ----- | ------------ | ------ | ----- | --------- | --------- |
| 90일  | **+57.82%**  | 26건   | 42.3% | **15.8%** | **0.273** |
| 365일 | **+147.33%** | 81건   | 37.0% | 54.5%     | **0.174** |

### 해석

- **90일 MDD 15.8%** — 9개 전략 중 압도적으로 낮음. 자본 보존력 최고
- **Sharpe 0.273** — 위험 한 단위당 수익이 가장 높음
- 하루 평균 0.2건의 신호 — 선택적 진입으로 피로감 없음
- 승률 37~42% — 낮은 승률이지만 손익비로 수익

## 실거래 적용 시 주의사항

1. **레버리지 조절**: MDD가 낮아 레버리지를 높여도 비교적 안전하지만, 급변동 구간에서는 배열 전환이 빠르게 일어날 수 있음
2. **횡보장 주의**: 횡보 구간에서 whipsaw가 반복될 수 있음 → `volume_mult` 1.5 이상으로 상향 권장
3. **적합 타임프레임**: 15m ~ 1h 권장. 1m은 노이즈 너무 많음
