일본어 첫 자기소개. 어디서 시작해야 할지 막막하다면 이 가이드를 따라가자.

기본 자기소개부터 취미, 특기, 한국인으로서 자주 하는 말까지 실전에서 바로 쓸 수 있게 정리했다.

---

## 기본 자기소개 구조

```
1. 이름
2. 출신 / 국적
3. 직업 / 학교
4. 취미 / 특기
5. 성격 / 한마디
6. 마무리 인사
```

---

## 1. 이름 말하기

- はじめまして。 (처음 뵙겠습니다.)
- 私は**[이름]**と申（もう）します。 (저는 〜라고 합니다.) — 정중
- **[이름]**と言います。 (〜라고 해요.) — 일반
- **[이름]**です。よろしくお願（ねが）いします。 (〜입니다. 잘 부탁드립니다.)

**이름 관련:**

- 名前は**キム·ドンオク**です。 (이름은 김동옥입니다.)
- ニックネームは**[닉네임]**です。 (닉네임은 〜입니다.)
- 名前の読み方は〜です。 (이름 읽는 법은 〜입니다.)

---

## 2. 출신 말하기

- **韓国（かんこく）**から来ました。 (한국에서 왔어요.)
- **ソウル**出身（しゅっしん）です。 (서울 출신이에요.)
- 国籍（こくせき）は**韓国**です。 (국적은 한국입니다.)
- 日本には去年来ました。 (일본에는 작년에 왔어요.)
- 日本語を勉強して2年になります。 (일본어를 공부한 지 2년이 됐어요.)

**자주 쓰이는 도시:**

- ソウル 서울 / 釜山（ぷさん）부산 / 大邱（てぐ）대구 / 仁川（いんちょん）인천

---

## 3. 직업·학교 말하기

- **会社員（かいしゃいん）**です。 (회사원이에요.)
- **学生**です。 (학생이에요.)
- **エンジニア**をしています。 (엔지니어를 하고 있어요.)
- **[회사명]**で働（はたら）いています。 (〜에서 일하고 있어요.)
- **[학교명]**に通（かよ）っています。 (〜에 다니고 있어요.)
- 専攻（せんこう）は**コンピューターサイエンス**です。 (전공은 컴퓨터공학이에요.)

**직업 어휘:**
| 한국어 | 일본어 |
|--------|--------|
| 회사원 | 会社員 |
| 학생 | 学生 |
| 선생님 | 先生 |
| 의사 | 医者（いしゃ） |
| 간호사 | 看護師（かんごし） |
| 프리랜서 | フリーランス |
| 개발자 | エンジニア |
| 디자이너 | デザイナー |

---

## 4. 취미·관심사 말하기

```
趣味（しゅみ）は [명사] です。
[동사 사전형] ことが好きです。
〜が好きです。
```

- 趣味は**映画鑑賞（えいがかんしょう）**です。 (취미는 영화 감상이에요.)
- **音楽を聴く**ことが好きです。 (음악 듣는 걸 좋아해요.)
- 最近（さいきん）**カフェ巡（めぐ）り**にはまっています。 (요즘 카페 투어에 빠져 있어요.)
- **旅行（りょこう）**が好きで、アジアを旅しています。  
  (여행을 좋아해서 아시아를 여행하고 있어요.)

**취미 어휘:**
| 취미 | 일본어 |
|------|--------|
| 독서 | 読書（どくしょ） |
| 요리 | 料理（りょうり） |
| 게임 | ゲーム |
| 여행 | 旅行（りょこう） |
| 운동 | 運動（うんどう） |
| 사진 | 写真（しゃしん） |
| 음악 감상 | 音楽鑑賞（おんがくかんしょう） |
| 드라마 시청 | ドラマを見ること |

---

## 5. 성격·자기 표현

- **明るい（あかるい）**性格（せいかく）だと思います。 (밝은 성격이라고 생각해요.)
- 少し**内向的（ないこうてき）**ですが、慣（な）れると話せます。  
  (조금 내성적이지만 익숙해지면 말할 수 있어요.)
- **人と話すのが好き**です。 (사람과 이야기하는 걸 좋아해요.)
- **のんびりした**性格です。 (느긋한 성격이에요.)

**성격 형용사:**
| 뜻 | 일본어 |
|----|--------|
| 밝다 | 明るい |
| 조용하다 | 静かな |
| 성실하다 | 真面目（まじめ）な |
| 친절하다 | 親切（しんせつ）な |
| 긍정적 | ポジティブな |
| 내성적 | 内向的（ないこうてき）な |

---

## 6. 한국인 학습자 자주 쓰는 문장

- 日本語はまだ勉強中（べんきょうちゅう）です。 (일본어는 아직 공부 중이에요.)
- 間違（まちが）えることもあると思いますが、よろしくお願いします。  
  (틀리는 것도 있을 것 같지만 잘 부탁드립니다.)
- 日本のアニメ・ドラマをきっかけに日本語を始めました。  
  (일본 애니·드라마를 계기로 일본어를 시작했어요.)
- 日本の文化（ぶんか）にとても興味（きょうみ）があります。  
  (일본 문화에 매우 관심이 있어요.)

---

## 마무리 인사

- よろしくお願いします。 (잘 부탁드립니다.)
- これからよろしくお願いします。 (앞으로 잘 부탁드립니다.)
- 何かわからないことがあったら、教えてください。  
  (모르는 게 있으면 알려주세요.)
- 仲良くしてください。 (잘 지내요/친하게 지내요.)

---

## 샘플 자기소개 (60초 분량)

> はじめまして。キム·ドンオクと申します。韓国のソウル出身で、現在（げんざい）エンジニアとして働いています。日本語は2年ほど勉強しています。趣味は旅行と映画鑑賞です。特に日本映画が大好きで、それが日本語を始めたきっかけでもあります。まだ話すのが上手じゃないですが、一生懸命（いっしょうけんめい）勉強しています。これからよろしくお願いします。

번역:

> 처음 뵙겠습니다. 김동옥이라고 합니다. 한국 서울 출신으로 현재 엔지니어로 일하고 있습니다. 일본어는 2년 정도 공부하고 있습니다. 취미는 여행과 영화 감상입니다. 특히 일본 영화를 매우 좋아하고, 그게 일본어를 시작한 계기이기도 합니다. 아직 말하는 게 서툴지만 열심히 공부하고 있습니다. 앞으로 잘 부탁드립니다.
