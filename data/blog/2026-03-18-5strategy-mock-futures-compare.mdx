## 오늘 만든 것

`sdo_auto_trader_crypto`에 **5개 전략 병렬 모의거래 비교 시스템**을 추가했다.

각 전략이 독립적인 `$100` 가상 계좌로 바이낸스 선물 실시간 데이터 위에서 돌고, 24시간 후 수익률·승률·MDD를 비교해 "어떤 전략이 이 시장에서 잘 먹히는가"를 확인한다.

## BNF 전략이 뭔가

일본의 전설적 개인투자자 **BNF (코테가와 타카시)**의 매매법을 암호화폐 선물에 맞게 재해석했다.

원본은 주식 시장에서 **25일 이동평균선 이격도 20% 이상** 벌어졌을 때 기술적 반등을 노리는 역추세 전략이다. 핵심 아이디어:

- 시장이 패닉으로 급락 → 이평선과 괴리 발생
- 거래량이 터진다 → 패닉 셀링 극값 근처
- 볼린저 밴드 하단 이탈 → 통계적 과매도
- **세 조건이 겹치는 순간 반등 베팅**

암호화폐 선물은 주식보다 변동성이 크므로 이격도 임계값을 `20% → 3%`로 조정했다.

```python
# LONG 진입 조건
if (disparity <= -3.0          # 25MA 대비 -3% 이하
        and price <= bb_lower  # 볼린저 하단 터치
        and vol_ratio >= 1.5): # 평균 거래량 1.5배 이상
    signal = BUY
```

## VWAP Reversion

**VWAP (거래량 가중 평균 가격)**은 기관 트레이더들이 많이 보는 기준선이다.

암호화폐는 24시간 거래되므로 "장 시작" 개념이 없다. 대신 **롤링 24h VWAP** (96개 15분봉)을 사용해서 현재가가 얼마나 VWAP에서 벗어났는지 측정한다.

```python
# 롤링 VWAP
typical_price = (high + low + close) / 3
vwap = (typical_price × volume).rolling(96).sum() / volume.rolling(96).sum()

# LONG: VWAP 하방 1.5% 이탈 + RSI < 35
deviation = (price - vwap) / vwap × 100
if deviation <= -1.5 and rsi < 35:
    signal = BUY
```

## Supertrend EMA

**Supertrend**는 ATR 기반 동적 지지/저항선이다. 가격이 선을 돌파하는 순간 추세 방향이 전환된다. EMA 20/50 크로스로 필터를 걸어 노이즈를 줄였다.

```
HL2 = (High + Low) / 2
Upper = HL2 + 3.0 × ATR(10)
Lower = HL2 - 3.0 × ATR(10)

LONG  조건: Supertrend 상향 + EMA20 > EMA50
SHORT 조건: Supertrend 하향 + EMA20 < EMA50
```

흥미로운 점: VWAP는 "너무 많이 떨어졌다 → 반등"을 보고 있고, Supertrend는 "추세가 하락이다 → 공매도"를 보고 있다. **같은 시장을 완전히 반대 방향으로 해석하고 있다.**
