`わけ`는 한자로 **訳**, 원래 "이유·사정·도리"라는 뜻이다. 이 한 단어가 뒤에 붙는 말에 따라 **납득 / 부분부정 / 강한 부정 / 사회적 제약** 네 가지로 완전히 다른 의미가 된다. N2 문법에서 가장 많이 나오고, 한국인이 가장 많이 헷갈리는 세트다. 핵심은 "わけ를 통째로 외우지 말고, 뒤에 뭐가 붙었는지로 구분"하는 것.

---

## ① 〜わけだ — 그래서 당연히 〜인 거구나 (납득)

앞의 상황을 근거로 **"어쩐지 그렇더라"하고 논리적으로 납득**할 때 쓴다. `道理（どうり）で`(어쩐지)와 자주 짝을 이룬다.

```
동사·い형용사 보통형 + わけだ
な형용사 + な + わけだ
명사 + という + わけだ
```

**예문:**

- 道理（どうり）で寒（さむ）い**わけだ**。窓（まど）が開（あ）いていた。  
  (어쩐지 춥더라. 창문이 열려 있었네.)
- 彼（かれ）は10年（ねん）も東京（とうきょう）に住（す）んでいたのか。日本語（にほんご）がぺらぺらな**わけだ**。  
  (그는 10년이나 도쿄에 살았구나. 그러니 일본어가 유창한 거구나.)
- 3割引（さんわりびき）か。安（やす）い**わけだ**。  
  (30% 할인이구나. 어쩐지 싸더라.)

> 💡 **포인트**: わけだ는 새로운 정보를 주는 게 아니라, 이미 본 결과에 대해 "원인을 알고 납득"하는 표현이다.

---

## ② 〜わけではない — 꼭 〜인 것은 아니다 (부분 부정)

전체를 부정하는 게 아니라 **"전부 그런 건 아니다"하고 일부만 부정**한다. 회화에서는 `わけじゃない`로 줄인다.

```
동사·い형용사 보통형 + わけではない
な형용사 + な / である + わけではない
명사 + という / な + わけではない
```

**예문:**

- 嫌（きら）いな**わけではない**が、あまり食（た）べない。  
  (싫어하는 건 아니지만, 별로 안 먹어.)
- お金（かね）が全（まった）くない**わけではない**。  
  (돈이 전혀 없는 건 아니야. — 조금은 있다)
- 毎日（まいにち）運動（うんどう）している**わけではない**。  
  (매일 운동하는 건 아니야. — 가끔은 한다)

> 💡 **포인트**: "100% 부정"이 아니라 "예외가 있다"는 뉘앙스. 뒤에 변명·보충이 자주 따라온다.

---

## ③ 〜わけがない — 〜일 리가 없다 (강한 부정·확신)

가능성을 **강하게 부정**한다. "절대 그럴 리 없다". 회화에서는 `わけない`로도 줄인다.

```
동사·い형용사 보통형 + わけがない
な형용사 + な + わけがない
명사 + の / である + わけがない
```

**예문:**

- そんな簡単（かんたん）な問題（もんだい）、彼（かれ）が解（と）けない**わけがない**。  
  (그런 쉬운 문제를 그가 못 풀 리가 없어.)
- 子供（こども）にこの漢字（かんじ）が読（よ）める**わけがない**。  
  (애가 이 한자를 읽을 수 있을 리가 없어.)
- 練習（れんしゅう）もせずに、試合（しあい）に勝（か）てる**わけがない**。  
  (연습도 안 하고 시합에 이길 리가 없어.)

> 💡 **〜はずがない와 비교**: 둘 다 "〜일 리 없다"지만, `はずがない`는 **논리적 근거** 중심, `わけがない`는 더 **주관적·감정적**인 단정이다.

---

## ④ 〜わけにはいかない — 〜할 수는 없다 (사회적·심리적 제약)

능력이 없어서가 아니라, **도덕·약속·상황 때문에 "그럴 수는 없다"**는 의미다. 능력 불가의 `できない`와 헷갈리면 안 된다.

```
동사 사전형 + わけにはいかない   → (하고 싶어도) ~할 수 없다
동사 ない형 + わけにはいかない   → ~안 할 수는 없다 = 해야만 한다
```

**예문:**

- 大事（だいじ）な会議（かいぎ）だから、休（やす）む**わけにはいかない**。  
  (중요한 회의라, 쉴 수는 없어.)
- 一度（いちど）約束（やくそく）したから、行（い）か**ないわけにはいかない**。  
  (한번 약속했으니, 안 갈 수는 없지. = 가야만 해.)
- 借（か）りたお金（かね）を返（かえ）さ**ないわけにはいかない**。  
  (빌린 돈을 안 갚을 수는 없지.)

> ⚠️ **한국인 자주 실수**: `わけにはいかない`를 "능력이 안 됨(できない)"으로 오해하기 쉽다. 이건 **"마음·능력은 되는데 사정상 못 한다"**는 제약 표현이다.

## 한눈에 비교

| 표현               | 의미              | 뉘앙스             | 명사 접속         |
| ------------------ | ----------------- | ------------------ | ----------------- |
| 〜わけだ           | 그래서 당연히 〜  | 납득·논리적 귀결   | 명사 + という     |
| 〜わけではない     | 꼭 〜인 건 아니다 | 부분 부정          | 명사 + という／な |
| 〜わけがない       | 〜일 리가 없다    | 강한 부정·불가능   | 명사 + の／である |
| 〜わけにはいかない | 〜할 수는 없다    | 사회적·심리적 제약 | (동사만 접속)     |

같은 동사 `やめる`(그만두다)로 4개를 비교하면 차이가 선명하다.

- 給料（きゅうりょう）が下（さ）がったのか。みんな辞（や）める**わけだ**。 (월급이 내렸구나. 다들 그만두는 게 당연하지.)
- 仕事（しごと）が嫌（いや）で辞（や）める**わけではない**。 (일이 싫어서 그만두는 건 아니야.)
- こんないい会社（かいしゃ）、辞（や）める**わけがない**。 (이런 좋은 회사를 그만둘 리가 없지.)
- 担当（たんとう）だから、途中（とちゅう）で辞（や）める**わけにはいかない**。 (담당이라, 도중에 그만둘 수는 없어.)

---

## 핵심 요약

| 뒤에 붙는 말 | 의미                        |
| ------------ | --------------------------- |
| だ           | 납득 (그래서 당연하구나)    |
| ではない     | 부분 부정 (다 그런 건 아님) |
| がない       | 강한 부정 (그럴 리 없음)    |
| にはいかない | 제약 (그럴 수는 없음)       |

> **JLPT 시험 팁**: 빈칸 문제에서 `ない형 + わけにはいかない`(=의무)와 `わけがない`(=불가능)를 바꿔치기한 함정 선택지가 단골이다. "안 할 수 없다 = 해야 한다"를 반사적으로 떠올리자.
