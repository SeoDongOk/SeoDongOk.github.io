일본어 형용사는 두 종류다. 어미에 `い`가 붙는 **い형용사**와, 명사처럼 다루는 **な형용사**다.

활용 규칙이 서로 다르기 때문에 구별하는 것부터 시작해야 한다.

## い형용사 vs な형용사 구분법

| 구분     | 기본형 끝        | 예시                 |
| -------- | ---------------- | -------------------- |
| い형용사 | い로 끝남        | 高い, 新しい, 難しい |
| な형용사 | い로 끝나지 않음 | 静か, 便利, きれい   |

> **주의:** `きれい(예쁘다)`, `嫌い(싫다)`는 `い`로 끝나지만 **な형용사**다. 외워두기.

---

## い형용사 활용

기본형에서 어미 `い`를 떼고 규칙에 맞게 바꾼다.

| 형태      | 규칙         | 예 (高い)    | 의미            |
| --------- | ------------ | ------------ | --------------- |
| 기본형    | —            | 高い         | 비싸다          |
| 부정형    | 〜くない     | 高くない     | 비싸지 않다     |
| 과거형    | 〜かった     | 高かった     | 비쌌다          |
| 과거부정  | 〜くなかった | 高くなかった | 비싸지 않았다   |
| 연결형    | 〜くて       | 高くて       | 비싸서 / 비싸고 |
| 부사형    | 〜く         | 高く         | 비싸게          |
| 명사 수식 | 〜い＋명사   | 高いもの     | 비싼 것         |
| 명사화    | 〜さ         | 高さ         | 높이/비싼 정도  |

### い형용사 예문

- この店（みせ）は **高くない** です。 (이 가게는 비싸지 않아요.)
- 昨日（きのう）は **寒かった** です。 (어제는 추웠어요.)
- このケーキは **甘くて** おいしいです。 (이 케이크는 달고 맛있어요.)

### 예외: いい (좋다)

`いい`는 **활용할 때 `よ`로 바뀐다**.

| 형태 | 활용                        |
| ---- | --------------------------- |
| 기본 | いい                        |
| 부정 | ~~いくない~~ → **よくない** |
| 과거 | ~~いかった~~ → **よかった** |
| 연결 | ~~いくて~~ → **よくて**     |

- 今日（きょう）は天気（てんき）が **よかった** です。 (오늘은 날씨가 좋았어요.)
- あまり **よくない** です。 (별로 좋지 않아요.)

## な형용사 활용

な형용사는 명사처럼 `だ / です`로 활용한다.

| 형태      | 규칙                              | 예 (静か)        | 의미                |
| --------- | --------------------------------- | ---------------- | ------------------- |
| 기본형    | 〜だ                              | 静かだ           | 조용하다            |
| 정중형    | 〜です                            | 静かです         | 조용해요            |
| 부정형    | 〜じゃない                        | 静かじゃない     | 조용하지 않다       |
| 정중부정  | 〜じゃないです / 〜ではありません | 静かじゃないです | 조용하지 않아요     |
| 과거형    | 〜だった                          | 静かだった       | 조용했다            |
| 정중과거  | 〜でした                          | 静かでした       | 조용했어요          |
| 과거부정  | 〜じゃなかった                    | 静かじゃなかった | 조용하지 않았다     |
| 연결형    | 〜で                              | 静かで           | 조용해서 / 조용하고 |
| 명사 수식 | 〜な＋명사                        | 静かな部屋       | 조용한 방           |

### な형용사 예문

- 彼女（かのじょ）は **親切（しんせつ）な** 人（ひと）です。 (그녀는 친절한 사람이에요.)
- この図書館（としょかん）は **静かで** 好（す）きです。 (이 도서관은 조용해서 좋아요.)
- 昨日（きのう）は **ひまじゃなかった** です。 (어제는 한가하지 않았어요.)

## い형용사 vs な형용사 명사 수식 비교

```
い형용사: [형용사 기본형] + 명사
  → 大きい 鞄（かばん）  큰 가방

な형용사: [어간] + な + 명사
  → 有名な レストラン  유명한 레스토랑
```
