> CFA Level 1의 자산군 과목은 투자 실무와 가장 직접적으로 연결된다. 주식, 채권, 파생상품, 대체투자를 각각 따로 보되, 마지막에는 포트폴리오 관점에서 연결해야 한다.

## 자산군 과목의 큰 그림

| 과목                    | 핵심 질문                                    |
| ----------------------- | -------------------------------------------- |
| Equity Investments      | 주식은 어떻게 발행·거래·평가되는가           |
| Fixed Income            | 채권의 가격과 위험은 어떻게 결정되는가       |
| Derivatives             | 기초자산에서 파생된 계약은 어떻게 작동하는가 |
| Alternative Investments | 전통 자산 외 투자수단은 어떤 특징이 있는가   |

이 네 과목은 각각 다르지만, 공통적으로 “수익과 위험을 어떻게 이해할 것인가”라는 질문으로 연결된다.

## Equity Investments

Equity는 주식 투자와 기업가치 평가의 기본 언어를 다룬다.

핵심 주제는 다음이다.

- 주식의 종류와 권리
- 시장 구조와 거래 방식
- 주가지수
- 산업과 기업 분석
- 배당할인모형
- 상대가치평가

Equity는 직관적이어서 초반 접근은 쉽다. 하지만 밸류에이션 문제에서는 공식의 의미를 이해해야 한다.

예를 들어 DDM을 공부할 때는 단순히 공식을 외우기보다 다음 질문을 같이 생각한다.

- 배당이 늘어나면 가치는 어떻게 변하는가?
- 요구수익률이 높아지면 가치는 어떻게 변하는가?
- 성장률이 요구수익률에 가까워지면 왜 민감해지는가?

## Fixed Income

Fixed Income은 가장 중요하게 볼 자산군 과목 중 하나다. 채권은 구조가 정해져 있지만, 가격과 위험의 관계가 깊다.

핵심 주제는 다음이다.

- 채권의 기본 구조
- 수익률 개념
- 가격과 수익률의 관계
- 듀레이션과 컨벡시티
- 신용위험
- 자산유동화

채권은 공식보다 직관을 먼저 잡아야 한다.

```text
할인율 상승 → 현재가치 하락 → 채권 가격 하락
할인율 하락 → 현재가치 상승 → 채권 가격 상승
```

이 기본 관계를 놓치면 듀레이션과 컨벡시티가 모두 흔들린다.

## Derivatives

Derivatives는 비중만 보면 낮아 보이지만, 처음 공부할 때 진입장벽이 있다.

핵심 상품은 다음이다.

- Forward
- Futures
- Swaps
- Options

Level I에서는 복잡한 전략보다 기본 구조가 중요하다.

| 상품    | 먼저 이해할 질문                              |
| ------- | --------------------------------------------- |
| Forward | 미래에 정한 가격으로 거래한다는 것은 무엇인가 |
| Futures | Forward와 표준화·증거금 구조가 어떻게 다른가  |
| Swap    | 현금흐름을 교환한다는 뜻은 무엇인가           |
| Option  | 권리와 의무의 비대칭성이 왜 중요한가          |

Derivatives는 그림을 그리면서 공부해야 한다. 현금흐름 방향을 시각화하면 훨씬 낫다.

## Alternative Investments

Alternative Investments는 전통적인 주식·채권 외 자산을 다룬다.

주요 범위는 다음이다.

- 헤지펀드
- 사모펀드
- 부동산
- 원자재
- 인프라

대체투자는 계산보다 특징 비교가 중요하다.

| 자산군   | 체크 포인트                      |
| -------- | -------------------------------- |
| 헤지펀드 | 전략, 수수료, 유동성 제약        |
| 사모펀드 | 바이아웃, 벤처, 가치창출 방식    |
| 부동산   | 직접투자와 간접투자, 임대수익    |
| 원자재   | 선물곡선, 보관비용, 롤수익       |
| 인프라   | 장기 현금흐름, 규제, 경기 민감도 |

## 공부 순서

내 자산군 공부 순서는 다음과 같다.

1. Equity로 시장과 밸류에이션 감 잡기
2. Fixed Income으로 할인율과 위험 이해하기
3. Derivatives로 기초자산과 계약 구조 이해하기
4. Alternative Investments로 비교표 만들기

Equity와 Fixed Income은 비중도 높고 실무 연결성도 크므로 시간을 더 배정한다. Derivatives와 Alternatives는 기본 구조를 빠르게 잡고 문제풀이로 확인한다.

## 자산군 과목 오답노트

자산군 과목의 오답노트는 “상품 구조”를 중심으로 적는다.

```text
상품/개념:
수익이 나는 조건:
손실이 나는 조건:
가격에 영향을 주는 변수:
헷갈린 선택지:
```

예를 들어 채권 문제라면 이자율, 만기, 쿠폰, 신용위험, 듀레이션을 같이 기록한다.

## 결론

자산군 과목은 CFA 공부를 투자 실무와 연결해주는 영역이다. 주식은 기업가치, 채권은 할인율과 위험, 파생상품은 계약 구조, 대체투자는 비교와 분산효과를 중심으로 잡는다.

앞으로 개별 과목 글에서는 Equity와 Fixed Income을 먼저 깊게 정리할 예정이다.
