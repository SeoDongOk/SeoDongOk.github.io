일본어 숫자는 외워야 할 게 많다. 기본 숫자도 두 가지 읽는 법이 있고, 세는 대상에 따라 조수사가 달라진다.  
한 번에 다 외우려 하면 무너진다. 자주 쓰는 것부터 순서대로 익히자.

---

## 기본 숫자 1〜10000

| 숫자   | 한자 | 읽기        |
| ------ | ---- | ----------- |
| 0      | 零   | ゼロ / れい |
| 1      | 一   | いち        |
| 2      | 二   | に          |
| 3      | 三   | さん        |
| 4      | 四   | し / よん   |
| 5      | 五   | ご          |
| 6      | 六   | ろく        |
| 7      | 七   | しち / なな |
| 8      | 八   | はち        |
| 9      | 九   | く / きゅう |
| 10     | 十   | じゅう      |
| 11     | 十一 | じゅういち  |
| 20     | 二十 | にじゅう    |
| 100    | 百   | ひゃく      |
| 1,000  | 千   | せん        |
| 10,000 | 万   | まん        |

> **4와 7, 9는 두 가지 읽기:** 상황마다 쓰임이 다르다. 4는 조수사 앞에서는 `よん`을 주로 쓰고, 7은 `なな`가 더 안전하다. 9는 `く`보다 `きゅう`가 헷갈림이 적다.

### 100, 1000 음변화 주의

| 숫자 | 읽기              |
| ---- | ----------------- |
| 100  | ひゃく            |
| 200  | にひゃく          |
| 300  | さんびゃく ← 변화 |
| 400  | よんひゃく        |
| 600  | ろっぴゃく ← 변화 |
| 800  | はっぴゃく ← 변화 |
| 1000 | せん              |
| 3000 | さんぜん ← 변화   |
| 8000 | はっせん ← 변화   |

## 일본어 고유어 세기 (ひとつ〜とお)

조수사 없이 물건을 셀 때 쓰는 일본어 고유어. 주로 1〜10에서만 쓴다.

| 숫자 | 고유어 | 읽기     |
| ---- | ------ | -------- |
| 1    | 一つ   | ひとつ   |
| 2    | 二つ   | ふたつ   |
| 3    | 三つ   | みっつ   |
| 4    | 四つ   | よっつ   |
| 5    | 五つ   | いつつ   |
| 6    | 六つ   | むっつ   |
| 7    | 七つ   | ななつ   |
| 8    | 八つ   | やっつ   |
| 9    | 九つ   | ここのつ |
| 10   | 十     | とお     |

- りんごを**三つ**ください。 (사과 세 개 주세요.)
- ケーキを**ひとつ** 食べました。 (케이크 하나 먹었어요.)

---

## 자주 쓰는 조수사 10가지

### 〜枚（まい） — 종이, 천처럼 얇고 납작한 것

- 紙（かみ）を3**枚**ください。 (종이 세 장 주세요.)
- チケットが2**枚**あります。 (티켓이 두 장 있어요.)

| 1장      | 2장    | 3장      | 4장      |
| -------- | ------ | -------- | -------- |
| いちまい | にまい | さんまい | よんまい |

### 〜本（ほん） — 가늘고 긴 것 (병, 연필, 우산, 도로 등)

- ビールを2**本**飲みました。 (맥주 두 병 마셨어요.)
- 鉛筆（えんぴつ）が3**本**あります。 (연필이 세 자루 있어요.)

| 1개      | 2개    | 3개      | 6개      | 8개      | 10개       |
| -------- | ------ | -------- | -------- | -------- | ---------- |
| いっぽん | にほん | さんぼん | ろっぽん | はっぽん | じゅっぽん |

> 1, 3, 6, 8, 10에서 발음 변화 있음.

### 〜個（こ） — 작은 물건, 과일, 달걀 등

- 卵（たまご）を6**個**買いました。 (달걀 여섯 개 샀어요.)
- お菓子（かし）を3**個**食べました。 (과자 세 개 먹었어요.)

### 〜冊（さつ） — 책, 노트 등 묶음 형태

- 本（ほん）を5**冊**読（よ）みました。 (책을 다섯 권 읽었어요.)

### 〜台（だい） — 기계, 차량, 가전 등

- 車（くるま）が2**台**あります。 (차가 두 대 있어요.)
- パソコンを1**台**買いました。 (컴퓨터 한 대 샀어요.)

### 〜匹（ひき） / 〜頭（とう） — 동물

- 猫（ねこ）が3**匹**います。 (고양이가 세 마리 있어요.) — 소·중형 동물
- 馬（うま）が2**頭**います。 (말이 두 마리 있어요.) — 소, 말처럼 큰 동물

### 〜羽（わ） — 새, 토끼

- 鳥（とり）が5**羽**います。 (새가 다섯 마리 있어요.)

### 〜杯（はい） — 잔, 그릇

- コーヒーを2**杯**飲みました。 (커피 두 잔 마셨어요.)
- ラーメンを1**杯**食べました。 (라면 한 그릇 먹었어요.)

| 1잔      | 2잔    | 3잔      |
| -------- | ------ | -------- |
| いっぱい | にはい | さんばい |

### 〜階（かい） — 층

- 3**階**に住（す）んでいます。 (3층에 살고 있어요.)
- エレベーターで5**階**まで行きます。 (엘리베이터로 5층까지 가요.)

### 〜人（にん） — 사람 수

- 3**人**で行きました。 (세 명이서 갔어요.)

| 1명    | 2명    | 3명〜      |
| ------ | ------ | ---------- |
| ひとり | ふたり | さんにん〜 |
