사역형은 **누군가에게 무언가를 시킨다**는 표현이다. "먹게 하다", "가게 하다", "공부시키다"에 해당한다.

허가(허락해서 하게 함)와 강제(억지로 시킴) 두 가지 뉘앙스가 모두 있다.

---

## 사역형 만들기

### 2그룹 동사: る → させる

| 사전형 | 사역형     |
| ------ | ---------- |
| 食べる | 食べさせる |
| 見る   | 見させる   |
| 起きる | 起きさせる |
| 覚える | 覚えさせる |

### 1그룹 동사: あ단 + せる

| 사전형 | 사역형   |
| ------ | -------- |
| 書く   | 書かせる |
| 飲む   | 飲ませる |
| 話す   | 話させる |
| 帰る   | 帰らせる |
| 待つ   | 待たせる |

> **う로 끝나는 동사:** う → わ + せる  
> 歌う → 歌わ**せる**

### 3그룹 (불규칙)

| 사전형 | 사역형   |
| ------ | -------- |
| する   | させる   |
| くる   | こさせる |

---

## 사역형 문장 구조

```
[시키는 사람]は [당하는 사람]を/に [동사 사역형]
```

- 타동사: **를 → に**
- 자동사: **를 → を**

| 구분        | 예                                            |
| ----------- | --------------------------------------------- |
| 자동사 + を | 子どもを泣かせた (아이를 울게 했다)           |
| 타동사 + に | 子どもに宿題をさせた (아이에게 숙제를 시켰다) |

---

## 허가 vs 강제

같은 사역형이지만 문맥에 따라 다르다.

### 허가 (〜させてください / させてもらう)

- 少し休ま**せてください**。 (잠깐 쉬게 해 주세요.)
- トイレに行か**せてください**。 (화장실에 가게 해 주세요.)
- やって**みさせてください**。 (해 보게 해 주세요.)

### 강제 (강요·지시)

- 部長が残業**させました**。 (부장님이 야근을 시켰어요.)
- 子どもに野菜を食べ**させた**。 (아이에게 채소를 먹게 했다.)
- 先生は生徒を廊下に立た**せた**。 (선생님이 학생을 복도에 세웠다.)

---

## 자주 쓰는 패턴

### 〜させてください — 허가 요청

가장 자주 쓰이는 사역 표현. "〜하게 해 주세요"

- 確認**させてください**。 (확인하게 해 주세요.)
- 説明**させてください**。 (설명하게 해 주세요.)
- 少し考え**させてください**。 (좀 생각하게 해 주세요.)

### 〜させてもらう/いただく — 허가 받아서 하다

- 早退**させてもらいます**。 (조기 퇴근하겠습니다. — 허가를 받아)
- 発表**させていただきます**。 (발표하겠습니다. — 공식·공손)

---

## 사역수동: 〜させられる — 억지로 〜하게 당하다

사역형 + 수동형. **원하지 않는데 억지로 시켜졌다**는 뉘앙스.

```
사역형 어간 + られる
```

| 동사   | 사역형     | 사역수동형               |
| ------ | ---------- | ------------------------ |
| 食べる | 食べさせる | 食べさせられる           |
| 飲む   | 飲ませる   | 飲まされる (구어체 단축) |
| 書く   | 書かせる   | 書かされる               |

- 嫌いなものを食べ**させられた**。 (싫어하는 걸 억지로 먹게 됐다.)
- 残業を**させられました**。 (야근을 억지로 해야 했습니다.)
- 子どもの頃、ピアノを練習**させられました**。  
  (어릴 때 피아노 연습을 (억지로) 해야 했습니다.)

---

## 핵심 요약

| 표현             | 의미               | 뉘앙스         |
| ---------------- | ------------------ | -------------- |
| 〜させる         | 시키다/하게 하다   | 허가 또는 강제 |
| 〜させてください | 〜하게 해 주세요   | 허가 요청      |
| 〜させてもらう   | 허가 받아 〜하다   | 겸손 표현      |
| 〜させられる     | 억지로 〜하게 되다 | 강제당함       |
