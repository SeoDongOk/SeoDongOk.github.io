> 투자자는 단순히 수익률이 양수인지 보지 않는다. 위험, 인플레이션, 대체 투자 기회까지 고려했을 때 충분한 보상을 주는지 본다. 이 기준이 요구수익률이다.

## 공식 범위 확인

2027 Quant 첫 모듈은 수익률뿐 아니라 요구수익률, 무위험수익률, 위험프리미엄, 인플레이션의 의미를 비교하고 해석하는 능력을 요구한다.

## 요구수익률이란

요구수익률은 투자자가 어떤 자산을 보유하기 위해 최소한 기대하는 수익률이다.

```text
요구수익률 ≈ 무위험수익률 + 위험프리미엄
```

현실에서는 세금, 유동성, 투자기간, 인플레이션 기대, 신용위험, 시장위험 등이 모두 반영된다. Level I에서는 이 요소들을 분해해서 해석하는 것이 중요하다.

## 무위험수익률

무위험수익률은 위험이 거의 없는 투자에서 기대할 수 있는 수익률이다. 보통 단기 국채 수익률을 근사치로 사용한다.

무위험수익률은 두 가지 요소로 나누어 생각할 수 있다.

| 요소                | 의미                                  |
| ------------------- | ------------------------------------- |
| Real risk-free rate | 인플레이션을 제외한 실질 무위험수익률 |
| Expected inflation  | 구매력 하락에 대한 보상               |

정확한 Fisher 관계는 곱셈 구조다.

```text
1 + nominal risk-free rate = (1 + real risk-free rate)(1 + expected inflation)
```

수익률이 낮을 때는 더하기 근사가 자주 쓰인다.

```text
nominal risk-free rate ≈ real risk-free rate + expected inflation
```

## 위험프리미엄

위험프리미엄은 불확실성을 감수하는 대가다.

| 위험프리미엄        | 의미                              |
| ------------------- | --------------------------------- |
| Equity risk premium | 주식 위험에 대한 보상             |
| Credit risk premium | 채무불이행 위험에 대한 보상       |
| Liquidity premium   | 빨리 팔기 어려운 자산에 대한 보상 |
| Maturity premium    | 장기 투자기간에 대한 보상         |

위험이 높다고 항상 실현수익률이 높아지는 것은 아니다. 요구수익률이 높아진다는 뜻은 “투자자가 더 높은 기대수익률을 요구한다”는 의미다.

## 인플레이션과 실질수익률

명목수익률이 7%이고 물가상승률이 3%라면 단순히 4%라고 말할 수도 있지만, 정확한 실질수익률은 다음이다.

```text
Real return = (1 + nominal return) / (1 + inflation rate) - 1
```

```text
(1.07 / 1.03) - 1 = 3.88%
```

수익률이 낮을 때는 근사치가 괜찮지만, 시험에서는 정확한 공식을 요구하는지 확인해야 한다.

## 시험에서 헷갈리는 지점

| 헷갈리는 지점                   | 정리                                                       |
| ------------------------------- | ---------------------------------------------------------- |
| 요구수익률과 기대수익률         | 요구수익률은 필요한 최소 보상, 기대수익률은 예상 평균 성과 |
| 위험이 높으면 실현수익률도 높다 | 아니다. 요구수익률이 높아질 뿐 실현 결과는 불확실하다      |
| 명목수익률과 실질수익률         | 실질수익률은 인플레이션을 제거한 구매력 기준 수익률        |
| 더하기 공식과 곱셈 공식         | 정확한 관계는 곱셈, 낮은 수익률에서는 더하기 근사 가능     |

## 체크리스트

- [ ] 요구수익률을 무위험수익률과 위험프리미엄으로 설명할 수 있다.
- [ ] Fisher 관계를 곱셈 공식으로 쓸 수 있다.
- [ ] 명목수익률과 실질수익률을 구분할 수 있다.
- [ ] 위험프리미엄의 종류를 사례로 설명할 수 있다.

이 모듈의 핵심은 “수익률은 숫자 하나가 아니라 보상 구조”라는 점이다. 뒤에서 채권, 주식, 포트폴리오를 볼 때도 이 관점이 계속 필요하다.
