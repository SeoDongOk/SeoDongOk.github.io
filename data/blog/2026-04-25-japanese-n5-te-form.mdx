마스형(ます形)이 일본어 정중체의 **뼈대**라면, 「**て形(테형)**」은 일본어 활용의 **만능 연결고리**다.

진행형(~ています), 부탁(~てください), 허가(~てもいいです), 금지(~てはいけません), 시간 순서(~てから), 동작 연결(~て、~) — N4~N3에서 마주치는 거의 모든 문법이 이 て形에서 출발한다.

## 1. て形이란 무엇인가

「て形」은 동사 활용 중 **음편(音便, おんびん)** 변화를 거쳐 만들어지는 형태로, 다양한 문법의 기초가 된다. 핵심 역할은 4가지다.

| 역할               | 예시               | 한국어             |
| ------------------ | ------------------ | ------------------ |
| **동작 연결**      | 食べ**て**寝る     | 먹**고** 자다      |
| **진행/상태**      | 食べ**ている**     | 먹**고 있다**      |
| **부탁/허가/금지** | 食べ**てください** | 먹**어 주세요**    |
| **시간 관계**      | 食べ**てから**行く | 먹**고 나서** 가다 |

문법적으로 て形은 동사의 **연용형(連用形, れんようけい)** 에 접속조사 「て」가 붙은 형태다. 1그룹 동사에서는 발음 편의를 위해 **음편변화**가 일어난다.

---

## 2. 그룹별 て形 만들기

### 2-1. 1그룹 (五段動詞) — 어미별 5가지 패턴

1그룹은 사전형 마지막 글자(う단)에 따라 て形이 5가지로 갈라진다. 이게 음편변화의 핵심이다.

| 사전형 어미    | て形 변화 | 예시                                                              |
| -------------- | --------- | ----------------------------------------------------------------- |
| **う, つ, る** | **っ**て  | 買（か）う → 買って / 待（ま）つ → 待って / 帰（かえ）る → 帰って |
| **ぬ, ぶ, む** | **ん**で  | 死（し）ぬ → 死んで / 遊（あそ）ぶ → 遊んで / 飲（の）む → 飲んで |
| **く**         | **い**て  | 書（か）く → 書いて / 聞（き）く → 聞いて                         |
| **ぐ**         | **い**で  | 泳（およ）ぐ → 泳いで / 急（いそ）ぐ → 急いで                     |
| **す**         | **し**て  | 話（はな）す → 話して / 貸（か）す → 貸して                       |

#### ⚠️ 유일한 예외 — 行く(いく)

「行く」는 어미가 「く」지만 て形은 **「行って」** 다. 「行いて」가 아니다.

- 書く → 書い**て** ⭕ (く → いて 규칙)
- 行く → 行っ**て** ⭕ (예외)

이 하나만큼은 무조건 외워둬야 한다. JLPT N5에서 100% 출제되는 함정이다.

### 2-2. 2그룹 (一段動詞) — 「る」 떼고 「て」 붙이기

| 사전형         | て形       |
| -------------- | ---------- |
| 食（た）べる   | 食べ**て** |
| 見（み）る     | 見**て**   |
| 寝（ね）る     | 寝**て**   |
| 起（お）きる   | 起き**て** |
| 教（おし）える | 教え**て** |

ます形을 만드는 방법(る를 떼고 ます 붙이기)과 동일하다. ます 자리에 て만 넣으면 끝.

### 2-3. 3그룹 (불규칙) — 외우기

| 사전형     | て形           |
| ---------- | -------------- |
| する       | し**て**       |
| 来（く）る | 来（き）**て** |
| 勉強する   | 勉強し**て**   |

### 외우는 꿀팁 — 「て形의 노래」

> 🎵 **う・つ・る → って**
> 🎵 **ぬ・ぶ・む → んで**
> 🎵 **く → いて**, **ぐ → いで**
> 🎵 **す → して**
> 🎵 **行く만 예외 → 行って**

리듬 타서 5~10번 반복해 읽으면 자동으로 입에 붙는다. 일본어 학원에서 실제로 쓰는 암기법이다.

## 3. て形 활용 TOP 7

て形을 만들 줄 알게 되었다면, 이제 N5 출제·회화에 등장하는 **7가지 활용**을 정리한다.

### ① 〜ています — 진행형 / 상태

현재 진행 중이거나 결과가 지속되는 상태. 한국어 "~하고 있어요"에 대응.

1. 今（いま）、ご飯（はん）を食べ**ています**。
   (지금 밥을 먹고 있어요.)
2. 妹（いもうと）は本を読ん**でいます**。
   (여동생은 책을 읽고 있어요.)
3. 結婚（けっこん）し**ています**。
   (결혼했어요 — 결혼한 상태가 지속!)

> 💡 「結婚しています」는 진행이 아니라 **기혼 상태**를 뜻한다. 한국어 "결혼하고 있어요"로 직역하면 어색하니 주의.

### ② 〜てください — 부탁 / 정중한 지시

"~해 주세요". 선생님, 의사, 점원이 자주 사용.

1. ちょっと待（ま）っ**てください**。 (잠깐 기다려 주세요.)
2. 名前（なまえ）を書い**てください**。 (이름을 써 주세요.)
3. ゆっくり話し**てください**。 (천천히 말해 주세요.)

### ③ 〜てもいいです — 허가

"~해도 돼요". 허락을 구하거나 허락할 때.

1. ここに座（すわ）っ**てもいいですか**。 (여기 앉아도 됩니까?)
2. 写真（しゃしん）を撮（と）っ**てもいいです**。 (사진 찍어도 돼요.)
3. トイレに行っ**てもいいですか**。 (화장실 가도 됩니까?)

### ④ 〜てはいけません — 금지

"~하면 안 돼요". 규칙·금지 표현. 회화에서는 「〜ちゃいけません」, 「〜ちゃだめ」로 줄여 쓰기도 한다.

1. ここでタバコを吸（す）っ**てはいけません**。 (여기서 담배 피우면 안 돼요.)
2. 授業中（じゅぎょうちゅう）、話し**てはいけません**。 (수업 중에 말하면 안 돼요.)
3. 嘘（うそ）をつい**てはいけません**。 (거짓말하면 안 돼요.)

### ⑤ 〜てから — ~하고 나서

시간 순서를 명확히 표현. "A 한 뒤에 B 한다".

1. 宿題（しゅくだい）をし**てから**、遊（あそ）びます。 (숙제를 하고 나서 놀아요.)
2. ご飯を食べ**てから**、歯（は）を磨（みが）きます。 (밥을 먹고 나서 이를 닦아요.)
3. 日本に来**てから**、3年（ねん）経（た）ちました。 (일본에 오고 나서 3년이 지났어요.)

### ⑥ 〜て、〜 — 동작 연결 (여러 동작 나열)

여러 동작을 순서대로 나열하거나 한 문장으로 이어 말할 때. 한국어 "~하고 ~하고"에 해당.

1. 朝起き**て**、顔を洗っ**て**、ご飯を食べます。 (아침에 일어나서 세수하고 밥을 먹어요.)
2. 友達に会っ**て**、映画を見ました。 (친구를 만나서 영화를 봤어요.)
3. お弁当（べんとう）を買っ**て**、公園で食べました。 (도시락을 사서 공원에서 먹었어요.)

### ⑦ 〜ています (직업·거주) — 특수 용법

직업·거주지처럼 "현재 지속되는 상태"도 ています로 표현한다.

1. 東京（とうきょう）に住（す）ん**でいます**。 (도쿄에 살고 있어요.)
2. 銀行（ぎんこう）で働（はたら）い**ています**。 (은행에서 일하고 있어요.)
3. 日本語を勉強し**ています**。 (일본어를 공부하고 있어요.)

## 4. 한국인이 자주 틀리는 포인트

### ① 「行く」를 규칙대로 변형

- ❌ 行**いて** (X)
- ⭕ 行**って** (예외)

### ② 「帰る」를 2그룹으로 착각

EP.1에서 다룬 함정 동사들(帰る、走る、入る、切る、知る)은 모두 **1그룹**이다.

- ❌ 帰**て** (2그룹으로 착각)
- ⭕ 帰**って** (1그룹, 어미 る → って)

### ③ ています의 "진행 vs 상태" 혼동

| 일본어             | 한국어      | 의미                  |
| ------------------ | ----------- | --------------------- |
| 食べています       | 먹고 있어요 | 진행 (지금 먹는 중)   |
| 結婚しています     | 결혼했어요  | 상태 (기혼)           |
| 知（し）っています | 알고 있어요 | 상태 (알고 있는 상태) |

「知る」는 반드시 「知っています」(O) / 「知りません」(O, 모른다)로 쓴다. 「知ります」는 거의 쓰지 않는다.

### ④ する의 て形을 「すて」로 착각

- ❌ すて (これは「捨てる(버리다)」!)
- ⭕ し**て** (する → して)

## 다음 편 예고 (EP.3)

て形을 정복했으니 다음은 **반말의 세계** — 「**ない形(부정 보통체)**」다.

- 食べる → 食べ**ない** (안 먹어)
- 行く → 行か**ない** (안 가)
- 来る → **こない** / する → **しない**

ない形은 て形과 짝을 이루는 핵심 활용이고, 「〜なくてもいいです(안 해도 돼요)」, 「〜ないでください(하지 마세요)」 같은 필수 문형의 출발점이다. 다음 편에서 이어진다.

## 마무리

て形은 N5에서 가장 양이 많고 가장 헷갈리는 활용이지만, 동시에 **N4~N1까지 평생 쓰는 기반 문법**이다. 1그룹 5패턴 + 예외 行く + 2그룹 + 3그룹 — 이 묶음이 입에 붙을 때까지 반복해서 소리 내어 연습하는 게 정공법이다.

「て形의 노래」를 다시 불러보자.

> う・つ・る → って / ぬ・ぶ・む → んで / く → いて / ぐ → いで / す → して / 行く만 예외 → 行って
