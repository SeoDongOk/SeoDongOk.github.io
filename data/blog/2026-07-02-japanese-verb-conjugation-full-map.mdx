일본어 문법이 어렵게 느껴지는 진짜 이유는 단어 뜻이 아니라 **동사 변형**이다.
바꿔 말하면, **동사 하나가 어떻게 여러 형태로 변하는지**만 잡으면 N4 문법의 절반은 끝난다.

이 글은 동사 하나(뿌리)에서 뻗어나가는 **모든 활용형을 하나의 맵으로** 정리한 완성판이다.
(N5 기본 활용만 필요하면 별도의 기본 변동표 글을 먼저 봐도 좋다. 이 글은 수동·사역까지 확장한 버전이다.)

## 핵심 관점: 동사 = 함수, 활용 = 파라미터 갈아끼우기

동사 하나는 **루트(root)**이고, 거기서 목적에 따라 형태가 파생된다.

```text
書く(쓰다)
 ├─ 書きます    (정중)
 ├─ 書いて      (연결)
 ├─ 書かない    (부정)
 ├─ 書いた      (과거)
 ├─ 書ける      (가능)
 ├─ 書こう      (의지)
 ├─ 書けば      (조건)
 ├─ 書かれる    (수동)
 ├─ 書かせる    (사역)
 └─ 書かせられる (사역수동)
```

이 전부가 「書く」 하나에서 나온 변형이다. 나머지 문법은 여기에 문형(〜てください, 〜なければ 등)을 붙이는 것뿐이다.

## 🔑 최대 꿀팁: ない형 어간 = 수동·사역의 뿌리

한국인 학습자가 수동·사역을 어렵게 느끼는데, 사실 **ない형만 만들 줄 알면 끝난다.**

```text
書く → 書か(ない)   ← ない형 어간 = 書か
         ├─ 書か + れる = 수동 (당하다)
         └─ 書か + せる = 사역 (시키다)
```

즉 부정형(書かない)을 만들 때 나오는 어간 **「書か」(a단)**에 `れる`를 붙이면 수동, `せる`를 붙이면 사역이다.
새로 외우는 게 아니라 **같은 뿌리에 어미만 갈아끼우는 것**이다.

## 변형별 만드는 법 & 의미

| 활용형      | 1그룹                         | 2그룹                  | 의미              |
| ----------- | ----------------------------- | ---------------------- | ----------------- |
| 수동 (受身) | a단 + れる (書かれる)         | る→られる (食べられる) | ~당하다/되다      |
| 사역 (使役) | a단 + せる (書かせる)         | る→させる (食べさせる) | ~시키다/하게 하다 |
| 사역수동    | a단 + せられる (書かせられる) | る→させられる          | 억지로 ~하다      |
| 가능 (可能) | e단 + る (書ける)             | る→られる (食べられる) | ~할 수 있다       |
| 의지 (意志) | o단 + う (書こう)             | る→よう (食べよう)     | ~하자/~해야지     |
| 조건 (仮定) | e단 + ば (書けば)             | る→れば (食べれば)     | ~하면             |

## ⚠️ 시험 단골 함정 3가지

1. **2그룹 가능 = 수동 동형**: 食べられる가 "먹을 수 있다"인지 "먹히다"인지 문맥으로 판단. (来られる도 마찬가지)
2. **1그룹 사역수동 구어 단축**: 書かせられる → **書かされる**, 飲ませられる → **飲まされる**. 단 `す`동사는 단축 불가(話させられる ○ / 話さされる ✗).
3. **する의 가능형은 できる**: 형태가 완전히 다르니 통째로 암기.

## 그룹별 변형 규칙 (한 줄 요약)

- **1그룹**: 어미 u단을 목적에 따라 **a·i·e·o단**으로 갈아끼운다. (書**く**→書**か**/書**き**/書**け**/書**こ**)
- **2그룹**: 마지막 **る를 떼고** 붙인다. (제일 규칙적)
- **3그룹**: する·来る 두 개만 **통째로** 외운다.

## 마무리

일본어 문법은 결국 **동사 변형 시스템**이다.

- 동사 = 뿌리(함수), 활용 = 목적별 파라미터
- ない형 어간(a단)을 잡으면 수동·사역까지 자동 연결
- 그룹별 규칙 3개(1그룹 단 교체 / 2그룹 る 제거 / 3그룹 암기)만 확실히

이 변형 맵 하나를 손에 익히면, 뒤에 붙는 문형이 아무리 늘어도 뼈대는 흔들리지 않는다.
단어를 외우기 전에, 이 "변형 뼈대"부터 소리 내어 굴려 보자.
