> 우리는 모집단 전체를 거의 알 수 없다. 그래서 표본을 뽑고, 표본 통계량으로 모집단을 추정한다. 이때 필요한 개념이 표본분포, 중심극한정리, 신뢰구간이다.

## 공식 범위 확인

2027 Quant의 Estimation and Hypothesis Testing 모듈은 중심극한정리, 신뢰구간, 표본추출 방법론의 적용을 설명하는 내용을 포함한다.

## 모집단과 표본

| 구분       | 의미                        |
| ---------- | --------------------------- |
| Population | 관심 있는 전체 집단         |
| Sample     | 모집단에서 뽑은 일부 관측치 |
| Parameter  | 모집단의 진짜 값            |
| Statistic  | 표본에서 계산한 값          |

예를 들어 모든 주식형 펀드의 평균수익률이 궁금하지만 전체를 알기 어렵다면 일부 펀드를 표본으로 뽑아 추정한다.

## 표본평균의 표준오차

표본평균은 표본마다 달라진다. 이 변동성을 표준오차라고 한다.

```text
Standard error of sample mean = σ / sqrt(n)
```

모집단 표준편차를 모르면 표본 표준편차 `s`를 사용한다.

```text
Estimated standard error = s / sqrt(n)
```

표본 수가 커질수록 표준오차는 작아진다.

## 중심극한정리

중심극한정리는 표본 수가 충분히 크면 표본평균의 분포가 정규분포에 가까워진다는 생각이다. 모집단이 완벽히 정규분포가 아니더라도 표본평균은 정규근사를 사용할 수 있게 된다.

이 개념 덕분에 신뢰구간과 가설검정이 가능해진다.

## 신뢰구간

신뢰구간은 모집단 모수가 있을 법한 범위를 제시한다.

```text
Confidence interval = point estimate ± reliability factor × standard error
```

예를 들어 표본평균이 8%, 표준오차가 2%, 신뢰수준에 해당하는 reliability factor가 1.96이라면:

```text
8% ± 1.96 × 2% = 4.08% ~ 11.92%
```

## z분포와 t분포

| 상황                                      | 주로 사용하는 분포 |
| ----------------------------------------- | ------------------ |
| 모집단 분산을 알고 표본이 충분히 큼       | z분포              |
| 모집단 분산을 모르고 표본 표준편차를 사용 | t분포              |

실무와 시험에서는 모집단 분산을 모르는 경우가 많아서 t분포가 자주 등장한다. 표본 수가 커질수록 t분포는 z분포에 가까워진다.

## 표본추출 방법

| 방법                   | 설명                                  |
| ---------------------- | ------------------------------------- |
| Simple random sampling | 모든 표본이 같은 확률로 선택          |
| Stratified sampling    | 모집단을 층으로 나누고 각 층에서 추출 |
| Systematic sampling    | 일정 간격으로 추출                    |
| Cluster sampling       | 군집 단위로 추출                      |

표본이 편향되면 추정도 편향된다. 표본추출 방식은 단순한 절차 문제가 아니라 결과의 신뢰성과 연결된다.

## 오답 포인트

| 오답 포인트              | 체크 방법                                              |
| ------------------------ | ------------------------------------------------------ |
| 표준편차와 표준오차 혼동 | 표준오차는 표본통계량의 변동성                         |
| 신뢰수준 해석 오류       | 특정 구간에 모수가 있을 확률이라는 식의 단순 해석 주의 |
| t분포와 z분포 혼동       | 모집단 분산을 아는지 확인                              |
| 표본 수 효과 무시        | `n`이 커지면 표준오차 감소                             |

## 체크리스트

- [ ] parameter와 statistic을 구분할 수 있다.
- [ ] 표준오차가 표본 수의 제곱근에 반비례함을 안다.
- [ ] 중심극한정리의 직관을 설명할 수 있다.
- [ ] 신뢰구간 공식을 쓸 수 있다.
