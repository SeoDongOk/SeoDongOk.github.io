> 포트폴리오 위험은 개별 자산 위험의 단순 평균이 아니다. 자산들이 함께 어떻게 움직이는지가 핵심이다.

## 공식 범위 확인

2027 Quant의 The Return and Risk of a Financial Portfolio 모듈은 포트폴리오 수익률의 기대값, 분산, 표준편차, 공분산, 상관계수를 계산·해석·평가하는 내용을 포함한다.

## 포트폴리오 기대수익률

포트폴리오 기대수익률은 각 자산 기대수익률의 가중평균이다.

```text
E(Rp) = w1E(R1) + w2E(R2) + ... + wnE(Rn)
```

예를 들어 A 자산 60%, B 자산 40%이고 기대수익률이 각각 8%, 4%라면:

```text
0.6×8% + 0.4×4% = 6.4%
```

## 두 자산 포트폴리오 분산

두 자산 포트폴리오의 분산은 다음 구조다.

```text
σp² = w1²σ1² + w2²σ2² + 2w1w2ρ12σ1σ2
```

마지막 항이 핵심이다. 상관계수 `ρ12`가 낮을수록 포트폴리오 분산은 줄어든다.

## 분산효과

분산투자는 단순히 자산 수를 늘리는 것이 아니다. 서로 완전히 같은 방향으로 움직이지 않는 자산을 조합해야 한다.

| 상관계수 | 분산효과               |
| -------- | ---------------------- |
| +1       | 분산효과 없음          |
| 0        | 분산효과 있음          |
| -1       | 이론적으로 큰 분산효과 |

상관계수가 +1보다 낮으면 포트폴리오 표준편차가 가중평균 표준편차보다 낮아질 수 있다.

## 공분산과 상관계수 연결

공분산과 상관계수는 다음 관계를 가진다.

```text
Cov(R1, R2) = ρ12σ1σ2
```

문제에서 공분산이 주어질 수도 있고 상관계수가 주어질 수도 있다. 두 값을 변환할 수 있어야 한다.

## 계산 예시

A와 B의 비중이 각각 50%, 표준편차가 각각 20%, 10%, 상관계수가 0.3이라고 하자.

```text
σp² = 0.5²×0.20² + 0.5²×0.10² + 2×0.5×0.5×0.3×0.20×0.10
     = 0.0100 + 0.0025 + 0.0030
     = 0.0155
σp = 12.45%
```

개별 표준편차의 단순 평균은 15%지만 포트폴리오 표준편차는 12.45%다. 이것이 분산효과다.

## 오답 포인트

| 오답 포인트                                   | 체크 방법                    |
| --------------------------------------------- | ---------------------------- |
| 기대수익률과 분산 모두 단순 가중평균으로 계산 | 분산에는 공분산 항 포함      |
| 표준편차를 바로 더함                          | 먼저 분산을 계산한 뒤 제곱근 |
| 상관계수와 공분산 혼동                        | `Cov = ρσ1σ2`                |
| 퍼센트 단위 오류                              | 계산 시 20%를 0.20으로 변환  |

## 체크리스트

- [ ] 포트폴리오 기대수익률을 가중평균으로 계산할 수 있다.
- [ ] 두 자산 포트폴리오 분산 공식을 쓸 수 있다.
- [ ] 상관계수가 낮을수록 분산효과가 커지는 이유를 설명할 수 있다.
- [ ] 공분산과 상관계수를 변환할 수 있다.
