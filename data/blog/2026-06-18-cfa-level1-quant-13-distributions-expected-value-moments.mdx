> 투자 결과는 확정값이 아니라 확률분포다. 기대값은 그 분포의 중심을, 분산은 불확실성의 크기를, 왜도와 첨도는 꼬리 위험의 형태를 보여준다.

## 공식 범위 확인

2027 Quant의 Statistical Distributions 모듈은 평균, 분산, 공분산의 unconditional expected value를 계산·해석·평가하고, 금융에서 쓰이는 주요 통계분포의 principal moments를 다룬다.

## 확률분포란

확률분포는 가능한 결과와 각 결과의 확률을 정리한 것이다.

예를 들어 어떤 자산의 1년 후 수익률이 다음과 같다고 하자.

| 상태 | 수익률 | 확률 |
| ---- | -----: | ---: |
| 호황 |    15% |  30% |
| 보통 |     6% |  50% |
| 침체 |    -8% |  20% |

이 표가 하나의 이산 확률분포다.

## 기대값

기대값은 확률가중평균이다.

```text
E(R) = Σ p_i R_i
```

위 예시에서는:

```text
0.30×15% + 0.50×6% + 0.20×(-8%) = 5.9%
```

기대값은 “가장 가능성 높은 값”이 아니라 장기 평균에 가까운 개념이다.

## 분산과 표준편차

확률분포의 분산은 기대값에서 떨어진 정도를 확률가중해 계산한다.

```text
Variance = Σ p_i [R_i - E(R)]^2
```

표준편차는 분산의 제곱근이다.

## 공분산

두 자산의 상태별 수익률이 주어지면 공분산도 확률가중으로 계산할 수 있다.

```text
Cov(RA, RB) = Σ p_i [RA_i - E(RA)][RB_i - E(RB)]
```

이 식은 포트폴리오 분산에서 그대로 이어진다.

## 주요 모멘트

분포를 설명하는 대표적인 네 가지 모멘트는 다음이다.

| 모멘트     | 의미            |
| ---------- | --------------- |
| 1차 모멘트 | 평균, 중심      |
| 2차 모멘트 | 분산, 흩어짐    |
| 3차 모멘트 | 왜도, 비대칭성  |
| 4차 모멘트 | 첨도, 꼬리 두께 |

CFA 문제에서는 “평균만 높은 자산”보다 “위험과 꼬리 특성까지 고려한 자산”을 해석하게 만들 수 있다.

## 자주 쓰는 분포의 직관

| 분포      | 직관                                           |
| --------- | ---------------------------------------------- |
| Uniform   | 범위 안의 결과가 동일한 확률                   |
| Binomial  | 성공/실패 같은 이산 결과 반복                  |
| Normal    | 평균 중심의 대칭 분포                          |
| Lognormal | 가격처럼 0보다 작아질 수 없는 변수에 자주 사용 |

정확한 공식보다 어떤 상황에 어떤 분포가 자연스러운지 이해하는 것이 중요하다.

## 오답 포인트

| 오답 포인트              | 체크 방법                            |
| ------------------------ | ------------------------------------ |
| 기대값을 최빈값으로 착각 | 기대값은 확률가중평균                |
| 확률 합계 오류           | 확률 합이 1인지 확인                 |
| 분산 계산에서 확률 누락  | 각 편차제곱에 확률을 곱함            |
| 가격과 수익률 분포 혼동  | 가격은 음수가 될 수 없다는 제약 확인 |

## 체크리스트

- [ ] 이산 확률분포에서 기대값을 계산할 수 있다.
- [ ] 확률가중 분산과 표준편차를 계산할 수 있다.
- [ ] 공분산을 상태별 확률로 해석할 수 있다.
- [ ] 평균, 분산, 왜도, 첨도를 모멘트 관점에서 설명할 수 있다.
