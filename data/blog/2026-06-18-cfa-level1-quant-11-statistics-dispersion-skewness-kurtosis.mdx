> 평균만 보면 위험을 놓친다. 같은 평균수익률이라도 변동성이 큰 자산과 작은 자산은 완전히 다르다. 분산, 표준편차, 왜도, 첨도는 수익률 분포의 위험을 읽는 도구다.

## 공식 범위 확인

2027 Quant의 Statistical Characteristics 모듈은 중심경향과 분산뿐 아니라 왜도와 첨도를 설명·해석·평가하는 내용을 포함한다.

## 분산과 표준편차

분산은 관측값이 평균에서 얼마나 떨어져 있는지를 제곱해서 평균낸 값이다.

```text
Population variance = Σ(xi - μ)^2 / N
```

표본분산은 보통 자유도 조정을 반영한다.

```text
Sample variance = Σ(xi - x̄)^2 / (n - 1)
```

표준편차는 분산의 제곱근이다.

```text
Standard deviation = sqrt(variance)
```

표준편차는 원래 데이터와 같은 단위를 가지므로 해석이 더 쉽다.

## Range와 Mean absolute deviation

흩어짐을 보는 다른 방식도 있다.

| 측도                    | 설명                     |
| ----------------------- | ------------------------ |
| Range                   | 최댓값 - 최솟값          |
| Mean absolute deviation | 평균에서의 절대편차 평균 |
| Variance                | 평균에서의 제곱편차 평균 |
| Standard deviation      | 분산의 제곱근            |

Range는 단순하지만 극단값 하나에 매우 민감하다.

## 왜도

왜도는 분포의 비대칭성을 나타낸다.

| 왜도          | 해석                                                  |
| ------------- | ----------------------------------------------------- |
| Positive skew | 오른쪽 꼬리가 김. 큰 양수 수익률 가능성이 꼬리에 있음 |
| Negative skew | 왼쪽 꼬리가 김. 큰 손실 가능성이 꼬리에 있음          |
| Zero skew     | 대칭에 가까움                                         |

투자자 입장에서는 평균과 표준편차가 같아도 negative skew가 큰 자산은 큰 손실 위험 때문에 부담스러울 수 있다.

## 첨도

첨도는 분포의 꼬리 두께와 중심 집중도를 보는 측도다. 정규분포의 kurtosis는 3이고, excess kurtosis는 `kurtosis - 3`으로 표현한다.

| 구분                | 의미                             |
| ------------------- | -------------------------------- |
| High kurtosis       | 꼬리 사건 가능성이 상대적으로 큼 |
| Excess kurtosis > 0 | 정규분포보다 fat tail 성격       |
| Excess kurtosis = 0 | 정규분포와 같은 기준             |

금융수익률은 정규분포보다 꼬리가 두꺼운 경우가 많다. 그래서 평균과 표준편차만으로 위험을 과소평가할 수 있다.

## 오답 포인트

| 오답 포인트                     | 체크 방법                   |
| ------------------------------- | --------------------------- |
| 분산과 표준편차 단위 혼동       | 표준편차는 원래 데이터 단위 |
| 표본분산 분모 오류              | 표본은 보통 `n-1`           |
| Positive skew 해석 오류         | 오른쪽 꼬리가 긴 분포       |
| Kurtosis와 excess kurtosis 혼동 | excess kurtosis는 3을 뺀 값 |

## 체크리스트

- [ ] 표본분산과 모집단분산의 분모 차이를 안다.
- [ ] 표준편차가 위험 측도로 쓰이는 이유를 설명할 수 있다.
- [ ] positive skew와 negative skew를 그림 없이 설명할 수 있다.
- [ ] excess kurtosis의 기준이 정규분포라는 점을 안다.
