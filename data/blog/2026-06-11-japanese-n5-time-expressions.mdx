"먹기 전에", "먹고 나서", "먹으면서", "먹는 동안" — 시간 관계를 나타내는 표현은 일상 회화에서 빠짐없이 등장한다.

---

## 〜前に — 〜하기 전에

```
동사 사전형 + 前に
명사 + の前に
```

**포인트:** 사전형(기본형)을 쓴다. ました형을 쓰지 않는다.

- 寝る**前に**、歯（は）を磨（みが）いてください。 (자기 전에 이를 닦아 주세요.)
- 食事（しょくじ）の**前に**、手を洗います。 (식사 전에 손을 씻어요.)
- 日本に来る**前に**、少し日本語を勉強しました。  
  (일본에 오기 전에 일본어를 조금 공부했어요.)
- 試験**の前に**、よく眠（ねむ）れませんでした。  
  (시험 전에 잘 자지 못했어요.)

> `〜前に`는 항상 **사전형(현재형)** — 이미 일어난 일이어도 사전형 사용.  
> ⭕ 日本に来る前に (일본에 오기 전에)  
> ❌ 日本に来た前に (X)

---

## 〜後で — 〜한 후에 / 〜하고 나서

```
동사 た형 + 後で
명사 + の後で
```

**포인트:** た형(과거형)을 쓴다.

- 食べた**後で**、散歩します。 (먹은 후에 산책해요.)
- 仕事が終わった**後で**、一杯飲みましょう。  
  (일이 끝난 후에 한잔합시다.)
- 授業**の後で**、友達と遊びました。 (수업 후에 친구와 놀았어요.)

### 〜後で vs 〜てから

|                 | 〜後で                | 〜てから           |
| --------------- | --------------------- | ------------------ |
| 형태            | た형+後で             | て형+から          |
| 강조            | 완료 후의 상황        | 순서·선후 관계     |
| 자연스러운 상황 | 일정한 시간 간격 있음 | 바로 이어지는 행동 |

- 食べてから、すぐ寝た。 (먹고 나서 바로 잤다.) — てから: 바로 이어짐
- 食べた後で、少し休んだ。 (먹은 후에 잠깐 쉬었다.) — 後で: 약간의 간격

---

## 〜ながら — 〜하면서 (동시 동작)

두 동작을 동시에 할 때. 주로 **부수적인 동작 + 주된 동작** 순서.

```
동사 ます형 어간 + ながら + 주된 동작
```

- 音楽を聴き**ながら**、勉強します。 (음악을 들으면서 공부해요.)
- 歩き**ながら**、スマホを見るのは危（あぶ）ない。  
  (걸으면서 스마트폰을 보는 건 위험해요.)
- コーヒーを飲み**ながら**、話しましょう。 (커피를 마시면서 이야기합시다.)

> **주의:** `ながら`는 동시 행동이 가능한 경우에만 쓴다.  
> ❌ 食べながら、寝た。 (먹으면서 잤다 — 불가능한 조합)

---

## 〜間(に) — 〜하는 동안 (에)

어떤 상태나 행동이 지속되는 시간 안에 다른 일이 일어날 때.

```
동사 て형 + いる間に
名詞 + の間に
```

- 彼女が寝ている**間に**、掃除（そうじ）した。  
  (그녀가 자는 동안에 청소했다.)
- 留学（りゅうがく）している**間に**、友達がたくさんできた。  
  (유학하는 동안 친구가 많이 생겼다.)
- 夏休みの**間に**、旅行したい。 (여름 방학 동안에 여행하고 싶어.)

### 〜間 vs 〜間に

- `〜間` — 그 시간 내내 계속 (상태 지속)
  - 雨が降っている**間**、家にいました。 (비가 오는 동안 내내 집에 있었어요.)
- `〜間に` — 그 시간 내의 어느 시점에 (점 사건)
  - 雨が降っている**間に**、買い物した。 (비 오는 동안에 장을 봤어요. — 그 사이에)

## 시간 관계 표현 정리

| 표현     | 접속                      | 의미                      |
| -------- | ------------------------- | ------------------------- |
| 〜前に   | 사전형 + 前に             | 〜하기 전에               |
| 〜後で   | た형 + 後で               | 〜한 후에                 |
| 〜てから | て형 + から               | 〜하고 나서 (순서 강조)   |
| 〜ながら | ます형 어간 + ながら      | 〜하면서 (동시)           |
| 〜間     | て+いる間 / 名詞+の間     | 〜하는 동안 (내내)        |
| 〜間に   | て+いる間に / 名詞+の間に | 〜하는 동안에 (그 사이에) |

---

## 한눈에 비교

```
A の前に B する       → B를 먼저 하고 A
A た後で B する       → A를 먼저 하고 B
A ながら B する       → A와 B를 동시에
A している間に B する  → A 도중에 B가 발생
```
