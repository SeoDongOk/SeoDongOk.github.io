일본어를 공부하다 보면 「**ます**」를 만나는 순간 모든 게 시작된다. 「行きます, 食べます, します」 — 교과서 첫 장부터 끝까지 따라다니는 그 형태다.

그런데 막상 **마스형의 변형**(과거 부정, 권유, 제안)이나 **마스형↔사전형 전환**으로 가면 한국인 학습자들이 자주 막힌다. 「行きませんかった」(X), 「帰ます」(X), 「ませんか와 ましょうか 차이를 모름」 같은 흔한 실수들.

이 글은 **JLPT N5 학습자를 위한 실전 연습편**이다. N5에서 반드시 잡아야 하는 마스형 변형 3가지를 예문 30개 이상과 미니 퀴즈로 익힌다.

> 📚 마스형의 어원·방언·비즈니스 경어 등 종합적인 내용은 [마스형 완전정복](/blog/2026-04-19-japanese-masu-form-complete-guide) 편을 참고.

## 마스형 4가지 기본 모양 복습

「**飲（の）む**」(마시다)를 예로 든 정중체 4형.

| 시제      | 형태                 | 의미              |
| --------- | -------------------- | ----------------- |
| 현재 긍정 | 飲**みます**         | 마십니다          |
| 현재 부정 | 飲**みません**       | 마시지 않습니다   |
| 과거 긍정 | 飲**みました**       | 마셨습니다        |
| 과거 부정 | 飲**みませんでした** | 마시지 않았습니다 |

이 4형이 정중체(です・ます体) 시제의 뼈대다. 여기서 권유(ましょう), 제안(ませんか/ましょうか) 등이 파생된다.

## 1. 과거 부정 「ませんでした」 마스터

### 핵심 규칙

과거 부정은 **「ません」 뒤에 「でした」를 그대로 붙인다**. 한 덩어리로 외우는 게 가장 빠르다.

- ❌ 飲みませんかった (자주 하는 실수)
- ⭕ 飲み**ません + でした**

한국어 "안 마셨어요"를 직역하려다 「飲まないでした」처럼 쓰는 사람이 많은데, 정중체 부정 과거는 무조건 **「~ませんでした」** 한 덩어리다.

### 동사 그룹별 ます形 만들기 (복습)

#### 1그룹 (五段動詞) — 「u」 → 「i」 + ます

| 사전형       | ます形   | 의미     |
| ------------ | -------- | -------- |
| 行（い）く   | 行きます | 가다     |
| 飲（の）む   | 飲みます | 마시다   |
| 書（か）く   | 書きます | 쓰다     |
| 読（よ）む   | 読みます | 읽다     |
| 買（か）う   | 買います | 사다     |
| 話（はな）す | 話します | 말하다   |
| 待（ま）つ   | 待ちます | 기다리다 |

#### 2그룹 (一段動詞) — 「る」 떼고 + ます

| 사전형       | ます形   | 의미     |
| ------------ | -------- | -------- |
| 食（た）べる | 食べます | 먹다     |
| 見（み）る   | 見ます   | 보다     |
| 寝（ね）る   | 寝ます   | 자다     |
| 起（お）きる | 起きます | 일어나다 |

#### 3그룹 (불규칙) — 통째로 외우기

| 사전형                 | ます形       | 의미     |
| ---------------------- | ------------ | -------- |
| する                   | します       | 하다     |
| 来（く）る             | 来（き）ます | 오다     |
| 勉強（べんきょう）する | 勉強します   | 공부하다 |

### 과거 부정 실전 예문 10개

1. 昨日（きのう）、学校（がっこう）に**行きませんでした**。
   (어제 학교에 가지 않았습니다.)
2. 朝（あさ）ご飯（はん）を**食べませんでした**。
   (아침을 먹지 않았습니다.)
3. テレビを**見ませんでした**。
   (TV를 보지 않았습니다.)
4. 友達（ともだち）に**会（あ）いませんでした**。
   (친구를 만나지 않았습니다.)
5. お酒（さけ）を**飲みませんでした**。
   (술을 마시지 않았습니다.)
6. 宿題（しゅくだい）を**しませんでした**。
   (숙제를 하지 않았습니다.)
7. 電話（でんわ）が**来ませんでした**。
   (전화가 오지 않았습니다.)
8. 本（ほん）を**読みませんでした**。
   (책을 읽지 않았습니다.)
9. 何（なに）も**買いませんでした**。
   (아무것도 사지 않았습니다.)
10. 昨日は**勉強しませんでした**。
    (어제는 공부하지 않았습니다.)

## 2. 「ませんか」 vs 「ましょうか」 — 회화의 핵심 두 표현

이 두 표현은 한국인 학습자가 가장 헷갈려하는 부분이다. 핵심 차이를 한 줄로 잡으면 평생 안 까먹는다.

- **〜ませんか** = "~하지 않으시겠어요?" → **상대방의 의향**을 묻는 권유
- **〜ましょうか** = "~할까요?" → **내가 해주겠다는 제안** 또는 **함께할지 가벼운 제안**

### 「ませんか」 — 부드러운 권유

상대방을 같이 행동에 초대하는 느낌. **부정형으로 묻지만 의미는 "같이 하자"**. 한국어 "안 가실래요?"가 사실은 "가자"는 의미인 것과 같은 감각이다.

#### 예문 5개

1. 一緒（いっしょ）に映画（えいが）を**見ませんか**。
   (같이 영화 보지 않으실래요?)
2. お茶（ちゃ）を**飲みませんか**。
   (차 한잔 마시지 않으실래요?)
3. 週末（しゅうまつ）、公園（こうえん）に**行きませんか**。
   (주말에 공원에 가지 않으실래요?)
4. 一緒に勉強し**ませんか**。
   (같이 공부하지 않으실래요?)
5. 晩（ばん）ご飯（はん）を**食べませんか**。
   (저녁 같이 먹지 않으실래요?)

### 「ましょうか」 — 제안 / 도움 제의

두 가지 의미가 있다.

#### ① 내가 해드릴까요? (도움 제의)

1. 荷物（にもつ）を**持（も）ちましょうか**。
   (짐을 들어드릴까요?)
2. 窓（まど）を**開（あ）けましょうか**。
   (창문을 열까요?)
3. タクシーを**呼（よ）びましょうか**。
   (택시를 부를까요?)

#### ② 같이 할까요? (가벼운 제안 — 상대가 동의할 거라 예상)

4. そろそろ**行きましょうか**。
   (슬슬 갈까요?)
5. コーヒーでも**飲みましょうか**。
   (커피라도 마실까요?)

### 비교표

|             | 〜ませんか           | 〜ましょうか                 |
| ----------- | -------------------- | ---------------------------- |
| 뉘앙스      | 권유 (의향 묻기)     | 제안 / 도움 제의             |
| 한국어      | "~하지 않을래요?"    | "~할까요?"                   |
| 누구 의지   | **상대방** 의향 우선 | **나** 또는 **우리**         |
| 거절 가능성 | 충분히 있음          | 거의 없음 (이미 합의 분위기) |

### 같은 상황 두 표현 비교

같은 "커피 마시자"라도 뉘앙스가 다르다.

- A: コーヒーを**飲みませんか**。 → "마실래요?" (상대 의향 우선, 정중함↑)
- B: コーヒーを**飲みましょうか**。 → "마실까요?" (이미 마실 분위기, 가벼움↑)

처음 만난 사람이나 윗사람한테는 **ませんか**가 더 안전하다. ましょうか는 친근한 사이에서 자연스럽다.

## 3. ます形 ↔ 사전형(辞書形) 전환 훈련

N5의 핵심 중의 핵심이다. 여기서 막히면 N4, N3 가서 「て形」, 「ない形」, 「た形」 다 못 만든다.

### 그룹별 전환 규칙

#### 1그룹: ます形의 「い단」을 「う단」으로

| ます形               | 사전형   | 변화    |
| -------------------- | -------- | ------- |
| 書**き**ます         | 書**く** | き → く |
| 飲**み**ます         | 飲**む** | み → む |
| 行**き**ます         | 行**く** | き → く |
| 読**み**ます         | 読**む** | み → む |
| 買**い**ます         | 買**う** | い → う |
| 話**し**ます         | 話**す** | し → す |
| 待**ち**ます         | 待**つ** | ち → つ |
| 帰（かえ）**り**ます | 帰**る** | り → る |

> 💡 **외우는 팁**: ます 앞 글자가 い단(き、み、し、ち、り、い、ぎ、び、に)이면, 같은 행의 う단(く、む、す、つ、る、う、ぐ、ぶ、ぬ)으로 바꾼다.

#### 2그룹: ます 떼고 「る」 붙이기

| ます形           | 사전형     |
| ---------------- | ---------- |
| 食べます         | 食べ**る** |
| 見ます           | 見**る**   |
| 寝ます           | 寝**る**   |
| 起きます         | 起き**る** |
| 教（おし）えます | 教え**る** |

> 💡 **포인트**: ます 앞 글자가 「**え단**」 또는 일부 「**い단(2그룹)**」이면 보통 2그룹.

#### 3그룹: 외우기

| ます形       | 사전형     |
| ------------ | ---------- |
| します       | する       |
| 来（き）ます | 来（く）る |

### ⚠️ 헷갈리는 함정 — 1그룹? 2그룹?

ます 앞이 「**い단**」인 동사 중에 1그룹도 있고 2그룹도 있다. 외워야 하는 단어들.

#### 1그룹인데 「~ります」로 끝나는 동사 (예외처럼 보이지만 1그룹!)

| ます形               | 사전형   | 의미     |
| -------------------- | -------- | -------- |
| 帰（かえ）**り**ます | 帰**る** | 돌아가다 |
| 走（はし）**り**ます | 走**る** | 달리다   |
| 入（はい）**り**ます | 入**る** | 들어가다 |
| 切（き）**り**ます   | 切**る** | 자르다   |
| 知（し）**り**ます   | 知**る** | 알다     |

> ⚠️ 「帰る」를 2그룹으로 착각해서 「帰ます」로 쓰면 X. 「帰**り**ます」가 정답.

### 미니 퀴즈 (사전형 → ます形)

다음 사전형을 ます形으로 바꿔보자.

## 오늘의 문법 정리 카드

| 항목               | 핵심 한 줄                                          |
| ------------------ | --------------------------------------------------- |
| **ませんでした**   | 과거 부정 = ません + でした (한 덩어리로!)          |
| **ませんか**       | 상대 의향 묻는 정중한 권유 ("~안 하실래요?")        |
| **ましょうか**     | 내가 해줄까? / 같이 할까? (이미 합의 분위기)        |
| **사전형↔ます形** | 1그룹: い단↔う단 / 2그룹: る↔ます / 3그룹: 외우기 |
| **함정 동사**      | 帰る、走る、入る、切る、知る → 1그룹!               |

## 다음 편 예고

마스형의 변형을 잡았다면, 이제 **「て形」**으로 넘어갈 차례다. 「て形」은 일본어의 **만능 연결고리** — 진행형(~ています), 의뢰(~てください), 허가(~てもいいです), 금지(~てはいけません) 등 셀 수 없이 많은 문형의 출발점이다.

다음 편에서는 「て形」의 그룹별 변환 규칙(특히 1그룹의 음편 변화: い·促·撥音便)과 핵심 활용 5가지를 다룬다.
