> Quant의 마지막 구간은 두 변수의 관계를 추정하는 회귀분석과, 금융 데이터 사이언스의 큰 그림이다. 통계가 투자 의사결정으로 연결되는 지점이다.

## 공식 범위 확인

2027 Quant의 Simple Linear Regression 모듈은 단순선형회귀의 설명과 해석, 최소제곱 추정, 회귀 가정, 잔차 분석, 적합도, 회귀계수, ANOVA, 예측값과 예측구간, CAPM 변수 추정을 포함한다. Introduction to Financial Data Science 모듈은 빅데이터, 머신러닝, AI가 금융 데이터 사이언스, 핀테크, 투자관리에서 사용되는 방식을 설명한다.

## 단순선형회귀

단순선형회귀는 하나의 독립변수로 종속변수를 설명하는 모형이다.

```text
Y = b0 + b1X + error
```

- `Y`: 종속변수
- `X`: 독립변수
- `b0`: 절편
- `b1`: 기울기
- `error`: 설명되지 않는 부분

기울기 `b1`은 X가 1단위 변할 때 Y가 평균적으로 얼마나 변하는지를 나타낸다.

## 최소제곱법

회귀계수는 보통 잔차제곱합을 최소화하는 방식으로 추정한다.

```text
Residual = actual Y - predicted Y
```

```text
SSE = sum of squared residuals
```

최소제곱법은 SSE를 가장 작게 만드는 절편과 기울기를 찾는다.

## 회귀 적합도

| 지표                       | 의미                        |
| -------------------------- | --------------------------- |
| R-squared                  | Y 변동 중 X가 설명하는 비율 |
| Standard error of estimate | 예측오차의 전형적 크기      |
| t-statistic                | 개별 회귀계수의 유의성 판단 |
| F-statistic                | 회귀모형 전체의 유의성 판단 |

R-squared가 높다고 항상 좋은 모델은 아니다. 과적합, 경제적 의미, 잔차 패턴을 함께 봐야 한다.

## 회귀 가정과 잔차 분석

단순회귀에서는 잔차가 무작위로 흩어져 있는지 확인해야 한다.

주의할 잔차 패턴:

- 곡선 형태: 선형관계 가정 위반 가능성
- 부채꼴 형태: 등분산성 위반 가능성
- 일정한 방향성: 독립성 문제 가능성
- 극단값: 이상치 영향 가능성

시험에서는 잔차 그림을 보고 어떤 가정이 의심되는지 묻는 문제가 나올 수 있다.

## 예측값과 예측구간

회귀식으로 특정 X에서의 Y 예측값을 구할 수 있다.

```text
Predicted Y = b0 + b1X
```

예측구간은 단일 예측값 주변의 불확실성을 반영한다. 신뢰구간보다 넓은 경우가 일반적이다. 개별 관측값에는 오차항의 불확실성까지 들어가기 때문이다.

## CAPM beta

CAPM 회귀에서는 개별 자산의 초과수익률을 시장 초과수익률로 설명한다.

```text
Ri - Rf = alpha + beta(Rm - Rf) + error
```

여기서 beta는 시장 초과수익률이 1% 변할 때 자산 초과수익률이 평균적으로 얼마나 변하는지를 나타낸다.

| beta     | 해석                 |
| -------- | -------------------- |
| beta > 1 | 시장보다 민감        |
| beta = 1 | 시장과 비슷한 민감도 |
| beta < 1 | 시장보다 덜 민감     |

## 금융 데이터 사이언스

금융 데이터 사이언스는 대규모 데이터, 머신러닝, AI를 이용해 투자 의사결정과 금융 서비스를 개선하는 영역이다.

활용 예:

- 뉴스와 공시 텍스트 분석
- 이상거래 탐지
- 신용위험 예측
- 포트폴리오 리밸런싱 지원
- 고객 행동 분석
- 알고리즘 트레이딩 신호 탐색

중요한 것은 모델이 복잡할수록 데이터 품질, 과적합, 해석가능성, 윤리 문제가 함께 커진다는 점이다.
