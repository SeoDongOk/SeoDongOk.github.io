복합동사는 동사 두 개를 결합해 새로운 의미를 만드는 표현이다. 기본 형태는 `ます형 어간 + 두 번째 동사`.

N4에서 자주 나오는 복합동사 패턴을 모두 정리했다.

---

## 기본 구조

```
동사 ます형 어간 + 복합동사
```

예: 食べます → 食べ + 始める → **食べ始める**

---

## 1. 〜始める — 〜하기 시작하다

어떤 동작이 시작될 때.

- 雨が降り**始めました**。 (비가 내리기 시작했어요.)
- 日本語を勉強し**始めたのは**去年です。 (일본어를 공부하기 시작한 건 작년이에요.)
- 赤ちゃんが泣き**始めた**。 (아기가 울기 시작했다.)
- 春になり、花が咲き**始めました**。 (봄이 되어 꽃이 피기 시작했어요.)

---

## 2. 〜続ける — 계속 〜하다

동작이 끊임없이 이어질 때.

- 3時間歩き**続けました**。 (3시간 계속 걸었어요.)
- 諦めずに頑張り**続けてください**。 (포기하지 않고 계속 힘내 주세요.)
- 雨が降り**続けています**。 (비가 계속 내리고 있어요.)
- 彼女は黙り**続けた**。 (그녀는 계속 침묵했다.)

---

## 3. 〜終わる — 다 〜하다 / 〜가 끝나다

동작이 완료될 때.

- 宿題をし**終わりました**。 (숙제를 다 했어요.)
- 本を読み**終わったら**、貸してください。 (책을 다 읽으면 빌려주세요.)
- レポートを書き**終わりました**。 (리포트를 다 썼어요.)

---

## 4. 〜出す — 갑자기 〜하기 시작하다

예상치 못하게 갑자기 시작될 때.

- 急に雨が降り**出した**。 (갑자기 비가 내리기 시작했다.)
- 子どもが急に走り**出した**。 (아이가 갑자기 뛰기 시작했다.)
- 彼女は突然泣き**出した**。 (그녀가 갑자기 울기 시작했다.)

**始める vs 出す:**

- `始める` → 의도적, 계획적으로 시작
- `出す` → 갑자기, 예상 밖으로 시작

---

## 5. 〜やすい — 〜하기 쉽다 / 〜하기 편하다

| 사용        | 예                                    |
| ----------- | ------------------------------------- |
| 동작이 쉽다 | 使いやすい (사용하기 쉽다)            |
| 경향이 있다 | 風邪をひきやすい (감기에 걸리기 쉽다) |

**예문:**

- このペンは書き**やすい**です。 (이 펜은 쓰기 쉬워요.)
- 彼は怒り**やすい**人です。 (그는 화내기 쉬운 사람이에요 / 성미가 급해요.)
- この道は滑り**やすい**。 (이 길은 미끄럽기 쉬워요 / 미끄럽기 쉬워요.)
- 太り**やすい**体質です。 (살찌기 쉬운 체질이에요.)

---

## 6. 〜にくい — 〜하기 어렵다 / 〜하기 불편하다

- この字は読み**にくい**です。 (이 글자는 읽기 어려워요.)
- このコートは動き**にくい**です。 (이 코트는 움직이기 불편해요.)
- 彼に話し**にくい**雰囲気がある。 (그에게는 말하기 어려운 분위기가 있다.)
- 細かい作業はし**にくい**。 (세밀한 작업은 하기 어렵다.)

---

## 7. 〜すぎる — 너무 〜하다 (과도)

정도가 지나칠 때.

```
동사 ます형 어간 + すぎる
い형 어간 + すぎる
な형 어간 + すぎる
```

- 食べ**すぎました**。 (너무 먹었어요.)
- 高**すぎます**。 (너무 비싸요.)
- 簡単**すぎる**。 (너무 쉽다.)
- 働き**すぎないで**ください。 (너무 일하지 마세요.)
- お酒を飲み**すぎた**。 (술을 너무 마셨다.)

---

## 8. 기타 자주 쓰는 복합동사

| 복합동사 | 의미          | 예                         |
| -------- | ------------- | -------------------------- |
| 〜忘れる | 깜빡 잊다     | 宿題を持ってき忘れた       |
| 〜直す   | 다시 〜하다   | 書き直す (다시 쓰다)       |
| 〜合う   | 서로 〜하다   | 話し合う (서로 이야기하다) |
| 〜込む   | 안으로 〜하다 | 飛び込む (뛰어들다)        |
| 〜返す   | 되돌려 〜하다 | 読み返す (다시 읽다)       |

---

## 핵심 요약

| 복합동사 | 의미            | 예         |
| -------- | --------------- | ---------- |
| 〜始める | 시작하다        | 食べ始める |
| 〜続ける | 계속하다        | 走り続ける |
| 〜終わる | 끝내다/다하다   | 読み終わる |
| 〜出す   | 갑자기 시작하다 | 泣き出す   |
| 〜やすい | 하기 쉽다       | 使いやすい |
| 〜にくい | 하기 어렵다     | 読みにくい |
| 〜すぎる | 너무 〜하다     | 食べすぎる |
