"비가 올 것 같다", "들었는데 〜래", "〜인 것 같다" — 이런 추량·전문 표현은 일본어에서 가장 헷갈리는 문법 중 하나다.

핵심은 **정보의 출처가 어디냐**로 구분하는 것이다.

---

## 한눈에 비교

| 표현            | 정보 출처                  | 의미                  |
| --------------- | -------------------------- | --------------------- |
| 〜らしい        | 외부 정보 (들었다, 읽었다) | 〜인 것 같다/〜래     |
| 〜そうだ (전문) | 외부 정보 그대로 전달      | 〜라고 한다           |
| 〜そうだ (양태) | 직접 보고 느낌             | 〜할 것 같다/〜보인다 |
| 〜ようだ        | 근거를 바탕으로 추측       | 〜인 것 같다          |
| 〜みたいだ      | ようだ의 회화체            | 〜인 것 같아          |

---

## 1. 〜らしい — 외부 정보 바탕 추측

뉴스·소문·간접 정보를 바탕으로 "〜인 것 같다"를 말할 때. 화자가 직접 보지 않은 정보.

**접속:**

- 동사 보통형 + らしい
- い형용사 + らしい
- な형용사 어간 + らしい
- 명사 + らしい

**예문:**

- 明日、雨が降る**らしい**。 (내일 비가 올 것 같아. — 일기예보에서)
- 彼女は結婚した**らしい**。 (그녀는 결혼한 것 같아. — 소문으로)
- あの店はおいしい**らしい**よ。 (그 가게는 맛있는 것 같아. — 들었는데)
- 田中さんは今日来ない**らしい**。 (다나카 씨는 오늘 안 오는 것 같아.)

> **らしい의 또 다른 용법:** 명사에 붙어 "〜답다"는 뜻도 있다.  
> 子ども**らしい** 子ども (아이다운 아이)  
> 男**らしい** (남자답다)

---

## 2. 〜そうだ (伝聞・전문) — 그대로 전달

외부 정보를 **그대로 전달**할 때. 뒤에 `です`를 붙인다.

**접속:** 동사·형용사 보통형 + そうだ

- 天気予報によると、明日は雨だ**そうです**。  
  (일기예보에 따르면 내일은 비래요.)
- ニュースで、事故があった**そうです**。 (뉴스에서 사고가 있었다고 해요.)
- 彼女は来ない**そうです**。 (그녀는 안 온대요.)

**らしい vs そうだ(전문) 차이:**

- `らしい` → 화자의 추측이 섞임
- `そうだ(전문)` → 정보를 그대로 전달 (뉴스·전달자 느낌)

---

## 3. 〜そうだ (様態・양태) — 보고 느끼는 추측

직접 보거나 느껴서 "〜할 것 같다"고 말할 때. 전문 そうだ와 **접속이 다르다**.

**접속:** 동사 ます형 어간 + そう / い형 어간 + そう / な형 어간 + そう

|        | 보통형 | 양태 そうだ  |
| ------ | ------ | ------------ |
| 食べる | 食べる | 食べ**そう** |
| 高い   | 高い   | 高**そう**   |
| 元気だ | 元気だ | 元気**そう** |

**예문:**

- あの雲、雨が降り**そうです**。 (저 구름, 비가 올 것 같아요. — 직접 봄)
- このケーキ、おいし**そう**！ (이 케이크, 맛있겠다! — 눈으로 봄)
- 彼女は元気**そうです**。 (그녀는 건강해 보여요.)
- 危な**そう**だから、やめよう。 (위험할 것 같으니까 그만두자.)

> **예외 주의:** いい → よ**さそう** (いそう가 아님)  
> ない → な**さそう** (なそう가 아님)

---

## 4. 〜ようだ / みたいだ — 근거 있는 추측

**직접적인 근거(보이는 것·들리는 것)**를 바탕으로 추측할 때.

**ようだ** = 문어체·격식  
**みたいだ** = 구어체·일상

**접속:**

- 동사·형용사 보통형 + ようだ/みたいだ
- 명사 + の + ようだ / 명사 + みたいだ

**예문:**

- 外が騒がしい。何かあった**ようだ**。  
  (밖이 시끄럽다. 뭔가 있는 것 같다. — 소리 들림)
- 彼は疲れている**みたいだ**。 (그는 피곤한 것 같아. — 보임)
- まるで夢の**ようだ**。 (마치 꿈 같다.)
- 彼女は病気**みたいです**。 (그녀는 아픈 것 같아요.)

**ようだ vs らしい:**

- `ようだ` → 직접 관찰한 근거
- `らしい` → 간접 정보 (들었다, 읽었다)

---

## 전체 비교 요약

|                | 접속              | 예             |
| -------------- | ----------------- | -------------- |
| 〜らしい       | 보통형 + らしい   | 雨が降るらしい |
| 〜そうだ(전문) | 보통형 + そうだ   | 雨だそうだ     |
| 〜そうだ(양태) | 어간 + そう       | 降りそう       |
| 〜ようだ       | 보통형 + ようだ   | 降るようだ     |
| 〜みたいだ     | 보통형 + みたいだ | 降るみたいだ   |

**시험 포인트:** `양태 そうだ`는 어간에 접속, 나머지는 보통형에 접속!
