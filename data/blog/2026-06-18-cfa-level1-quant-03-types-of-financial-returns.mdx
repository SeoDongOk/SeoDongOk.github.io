> 수익률은 하나가 아니다. 같은 투자 성과도 어떤 방식으로 계산하느냐에 따라 숫자가 달라진다. CFA Level I Quant에서는 수익률 종류를 계산하고 비교하며 해석할 수 있어야 한다.

## 공식 범위 확인

2027 Topic Outlines의 Types of Financial Returns 모듈은 금융자산, 금융상품, 지표의 다양한 수익률을 계산·비교·해석하는 범위를 다룬다.

## 단순수익률

가장 직관적인 수익률은 단순수익률이다.

```text
R = (P1 - P0 + CF) / P0
```

- `P0`: 기초 가격
- `P1`: 기말 가격
- `CF`: 기간 중 받은 현금흐름

단순수익률은 한 기간의 성과를 설명하기 쉽다. 하지만 여러 기간을 연결할 때는 단순히 더하면 안 된다.

## 로그수익률

로그수익률은 가격 비율의 자연로그다.

```text
Log return = ln(P1 / P0)
```

로그수익률은 연속복리와 연결되고, 여러 기간을 더해서 누적 로그수익률을 구할 수 있다는 장점이 있다. 다만 Level I에서는 로그수익률의 해석과 단순수익률과의 차이를 이해하는 정도가 중요하다.

## 산술평균수익률

산술평균은 기간별 수익률을 더해 기간 수로 나눈다.

```text
Arithmetic mean = (R1 + R2 + ... + Rn) / n
```

산술평균은 “한 기간의 기대수익률”을 추정할 때 유용하다. 하지만 실제 장기 누적 성과를 보여주지는 않는다.

## 기하평균수익률

기하평균은 복리 성장률을 보여준다.

```text
Geometric mean = [(1 + R1)(1 + R2)...(1 + Rn)]^(1/n) - 1
```

장기 투자 성과를 설명할 때는 기하평균이 더 적절하다. 변동성이 있으면 산술평균은 보통 기하평균보다 크거나 같다.

## 예시

1년차 수익률이 +20%, 2년차 수익률이 -10%라고 하자.

```text
산술평균 = (20% - 10%) / 2 = 5%
누적수익률 = 1.20 × 0.90 - 1 = 8%
기하평균 = sqrt(1.08) - 1 = 약 3.92%
```

산술평균 5%를 2년 동안 복리로 얻은 것이 아니다. 실제 복리 성장률은 약 3.92%다.

## Money return과 percentage return

금액 기준 수익과 비율 기준 수익도 구분해야 한다.

| 구분              | 설명                           |
| ----------------- | ------------------------------ |
| Money return      | 원화, 달러 같은 금액 기준 이익 |
| Percentage return | 투자원금 대비 비율 성과        |

100만 원을 투자해 10만 원을 벌면 10%지만, 1억 원을 투자해 10만 원을 벌면 0.1%다. 투자 성과 비교에는 비율 수익률이 더 적절하다.

## 오답 포인트

| 상황             | 주의점                                       |
| ---------------- | -------------------------------------------- |
| 여러 기간 수익률 | 단순 합산이 아니라 gross return 곱셈         |
| 장기 성과 비교   | 산술평균보다 기하평균 확인                   |
| 변동성 큰 수익률 | 산술평균과 기하평균 차이가 커짐              |
| 금액 수익 비교   | 투자원금이 다르면 percentage return으로 비교 |

## 체크리스트

- [ ] 단순수익률과 로그수익률을 구분할 수 있다.
- [ ] 산술평균과 기하평균의 용도를 구분할 수 있다.
- [ ] 여러 기간 수익률을 gross return으로 연결할 수 있다.
- [ ] 장기 성과에는 기하평균이 더 적절한 이유를 설명할 수 있다.
