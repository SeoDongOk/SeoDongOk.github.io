N4에서 가장 헷갈리는 문법이 **가정형 4가지**다. 전부 "〜하면"으로 번역되지만 쓰임이 다르다.

한 번에 다 외우려 하지 말고, 각각의 핵심 뉘앙스를 먼저 잡자.

---

## 4가지 한눈에 비교

| 가정형 | 핵심 뉘앙스      | 예                   |
| ------ | ---------------- | -------------------- |
| 〜ば   | 조건·일반 법칙   | 春になれば暖かい     |
| 〜たら | 그 일이 완료되면 | 家に帰ったら電話して |
| 〜と   | 자연 결과·법칙   | 右に曲がると駅がある |
| 〜なら | 상대방 말 전제   | 日本に行くなら京都へ |

---

## 1. 〜ば — 조건이 맞으면

### 만들기

**동사:** え단 + ば (1그룹), る → れば (2그룹)

| 그룹  | 동사   | ば형     |
| ----- | ------ | -------- |
| 1그룹 | 書く   | 書けば   |
| 1그룹 | 飲む   | 飲めば   |
| 1그룹 | 帰る   | 帰れば   |
| 2그룹 | 食べる | 食べれば |
| 2그룹 | 起きる | 起きれば |
| 3그룹 | する   | すれば   |
| 3그룹 | くる   | くれば   |

**い형용사:** い → ければ

- 安い → 安ければ
- 高い → 高ければ

**な형용사·명사:** 〜であれば / 〜ならば

### 사용

- お金が**あれば**、旅行したい。 (돈이 있으면 여행하고 싶다.)
- 春に**なれば**、桜が咲く。 (봄이 되면 벚꽃이 핀다. — 일반 법칙)
- もっと早く来**れば**よかった。 (더 일찍 왔으면 좋았을 텐데. — 후회)

### 주의: ば + 의지·명령은 어색

❌ 早く起きれば、行きましょう。  
⭕ 早く起きたら、行きましょう。

---

## 2. 〜たら — 그것이 완료되면

た형 + ら. **순서가 있는 상황**, 특정 사건 후에 다른 일이 일어날 때.

- 家に帰っ**たら**、電話してください。 (집에 돌아가면 전화해 주세요.)
- 宿題が終わっ**たら**、遊んでいいよ。 (숙제 끝나면 놀아도 돼.)
- もし宝くじが当たっ**たら**、どうする? (만약 복권 당첨되면 어떻게 할 거야?)
- 日本に行っ**たら**、ラーメンを食べた。 (일본에 갔더니 라면을 먹었다. — 과거 결과)

**たら만 가능한 패턴:**

- **〜たら + 의지·명령·희망** → OK
  - 終わったら来てください。 (끝나면 와 주세요.)
- **〜たら + 過去の発見** → OK
  - 開けたら、猫がいた。 (열었더니 고양이가 있었다.)

---

## 3. 〜と — 자연스럽게 결과가 따라온다

항상 그 결과가 일어나는 **자연·법칙·기계적 결과**에 쓴다. 뒤에 의지·명령·희망은 불가.

```
동사 사전형 / ない형 + と
```

- 春に**なると**、桜が咲く。 (봄이 되면 벚꽃이 핀다. — 자연법칙)
- 右に曲がる**と**、駅があります。 (오른쪽으로 꺾으면 역이 있어요. — 길 안내)
- このボタンを押す**と**、ドアが開く。 (이 버튼을 누르면 문이 열린다. — 기계)
- 0に何をかける**と**、0になる。 (0에 무엇을 곱하면 0이 된다. — 수학)

**と + 의지·명령·희망은 ❌**

❌ 駅に着くと、電話してください。 → 「着いたら」를 써야 함  
⭕ 駅に着い**たら**、電話してください。

---

## 4. 〜なら — 그것이 사실이라면 / 상대방 말 전제

상대방이 말한 것, 또는 이미 알려진 사실을 전제로 할 때.

```
명사 / 동사 사전형 · た형 + なら
```

- 日本に行く**なら**、京都がいいですよ。  
  (일본에 간다면 교토가 좋아요. — "간다고 하니까" 전제)
- 田中さんが来る**なら**、私も行く。  
  (다나카 씨가 온다면 나도 갈게. — 조건 전제)
- 安い**なら**、買う。 (싸다면 살게.)
- 学生**なら**、割引があります。 (학생이라면 할인이 있어요.)

**포인트:** 상대방이 먼저 말한 내용에 반응할 때 자주 나온다.

> A: 日本語を勉強したい。  
> B: 日本語を勉強する**なら**、この本がいいよ。

---

## 4가지 선택 가이드

```
자연·법칙·기계적 결과   → と
완료 후 결과·순서       → たら
조건·반사실 가정       → ば
상대방 말 전제         → なら
```

### 실전 헷갈리는 예

| 문장                    | 맞는 가정형 | 이유           |
| ----------------------- | ----------- | -------------- |
| 집에 오면 전화해        | たら        | 순서, 완료 후  |
| 봄이 되면 따뜻해진다    | と / ば     | 자연법칙       |
| 한국에 간다면 서울 가봐 | なら        | 상대방 말 전제 |
| 돈이 있으면 가고 싶다   | ば / たら   | 가정 조건      |
