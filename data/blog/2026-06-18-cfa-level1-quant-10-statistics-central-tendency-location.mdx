> 수익률 데이터가 주어졌을 때 가장 먼저 묻는 질문은 “대략 어디에 모여 있는가?”다. 중심경향과 위치 측도는 데이터를 한눈에 요약하는 첫 번째 도구다.

## 공식 범위 확인

2027 Quant의 Statistical Characteristics 모듈은 중심경향, 위치, 분산을 계산·해석·평가하는 능력을 요구한다.

## 산술평균

산술평균은 모든 관측값을 더해 관측 수로 나눈 값이다.

```text
Mean = (x1 + x2 + ... + xn) / n
```

장점은 모든 데이터를 반영한다는 것이다. 단점은 극단값에 민감하다는 것이다.

## 가중평균

가중평균은 각 값에 중요도 또는 비중을 반영한다.

```text
Weighted mean = w1x1 + w2x2 + ... + wnxn
```

포트폴리오 기대수익률은 대표적인 가중평균이다.

## 중앙값

중앙값은 데이터를 정렬했을 때 가운데 위치한 값이다. 극단값에 덜 민감하다.

예를 들어 수익률이 다음과 같다고 하자.

```text
-5%, 2%, 3%, 4%, 50%
```

평균은 극단적인 50%의 영향을 크게 받지만, 중앙값은 3%다.

## 최빈값

최빈값은 가장 자주 등장하는 값이다. 범주형 데이터나 반복값이 많은 데이터에서 유용하다. 금융수익률처럼 연속형 데이터에서는 뚜렷한 최빈값이 없을 수도 있다.

## 분위수와 위치

위치 측도는 데이터 안에서 특정 값이 어느 위치에 있는지 보여준다.

| 측도       | 의미             |
| ---------- | ---------------- |
| Quartile   | 데이터를 4등분   |
| Quintile   | 데이터를 5등분   |
| Decile     | 데이터를 10등분  |
| Percentile | 데이터를 100등분 |

예를 들어 어떤 펀드가 90th percentile에 있다면 비교 그룹에서 상위권 성과를 냈다고 해석할 수 있다. 단, percentile의 정의는 문맥에 따라 “값 이하의 비율”을 의미하는 경우가 많으므로 문제 표현을 확인해야 한다.

## Trimmed mean과 Winsorized mean

극단값 영향을 줄이기 위한 평균도 있다.

| 측도            | 방식                                        |
| --------------- | ------------------------------------------- |
| Trimmed mean    | 양쪽 극단값 일부를 제거하고 평균 계산       |
| Winsorized mean | 극단값을 특정 경계값으로 대체하고 평균 계산 |

수익률 데이터에 이상치가 있을 때 평균 왜곡을 줄이는 데 유용하다.

## 오답 포인트

| 오답 포인트                 | 체크 방법                                 |
| --------------------------- | ----------------------------------------- |
| 평균과 중앙값 혼동          | 극단값이 있으면 중앙값이 더 안정적        |
| 가중평균에서 비중 합계 오류 | 가중치 합이 1인지 확인                    |
| 분위수 해석 오류            | 높은 percentile이 좋은 성과인지 문맥 확인 |
| 표본 수가 짝수인 중앙값     | 가운데 두 값의 평균 사용                  |

## 체크리스트

- [ ] 산술평균, 가중평균, 중앙값, 최빈값을 구분할 수 있다.
- [ ] 극단값이 평균에 미치는 영향을 설명할 수 있다.
- [ ] 분위수와 percentile을 해석할 수 있다.
- [ ] trimmed mean과 winsorized mean의 차이를 설명할 수 있다.
