> CFA Level 1 준비를 시작하면서 가장 먼저 해야 할 일은 “무엇을 공부할까”보다 **시험이 어떤 형태로 나오는지**를 아는 것이다. 구조를 알아야 공부 방식도 결정된다.

## 시험 형식 요약

CFA Institute의 Level I 안내를 기준으로 정리하면 다음과 같다.

| 항목             | 내용           |
| ---------------- | -------------- |
| 문항 수          | 객관식 180문항 |
| 세션             | 2개 세션       |
| 세션별 시간      | 약 135분       |
| 세션별 문항      | 90문항         |
| 문항당 평균 시간 | 약 90초        |
| 선택지           | A, B, C 세 개  |
| 오답 감점        | 없음           |

숫자로 보면 단순하다. 하지만 180문항을 끝까지 집중해서 푸는 것은 체력전이다. 특히 Level I은 범위가 넓기 때문에, **모든 과목을 얕게라도 놓치지 않는 전략**이 필요하다.

## 90초 룰

문항당 90초는 생각보다 짧다.

- 문제 읽기: 15~25초
- 풀이 방향 결정: 10~20초
- 계산 또는 판단: 30~45초
- 선택지 확인: 5~10초

복잡한 계산 문제에서 2분 이상 쓰기 시작하면, 다른 쉬운 문제의 시간을 가져오게 된다. 그래서 실전에서는 세 가지 판단이 필요하다.

1. 바로 풀 문제
2. 표시해두고 나중에 풀 문제
3. 추측하고 넘어갈 문제

오답 감점이 없기 때문에 빈칸을 남기는 것은 최악이다. 모르는 문제도 반드시 답을 선택해야 한다.

## 문제 문장 스타일에 익숙해지기

Level I 문제는 단순 암기만 묻지 않는다. “가장 적절한 것”, “가장 가능성이 높은 것”, “가장 정확한 설명” 같은 표현으로 개념의 경계를 묻는다.

그래서 공부할 때는 다음처럼 정리한다.

| 개념 정리 방식 | 예시                        |
| -------------- | --------------------------- |
| 정의           | 이 개념이 무엇인가          |
| 조건           | 언제 적용되는가             |
| 반례           | 언제 적용되지 않는가        |
| 비교           | 비슷한 개념과 무엇이 다른가 |
| 계산           | 숫자로 어떻게 표현되는가    |

특히 Ethics와 FSA는 문제 문장의 작은 표현 차이가 정답을 바꾼다.

## 시험 전 필수 준비물

공식 안내에서 특히 중요한 준비물은 다음이다.

- 유효한 국제 여행 여권
- 승인된 계산기
  - Texas Instruments BA II Plus
  - Hewlett Packard 12C
- 시험 예약 확인
- 시험장 위치 확인
- 입실 시간 확인

계산기는 단순히 구매했다고 끝이 아니다. TVM, NPV, IRR, 통계 기능을 손에 익혀야 한다.

## PSM 체크

CFA Program은 각 레벨에서 Practical Skills Module, 즉 PSM을 완료해야 결과를 받을 수 있도록 안내한다. 따라서 “시험만 보면 끝”이 아니라, PSM 완료 여부도 일정에 넣어야 한다.

내 계획은 다음과 같다.

- 2026년 12월까지 PSM 후보 확인
- 2027년 1월 Mock 기간 전에 PSM 완료
- 시험 직전에는 시험 공부에만 집중

PSM을 2월에 미루면 시험 직전 루틴을 흐트러뜨릴 수 있다.

## 예약과 일정 관리

CFA 시험은 보통 2월, 5월, 8월, 11월 창으로 운영된다. 단, 실제 응시 가능일과 등록 마감일은 레벨과 기간별로 달라질 수 있으므로 [Dates & Fees](https://www.cfainstitute.org/programs/cfa-program/dates-fees)에서 반드시 최종 확인해야 한다.

내가 별도로 관리할 일정은 다음이다.

| 체크 항목          | 목표 시점      |
| ------------------ | -------------- |
| 등록 상태 확인     | 가능한 즉시    |
| 시험 예약          | 등록 후 즉시   |
| 여권 유효기간 확인 | 2026년 7월 안  |
| 계산기 구매        | 2026년 7월 안  |
| PSM 완료           | 2027년 1월 전  |
| Mock Exam          | 2027년 1월부터 |

## 실전 시뮬레이션 기준

Mock을 볼 때는 단순히 점수만 기록하지 않는다.

```text
세션 1: 135분 / 90문항
휴식: 실제 시험처럼 짧게
세션 2: 135분 / 90문항
채점: 즉시
분석: 과목별, 오답 유형별 분류
```

Mock을 쪼개서 풀면 체력과 집중력 변수를 확인하기 어렵다. 최소한 시험 한 달 전부터는 전체 세션을 한 번에 보는 연습이 필요하다.

## 나의 체크리스트

- [ ] 2027년 2월 시험 등록 및 예약 완료
- [ ] 여권 정보 확인
- [ ] 계산기 구매 및 초기화 방법 숙지
- [ ] 공식 커리큘럼/학습 도구 접근 확인
- [ ] 과목별 비중표 출력 또는 노트화
- [ ] 주간 공부시간 기록표 생성
- [ ] PSM 완료 계획 수립
- [ ] Mock Exam 일정 최소 2회 확보

이 체크리스트는 공부를 시작하기 위한 최소 안전장치다. 준비의 핵심은 많은 자료를 모으는 것이 아니라, **시험 구조에 맞춰 반복 가능한 루틴을 만드는 것**이다.
