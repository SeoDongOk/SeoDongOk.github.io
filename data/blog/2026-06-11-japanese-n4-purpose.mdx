"공부하기 위해", "잊지 않도록", "말할 수 있게 됐다" — 목적·변화·노력을 나타내는 표현들을 정리한다.

---

## 1. 〜ために — 〜을 위해 (목적)

의지·목적이 있는 행동에 쓴다. 주어가 **의지를 가지고 행동**할 때.

```
동사 사전형 + ために
명사 + のために
```

- 日本語を勉強する**ために**、毎日練習しています。  
  (일본어를 공부하기 위해 매일 연습하고 있어요.)
- 健康の**ために**、走っています。 (건강을 위해 달리고 있어요.)
- 夢を叶える**ために**、努力します。 (꿈을 이루기 위해 노력할게요.)
- 家族の**ために**、働いています。 (가족을 위해 일하고 있어요.)

**ために 뒤에는 의지·의도 표현이 온다.**

---

## 2. 〜ように — 〜하도록 (목표·희망)

목표 상태·상황이 되도록 행동할 때. 뒤에 노력·요청·기도 등이 온다.

```
동사 보통형(가능형 포함) + ように
```

- 漢字が書ける**ように**、練習しています。  
  (한자를 쓸 수 있도록 연습하고 있어요.)
- 遅刻しない**ように**、早く起きます。  
  (지각하지 않도록 일찍 일어나요.)
- 忘れない**ように**、メモしてください。  
  (잊지 않도록 메모해 주세요.)
- うまく話せる**ように**なりたい。 (잘 말할 수 있게 되고 싶다.)

### ために vs ように 차이

|           | ために                     | ように                     |
| --------- | -------------------------- | -------------------------- |
| 뉘앙스    | 명확한 목적·의지           | 목표 상태가 되도록         |
| 뒤의 표현 | 의지·의도                  | 노력·요청·기원             |
| 접속      | 사전형 + ために            | 보통형 + ように            |
| 예        | 試験に受かるために勉強する | 試験に受かるように勉強する |

> `ために`는 목적이 명확하고 직접적, `ように`는 "그렇게 되도록" 노력·기원하는 뉘앙스.

---

## 3. 〜ようになる — 〜하게 됐다 (변화)

이전에는 못 했지만 **지금은 할 수 있게 된** 변화를 나타낸다.

```
동사 보통형 + ようになる
```

- 日本語が話せる**ようになりました**。  
  (일본어를 말할 수 있게 됐어요.)
- 辛い食べ物が食べられる**ようになりました**。  
  (매운 음식을 먹을 수 있게 됐어요.)
- 毎日運動する**ようになりました**。 (매일 운동하게 됐어요.)
- 子どもが一人で寝られる**ようになりました**。  
  (아이가 혼자 잘 수 있게 됐어요.)

**부정 변화:** 〜なくなる (〜하지 않게 됐다)

- 最近、テレビを見**なくなりました**。 (요즘 TV를 안 보게 됐어요.)

---

## 4. 〜ようにする — 〜하도록 하다 (노력·습관)

의식적으로 **노력하거나 습관을 만드는** 표현.

```
동사 보통형 + ようにする / ようにしている
```

- 毎日野菜を食べる**ようにしています**。  
  (매일 채소를 먹도록 하고 있어요.)
- 夜は早く寝る**ようにしています**。 (밤에 일찍 자도록 하고 있어요.)
- なるべく歩く**ようにします**。 (가능한 한 걷도록 할게요.)
- 遅刻しない**ようにしています**。 (지각하지 않도록 하고 있어요.)

### ようになる vs ようにする

|      | ようになる         | ようにする                |
| ---- | ------------------ | ------------------------- |
| 의미 | 〜하게 됐다 (결과) | 〜하도록 한다 (과정·노력) |
| 주체 | 자연스러운 변화    | 의식적인 노력             |
| 예   | 話せるようになった | 話せるようにしている      |

---

## 관련 표현: 〜ために vs 〜のに

|      | ために           | のに                     |
| ---- | ---------------- | ------------------------ |
| 의미 | 목적 (의지 있음) | 필요 조건·비판           |
| 예   | 勉強するために   | 合格するのに時間がかかる |
| 뒤에 | 의지             | 사실 전달                |

- 日本語を話す**ために**、毎日練習している。 (목적: 말하기 위해)
- 日本語を話す**のに**、10年かかった。 (필요: 말하는 데 10년 걸렸다)

---

## 핵심 요약

| 표현         | 의미                 | 예                   |
| ------------ | -------------------- | -------------------- |
| 〜ために     | 〜을 위해 (목적)     | 勉強するために       |
| 〜ように     | 〜하도록 (목표)      | 忘れないように       |
| 〜ようになる | 〜하게 됐다 (변화)   | 話せるようになった   |
| 〜ようにする | 〜하도록 하다 (노력) | 早起きするようにした |
