> 목표는 단순하다. **2027년 2월 CFA Level 1 합격권에 들어갈 만큼, 2026년 6월부터 시험 직전까지 300시간 이상의 누적 공부량을 만드는 것.** 이 글은 그 첫 번째 공개 로드맵이다.

## 기준 정보

CFA Institute의 Level I 안내에 따르면 Level I은 객관식 180문항, 135분 세션 2개로 구성된다. 또한 합격자들은 각 레벨 준비에 평균 300시간 이상을 사용한다고 안내되어 있다. 정확한 시험 예약일과 데드라인은 [CFA Institute Dates & Fees](https://www.cfainstitute.org/programs/cfa-program/dates-fees)에서 최종 확인한다.

내 준비 기준일은 **2026년 6월 17일**이고, 목표 응시 창은 **2027년 2월**이다. 시험일은 아직 개인 예약을 확정하지 않았으므로, 이 로드맵에서는 2월 말까지 약 8개월을 준비 기간으로 잡는다.

## 총 공부시간 목표

| 구분           |         목표 |
| -------------- | -----------: |
| 최소 누적 시간 |      300시간 |
| 안전 목표      |      350시간 |
| 주간 평균      |     9~11시간 |
| 평일           |   하루 1시간 |
| 주말           | 하루 2~3시간 |

300시간은 감으로는 크지만, 주간 단위로 쪼개면 관리 가능한 숫자가 된다.

- 30주 × 10시간 = 300시간
- 35주 × 10시간 = 350시간
- 평일 5시간 + 주말 5시간 = 주 10시간

즉, 핵심은 하루에 몰아서 하는 것이 아니라 **누적 시간을 끊기지 않게 기록하는 것**이다.

## 전체 기간을 5단계로 나누기

### 1단계: 2026년 6월~7월 — 구조 파악과 기초 세팅

목표는 완벽한 이해가 아니다. 커리큘럼의 지도를 머릿속에 넣는 것이다.

- 시험 구조 정리
- 10개 과목 비중 파악
- 계산기 구매 및 기본 조작 익히기
- Learning Ecosystem 또는 사용할 QBank 환경 세팅
- Quant, FSA, Ethics부터 첫 회독 시작

이 기간에는 진도를 빨리 빼기보다, CFA식 문장과 문제 스타일에 익숙해지는 데 집중한다.

### 2단계: 2026년 8월~10월 — 1회독 완주

1회독의 목표는 **모르는 것을 확인하는 것**이다.

- 모든 과목을 한 번씩 훑는다.
- 각 과목마다 핵심 공식과 키워드를 기록한다.
- 문제는 정답률보다 풀이 흐름을 중시한다.
- FSA, Fixed Income, Quant처럼 누적형 과목은 다시 풀 문제를 표시한다.

이 시기에는 완벽주의가 가장 위험하다. 한 과목에서 오래 막히면 전체 커리큘럼을 끝내지 못한다.

### 3단계: 2026년 11월~12월 — 문제 중심 2회독

2회독부터는 읽는 공부보다 푸는 공부가 중심이다.

- 과목별 QBank 반복
- 오답노트 작성
- Ethics 사례형 문장에 익숙해지기
- 계산 문제 풀이 속도 개선
- 약점 과목별 2차 요약본 만들기

이때부터는 **정답률을 숫자로 추적**한다. 감으로 “이해했다”고 느끼는 것과 실제 문제에서 맞히는 것은 다르다.

### 4단계: 2027년 1월 — Mock Exam 모드

1월은 실전 모드로 바꾼다.

- 최소 2회 이상 Mock Exam
- 세션별 시간 관리 연습
- 90초 안에 풀 문제와 넘길 문제 구분
- 과목별 점수 편차 확인
- 공식 암기보다 오답 패턴 제거

Mock을 본 뒤에는 점수 자체보다 **왜 틀렸는지의 유형**을 분류한다.

| 오답 유형      | 대응                        |
| -------------- | --------------------------- |
| 개념 미이해    | 해당 Learning Module 재학습 |
| 공식 암기 부족 | 공식 카드 재정리            |
| 계산 실수      | 계산기 입력 루틴 고정       |
| 문제 문장 오독 | 키워드 표시 훈련            |
| 시간 부족      | 버릴 문제 판단 기준 만들기  |

### 5단계: 2027년 2월 — 최종 압축과 컨디션 관리

시험 직전에는 새로운 내용을 많이 넣기보다, 이미 봤던 내용을 잃지 않는 것이 중요하다.

- Ethics 매일 짧게 반복
- FSA와 Fixed Income 핵심 공식 재확인
- Mock 오답 재풀이
- 계산기와 여권 체크
- PSM 완료 여부 확인
- 시험장 위치와 입실 시간 확인

2월에는 공부량보다 실전 안정성이 중요해진다.

## 과목별 큰 전략

| 우선순위 | 과목                 | 이유                                        |
| -------- | -------------------- | ------------------------------------------- |
| 1        | Ethics               | 비중이 가장 크고, 마지막까지 감각 유지 필요 |
| 1        | FSA                  | 범위가 넓고 누적 이해가 필요                |
| 1        | Fixed Income         | 공식과 개념이 함께 필요                     |
| 2        | Equity               | 직관적이지만 밸류에이션은 반복 필요         |
| 2        | Quant                | 초반에 잡아야 다른 과목 이해가 편함         |
| 2        | Portfolio Management | Quant와 연결됨                              |
| 3        | Economics            | 범위가 넓어 요약 중심 필요                  |
| 3        | Corporate Issuers    | 재무관리식 계산과 개념 병행                 |
| 3        | Derivatives          | 비중은 낮지만 개념이 낯설 수 있음           |
| 3        | Alternatives         | 암기와 개념 비교 중심                       |

## 내가 사용할 주간 루틴

```text
월: Ethics 1시간
화: FSA 1시간
수: Quant 또는 Fixed Income 1시간
목: Equity 또는 Portfolio 1시간
금: 오답 복습 1시간
토: 신규 진도 2~3시간
일: QBank + 주간 회고 2시간
```

주간 회고에서는 세 가지만 기록한다.

1. 이번 주 누적 시간
2. 이번 주 가장 약했던 개념
3. 다음 주 반드시 끝낼 범위

## 블로그에 남길 기록 방식

CFA 준비는 장기전이라서, 머릿속으로만 하면 흐려진다. 그래서 블로그에는 세 종류의 글을 남길 예정이다.

- **개념 정리**: 과목별 핵심 개념을 내 언어로 정리
- **공부 로그**: 주간 공부 시간과 오답 패턴 기록
- **Mock 후기**: 실전 점수, 시간 관리, 약점 분석

이 시리즈의 목적은 멋진 합격 수기를 미리 쓰는 것이 아니다. 준비 과정 자체를 공개해서, 나중에 무엇이 효과적이었고 무엇이 낭비였는지 검증할 수 있게 만드는 것이다.
