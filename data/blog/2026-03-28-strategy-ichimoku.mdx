## 전략 개요

**이치모쿠 키제이코(一目均衡表)** 는 일본 언론인 호소다 고이치가 1969년에 고안한 기술적 분석 도구다.
이름 그대로 "한눈(一目)에 균형(均衡)을 파악하는 표(表)"라는 의미로,
단 하나의 차트로 **추세 방향, 지지/저항 구간, 모멘텀, 시간적 관계**를 동시에 파악할 수 있다.

---

## 5개 선의 구조와 의미

### 전환선 (Tenkan-sen) — 9기간

```python
전환선 = (최근 9봉 고가 + 최근 9봉 저가) / 2
```

단기 추세의 균형점. 가격이 전환선 위 = 단기 상승.
전환선과 기준선의 교차가 진입/청산 신호.

### 기준선 (Kijun-sen) — 26기간

```python
기준선 = (최근 26봉 고가 + 최근 26봉 저가) / 2
```

중기 추세의 균형점. **강력한 지지/저항선** 역할.
가격이 기준선 아래로 내려오면 약세 전환 신호.

### 선행스팬 A (Senkou Span A) — 26봉 앞에 그림

```python
선행스팬A = (전환선 + 기준선) / 2  # 26봉 후에 표시
```

구름의 한쪽 경계. 전환선과 기준선의 중간값.

### 선행스팬 B (Senkou Span B) — 52기간, 26봉 앞에 그림

```python
선행스팬B = (최근 52봉 고가 + 최근 52봉 저가) / 2  # 26봉 후에 표시
```

구름의 다른 경계. 더 장기적인 균형점.

### 후행스팬 (Chikou Span) — 26봉 뒤에 그림

```python
후행스팬 = 현재 종가  # 26봉 전 위치에 표시
```

현재 가격을 26봉 전과 비교. 후행스팬이 26봉 전 가격 위 = 강세.

---

## 구름 (Kumo) 이해

선행스팬 A와 B 사이의 공간이 **구름(Kumo)** 이다.

```
A > B: 양운(bullish cloud) → 초록색 → 지지 구간
A < B: 음운(bearish cloud) → 빨간색 → 저항 구간
```

구름의 두께 = 지지/저항의 강도. 두꺼울수록 뚫기 어렵다.

## 진입 로직

### LONG 조건

```python
# 1. 전환선이 기준선을 상향 돌파 (단기 골든크로스)
tk_cross_up = tenkan_prev <= kijun_prev and tenkan_cur > kijun_cur

# 2. 현재가가 구름 위 (강한 상승 추세 확인)
above_cloud = price > max(senkou_a, senkou_b)

# strict=True 시 추가 조건:
# 3. 후행스팬이 26봉 전 가격 위
chikou_bullish = close_now > close_26bars_ago

# 4. 양운 (A > B)
bullish_cloud = senkou_a > senkou_b
```

### SHORT 조건

```python
# 1. 전환선이 기준선을 하향 돌파
tk_cross_dn = tenkan_prev >= kijun_prev and tenkan_cur < kijun_cur

# 2. 현재가가 구름 아래
below_cloud = price < min(senkou_a, senkou_b)
```

## strict 모드 vs 완화 모드

```python
# strict=True (고신뢰, 신호 적음)
# 전환선/기준선 크로스 + 구름 위치 + 후행스팬 + 구름 색상
# 4가지 모두 충족해야 진입

# strict=False (고빈도, 신호 많음)  ← 백테스트에 사용한 설정
# 전환선/기준선 크로스 + 구름 위치만 확인
```

백테스트는 `strict=False`로 진행했다.

## 파라미터 설명

| 파라미터       | 기본값 | 설명                                 |
| -------------- | ------ | ------------------------------------ |
| `tenkan`       | 9      | 전환선 기간                          |
| `kijun`        | 26     | 기준선 기간. 가장 중요한 파라미터    |
| `senkou_b`     | 52     | 선행스팬B 기간 (보통 kijun × 2)      |
| `displacement` | 26     | 구름 앞당김 기간 (보통 kijun과 동일) |
| `strict`       | True   | True: 4조건 모두 / False: 2조건만    |

### 타임프레임별 권장 파라미터

```
# 일봉 (원래 설계)
tenkan=9, kijun=26, senkou_b=52

# 4시간봉 (중기)
tenkan=9, kijun=26, senkou_b=52  # 동일 가능

# 1시간봉 (권장 최소 타임프레임)
tenkan=9, kijun=26, senkou_b=52

# 15분봉 (노이즈 많음, strict=False + 주의)
tenkan=9, kijun=26, senkou_b=52
```

이치모쿠는 **원래 일봉을 위해 설계된 지표**다.
일봉 기준 9일, 26일(약 1개월), 52일(약 2개월)은 의미 있는 기간이다.
15m에서는 9캔들 = 2시간15분에 불과해 노이즈가 많다.

### 왜 90일 -95%, 365일 +398%인가?

이 극단적인 차이가 이치모쿠 전략의 핵심 특성을 보여준다.

**최근 90일 (2025.12 ~ 2026.03)**: SOL은 전반적으로 하락/횡보 구간이었다.
이치모쿠는 추세가 없는 구간에서 구름 위/아래를 빠르게 넘나들며 잦은 신호를 발생시킨다.
15m처럼 짧은 타임프레임에서 노이즈가 증폭되어 연속 손절이 발생했다.

**1년 전체 (2025.03 ~ 2026.03)**: SOL이 큰 상승 추세를 탔던 구간(2025 상반기)이 포함된다.
이치모쿠는 강한 추세에서 구름 위에 가격이 유지되는 동안 포지션을 끌고 가는 힘이 있다.
한두 번의 큰 추세 구간에서 대부분의 수익이 발생했다.
