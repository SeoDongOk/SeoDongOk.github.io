일본어를 공부할 때 가장 먼저 만나는 조동사가 「**ます**」다. "行きます, 食べます, します" — 교과서 첫 페이지부터 끝까지 따라다닌다. 그런데 정작 **「ます가 왜 ます인지」**, **「지방마다 어떻게 변형되는지」**, **「비즈니스 경어와 어떻게 연결되는지」** 를 체계적으로 아는 사람은 의외로 드물다.

이 글은 마스형의 **어원**, **활용 규칙**, **시제**, **파생 문형**, **방언**, **비즈니스 사용**까지 — 일본어 정중체의 뼈대를 한 편으로 정리한다. 초급 학습자의 총정리용으로도, 중급 이상의 교양 확장용으로도 쓸 수 있게 구성했다.

---

## 1. 마스형이란 무엇인가

**ます形(마스形, masu-kei)** 은 일본어 동사에 조동사 「ます」가 붙은 활용형이다. 문법적으로는 동사의 **연용형(連用形, れんようけい)** 에 「ます」가 접속한 구조다.

- **기능**: 정중체(丁寧体, ていねいたい)의 기본형
- **대응**: 반말체(普通体 / タメ口) ↔ 정중체(です・ます体)의 대립
- **역할**: 모르는 사람, 연장자, 공식 석상, 비즈니스 등에서 기본값으로 쓰는 형태

### 3대 특징

1. **예의의 최저 기준선** — 일본 사회에서 성인이 처음 만난 상대에게 쓰는 기본 수준. 「丁寧」 이상의 경어(겸양어·존경어)가 아닌, 가장 일반적인 공손 표현.
2. **시제·긍정부정을 모두 담당** — ます / ました / ません / ませんでした 네 가지가 정중체 시제의 기본 4형.
3. **다양한 파생 문법의 기반** — 권유(ましょう), 제안(ませんか), 확인(ましょうか) 등 핵심 문형이 모두 ます形에서 출발.

## 2. 어원과 역사적 기원

「ます」의 유래는 국어학자 간에도 합의가 없지만, 네 가지 학설이 있다.

### 학설 1: まゐらす(参らす) 축약설 (가장 유력) [주1]

고전 일본어에서 자기 행동을 낮추는 겸양 동사 「**まゐる(参る)**」(가다·오다의 겸양형)에 사역 조동사 「す」가 붙은 「**まゐらす**」가 있었다.

```
まゐらす → まらする → まらす → まっす → ます
```

헤이안 말기~가마쿠라 시대(12~13C)에 축약이 시작되어, **무로마치 시대(14~16C)에 「ます」로 정착**했다는 설. 국어학계 다수설이다.

### 학설 2: まうす(申す) 파생설

"말씀드리다"의 겸양 동사 「**まうす(申す)**」에서 ます가 떨어져 나왔다는 소수설.

### 학설 3: ます(坐す/在す) 경어 동사설

고어에서 "**계시다·앉으시다**"를 의미하던 존재 동사 「**ます(坐す)**」에서 존경 의미가 먼저 발생했다는 설. 에도 시대 국학자 **本居宣長(모토오리 노리나가)** 가 『詞の玉緒』 등에서 지지했다 [주2].

### 학설 4: 복합 기원설

위 여러 뿌리가 **합류**하여 「ます」가 형성됐다는 현대의 견해. 특히 학설 1과 3의 영향이 겹친다고 본다 [주1].

### 시대별 정착 과정

| 시대     | 연대      | 상태                                                     |
| -------- | --------- | -------------------------------------------------------- |
| 헤이안   | 794–1185  | まゐらす 형태로 겸양 표현 존재                           |
| 가마쿠라 | 1185–1333 | まらす・まっする 등 축약 시작                            |
| 무로마치 | 1336–1573 | 「ます」 형태 확립, 구어에 침투                          |
| 에도     | 1603–1868 | **조닌(町人) 계층의 대중적 정중어로 폭발적 확산**        |
| 메이지   | 1868–1912 | 표준어 정비 과정에서 **「丁寧体」의 공식 기준으로 채택** |
| 현대     | 1945~     | 학교 교육·방송·비즈니스의 기본값                         |

원래는 **겸양에서 출발**했지만, 에도 시대에 대중화되면서 **"자신을 낮추는" 뉘앙스가 희석**되어 오늘날에는 "보통 정도의 공손함"을 나타내는 중립적 형태가 됐다. 이것이 일본어 경어 체계를 이해하는 중요한 역사적 배경이다 [주3].

## 3. 동사 3그룹별 ます形 변환 규칙

> **동사를 3그룹으로 분류 → 각 그룹 규칙에 따라 연용형을 만든 뒤 ます를 붙인다.**

### 1그룹: 五段動詞 (U-verbs)

**식별법**: 마지막 음절이 う段(u단)으로 끝나는 동사
**규칙**: う段 → い段 + ます

| 어미 | 예시 | ます形       | 로마자                |
| ---- | ---- | ------------ | --------------------- |
| う   | 買う | 買い**ます** | kau → kaimasu         |
| く   | 書く | 書き**ます** | kaku → kakimasu       |
| ぐ   | 泳ぐ | 泳ぎ**ます** | oyogu → oyogimasu     |
| す   | 話す | 話し**ます** | hanasu → hanashimasu  |
| つ   | 待つ | 待ち**ます** | matsu → **machimasu** |
| ぬ   | 死ぬ | 死に**ます** | shinu → shinimasu     |
| ぶ   | 遊ぶ | 遊び**ます** | asobu → asobimasu     |
| む   | 飲む | 飲み**ます** | nomu → nomimasu       |
| る   | 作る | 作り**ます** | tsukuru → tsukurimasu |

주의: **つ→ち**, **す→し**, **う→い** 는 단순 모음 교체가 아니라 자음까지 바뀌는 변화라 발음에 신경 써야 한다.

### 2그룹: 一段動詞 (Ru-verbs)

**식별법**: 어미가 「る」이고 앞 음절이 **い단 또는 え단**
**규칙**: る 제거 + ます

| 사전형 | 의미     | ます形   |
| ------ | -------- | -------- |
| 見る   | 보다     | 見ます   |
| 起きる | 일어나다 | 起きます |
| 食べる | 먹다     | 食べます |
| 寝る   | 자다     | 寝ます   |
| 教える | 가르치다 | 教えます |

### 3그룹: 不規則動詞 (2개뿐)

| 사전형     | 의미 | ます形             | 주의                |
| ---------- | ---- | ------------------ | ------------------- |
| する       | 하다 | **します**         | 어간 자체 변화      |
| 来る(くる) | 오다 | **来ます(きます)** | **く→き 발음 변화** |

**する**는 무한히 확장된다. 「勉強する → 勉強します」, 「コピーする → コピーします」, 「チェックする → チェックします」 — 외래어·한자어와 결합해 신조어를 계속 만들어낸다. 이것이 현대 일본어 동사 생산성의 핵심이다.

### 함정 동사 — "2그룹처럼 생긴 1그룹"

외형은 「い/e단 + る」이지만 **실제로는 1그룹**으로 활용하는 동사들. 초보자가 가장 많이 틀리는 부분.

| 사전형               | 의미       | 올바른 ます形 | ❌ 흔한 오답 |
| -------------------- | ---------- | ------------- | ------------ |
| 帰る(かえる)         | 돌아가다   | 帰**り**ます  | 帰えます     |
| 切る(きる)           | 자르다     | 切**り**ます  | 切ます       |
| 入る(はいる)         | 들어가다   | 入**り**ます  | 入ます       |
| 走る(はしる)         | 달리다     | 走**り**ます  | 走ます       |
| 知る(しる)           | 알다       | 知**り**ます  | 知ます       |
| 要る(いる, 필요하다) | 필요하다   | 要**り**ます  | 要ます       |
| 滑る(すべる)         | 미끄러지다 | 滑**り**ます  | 滑べます     |
| 喋る(しゃべる)       | 수다떨다   | 喋**り**ます  | 喋べます     |
| 減る(へる)           | 줄다       | 減**り**ます  | 減えます     |
| 焦る(あせる)         | 초조하다   | 焦**り**ます  | 焦えます     |

**특히 주의**: 「いる」 두 형제

| 동사       | 의미             | 그룹  | ます形   |
| ---------- | ---------------- | ----- | -------- |
| 居る(いる) | 있다 (생물 존재) | 2그룹 | います   |
| 要る(いる) | 필요하다         | 1그룹 | 要ります |

이 11개 정도를 통째로 외워두면 실전 95%는 커버된다.

## 4. 시제와 부정형

일본어는 **비과거(현재=미래) / 과거** 2시제 체계다. 그래서 마스형의 시제 변형도 4가지뿐이다.

| 형태           | 시제·극성      | 예시 (飲む)          | 한국어            |
| -------------- | -------------- | -------------------- | ----------------- |
| 〜ます         | 현재·미래 긍정 | 飲み**ます**         | 마십니다          |
| 〜ました       | 과거 긍정      | 飲み**ました**       | 마셨습니다        |
| 〜ません       | 현재·미래 부정 | 飲み**ません**       | 마시지 않습니다   |
| 〜ませんでした | 과거 부정      | 飲み**ませんでした** | 마시지 않았습니다 |

### 「ました」의 완료 뉘앙스

단순 과거뿐 아니라 **완료의 순간 감정**을 담는 특수 용법이 있다.

- わかり**ました**！ (알겠습니다! — 방금 이해한 순간)
- でき**ました**！ (다 됐어요! — 완성 순간)
- 疲れ**ました**。 (피곤합니다 ← 직역: 피곤해졌습니다)

영어 현재완료나 한국어 "~해졌다"와 비슷한 뉘앙스.

### 「ません」 vs 「ないです」 — 뉘앙스 차이

현대 구어에서는 동사 **ない형 + です** 로도 정중 부정을 만들 수 있다.

| 형태           | 예               | 뉘앙스                  |
| -------------- | ---------------- | ----------------------- |
| 〜ません       | 飲みません       | **격식**, 비즈니스 기본 |
| 〜ないです     | 飲まないです     | **회화적**, 친근함      |
| 〜ませんでした | 行きませんでした | **격식** 과거 부정      |
| 〜なかったです | 行かなかったです | **회화적** 과거 부정    |

비즈니스 메일·공식 발표에서는 「ません/ませんでした」가 표준. 일상 대화에서는 「ないです/なかったです」가 부드럽게 들린다.

### 의문형 — 「か」를 붙일 뿐

| 형태             | 예                 |
| ---------------- | ------------------ |
| 〜ますか         | 行きますか         |
| 〜ましたか       | 行きましたか       |
| 〜ませんか       | 行きませんか       |
| 〜ませんでしたか | 行きませんでしたか |

주의: **「〜ませんか」는 이중 용법**이다.

1. 단순 부정 의문 — "안 오셨나요?"
2. 부드러운 권유 — "함께 가시지 않을래요?"

## 5. 파생 문형 12선 — ます의 확장 생태계

「ます」는 단순 "~ㅂ니다"를 넘어 수많은 문형의 허브다.

### (1) 〜ましょう — 권유 / 의지

- **함께 ~합시다**: 行き**ましょう** / 始め**ましょう**
- **제가 ~하겠습니다**: 私が持ち**ましょう**

### (2) 〜ませんか — 부드러운 권유

- 一緒に行き**ませんか**? (함께 가시지 않을래요?)
- コーヒーでも飲み**ませんか**?

**ましょう vs ませんか**: 전자는 주도적·단정, 후자는 상대 의향 확인·부드러움. 데이트 신청은 「ませんか」가 훨씬 자연스럽다.

### (3) 〜ましょうか — 제안 / 도움 제의

- ドアを閉め**ましょうか**? (문 닫을까요?)
- 荷物、持ち**ましょうか**? (짐 들어드릴까요?)

### (4) 〜ますと — 정중한 조건

- こちらを押し**ますと** 電源が入ります。 (이쪽을 누르시면 전원이 들어옵니다)

비즈니스 설명·안내 방송에서 표준.

### (5) 〜まして — ます의 て形 (고급 연결)

- お電話し**まして** すみません。 (전화드려서 죄송합니다)
- 失礼し**まして**…

비즈니스 메일·공식 문서의 **고급 연결**.

### (6) 〜ますので / 〜ますから — 정중한 이유 접속

- 会議があり**ますので**、失礼します。

「ので」는 객관적·공손, 「から」는 주관적·직접적. 메일에서는 「ので」가 기본.

### (7) 〜ますが / 〜ますけれど(ども) — 정중한 역접

- よろしいかと思い**ますが**、ご確認ください。
- 申し訳ござい**ませんけれども**、満席です。

### (8) 〜ますね / 〜ますよ / 〜ますよね — 종조사 결합

| 형태       | 예           | 뉘앙스      |
| ---------- | ------------ | ----------- |
| 〜ますね   | 行きますね   | 확인·공유   |
| 〜ますよ   | 行きますよ   | 알림·강조   |
| 〜ますよね | 行きますよね | 동의 구하기 |

이 3형제를 자연스럽게 쓰는 게 일본어 중급의 분기점.

### (9) お〜ください — 정중한 요청

**お + 마스어간 + ください** 공식은 경어 시스템의 핵심 문형.

- お**待ち**ください (기다려 주십시오)
- お**書き**ください (써 주십시오)
- ご**連絡**ください (연락 주십시오) — 한자어는 「ご」

### (10) ませ — 접객 전용 고어 명령

- いらっしゃい**ませ**！ (어서 오십시오!)
- お気をつけ**ませ**。 (조심히 가십시오)

일상 회화에서는 거의 안 쓰지만 **접객·료칸·백화점**의 필수 표현.

### (11) 〜ましたら — 정중한 가정

- 何かござい**ましたら**、ご連絡ください。 (무슨 일 있으시면 연락 주세요)

비즈니스 메일 맺음말 정석.

### (12) 〜ますでしょうか — 초정중 의문 (논쟁형)

- お待ちいただけ**ますでしょうか**?

원칙상 ます + でしょう 결합은 문법적으로 논쟁이 있지만, 접객·고객 대응에서 **초정중 관용구**로 완전히 정착했다.

## 6. 지역 방언 속의 마스형

오늘 우리가 "일본어 표준어"라 부르는 것의 상당 부분은 **에도(현 도쿄) 야마노테 지역** 방언 기반이다. 메이지 시대 표준어 정책으로 **「ます」가 전국 공식 정중어**가 됐지만, 각 지방은 자기만의 정중 표현을 유지해왔다 [주5].

### 関西弁 — 오사카·교토의 정중어

**오사카(大阪弁)**: 상인의 도시 특유의 공손+친근 혼합체

| 표준어     | 오사카                        |
| ---------- | ----------------------------- |
| 〜ます     | 〜**まっせ** / 〜**まんねん** |
| 〜ません   | 〜**まへん**                  |
| 行きます   | 行きまっせ                    |
| 知りません | 知りまへん                    |

- 「まっせ」 = 단언·강조 ("~하고말고요")
- 「まへん」 = ません의 축약 변형 (나라·와카야마에서도 사용)

**교토(京都弁)**: 헤이안 귀족 언어의 우아한 잔향

- 「〜どす」 = です의 교토식: 「こちらどす」
- 「〜どすえ」 = 친근 정중: 「寒どすえ」
- 「〜はる」 = 존경의 간사이식: 「先生、来はる?」

게이샤 화류계, 료칸, 노포에서 여전히 살아 있다.

### 九州弁 — 고어의 잔재

**하카타(博多, 福岡)**: 「〜しよる」(진행) vs 「〜しとる」(결과·완료) 를 정확히 구분. 표준어 「している」가 문맥에 따라 갈린다.

**사쓰마(薩摩, 鹿児島)**: 현대 일본어 중 가장 난해하기로 꼽힌다.

| 표준어     | 사쓰마                    |
| ---------- | ------------------------- |
| 〜ます     | 〜**もす** / 〜**もんす** |
| 行きます   | 行**っ**もす              |
| 知りません | 知**ら**もはん            |

「もす」는 ます의 고어 형태 「まうす」 계열의 잔존. **오히려 표준어보다 오래된 형태가 살아 있는** 사례.

### 東北弁 — 'ズーズー弁' 의 경어

| 표준어   | 도호쿠                      |
| -------- | --------------------------- |
| 行きます | 行**ぎ**あす / 行**ぎ**んす |
| 食べます | 食**べ**あす / 食**べ**んす |

「〜んす / 〜あす」가 ます의 도호쿠 변형. 남부 번(岩手 북부)에서는 「〜えす」.

### 沖縄 — 琉球語 계통

오키나와어(ウチナーグチ)는 엄밀히는 일본어와 다른 언어 계통 [주6]. 현대는 "ウチナーヤマトゥグチ"(표준어 + 오키나와 억양)가 주류.

**전통 琉球語**:

| 표준어     | 오키나와                      |
| ---------- | ----------------------------- |
| 〜します   | 〜**さびーん** / 〜**さびら** |
| 〜しません | 〜**さびらん**                |
| 〜しました | 〜**さびたん**                |

琉球王朝 시대 공식 정중어의 흔적.

### 시대극 속 고어 — まする / ござる

근세 에도 초중기까지 「〜まする」가 주류였다. 시대극에서 무사가 "さようにござい**まする**"(그러하옵니다) 라고 하는 것이 그것. **의도적 고풍 연출**용으로 현재도 쓰인다.

### 방언 ます 총정리

| 지역       | 대표 변형                  | 예시           |
| ---------- | -------------------------- | -------------- |
| 大阪       | まっせ / まんねん / まへん | 行きまっせ     |
| 京都       | どす / どすえ / はる       | そうどす       |
| 博多       | しよる / しとる            | 行きよります   |
| 鹿児島     | もす / もんす              | 行きもす       |
| 東北       | んす / あす / えす         | 行ぎあす       |
| 沖縄(전통) | さびーん / さびら          | しさびーん     |
| 시대극     | まする / ございまする      | さようでまする |

## 7. 정중도 5단계와 마스형의 위치

일본어 경어 체계는 5단계로 정리된다 [주4].

| 단계 | 명칭       | 특징              | 예                        |
| ---- | ---------- | ----------------- | ------------------------- |
| 1    | **タメ口** | 반말              | 行く / 食べる             |
| 2    | **丁寧語** | **ます・です체**  | 行きます / 食べます       |
| 3    | **尊敬語** | 상대를 높임       | いらっしゃる / 召し上がる |
| 4    | **謙譲語** | 자신을 낮춤       | 参る / いただく           |
| 5    | **美化語** | 사물 미화 (お/ご) | お茶 / ご飯               |

### 핵심 통찰

**마스형은 경어의 뼈대**다. 존경어·겸양어는 ます형 위에 얹히는 "옷"일 뿐이다.

| 기본 (ます체) | 존경어               | 겸양어                  |
| ------------- | -------------------- | ----------------------- |
| 食べます      | **召し上がります**   | **いただきます**        |
| 行きます      | **いらっしゃいます** | **参ります / 伺います** |
| 見ます        | **ご覧になります**   | **拝見します**          |
| 言います      | **おっしゃいます**   | **申します**            |
| 知っています  | **ご存知です**       | **存じております**      |
| します        | **なさいます**       | **いたします**          |

경어를 배우려면 **마스형이 먼저 체화**돼야 한다.

### 언제 ます를 쓰고 언제 떼는가

**쓰는 상황**: 처음 만난 사람 / 연장자 / 비즈니스 / 공식 석상 / 접객 / 공공 기관 / 전화

**떼는(タメ口) 상황**: 친한 친구·연인·가족 / 동아리 동년배 / SNS / 혼잣말 / 동물·아이에게

### "タメ口になる"의 사회적 의미

일본에서 **「タメ口になる(타메구치가 되다)」** 는 **관계의 친밀도 진입**을 의미한다. 한국어 "말 놓자"와 동일한 의례다.

- 「**タメ口で大丈夫?**」 (반말해도 돼?) — 친해지기 전 한번 물어보는 게 매너
- 친해졌는데 계속 ます를 쓰면 **"거리감 두고 싶다"** 는 신호

### 한국어와의 미세한 차이

| 한국어            | 일본어        | 특징                    |
| ----------------- | ------------- | ----------------------- |
| -ㅂ니다 / -습니다 | **ます**      | 격식                    |
| -아/어요          | (없음)        | 한국어 특유의 중간 레벨 |
| -아/어            | 반말 (タメ口) | 반말                    |

한국어 "-아/어요"에 해당하는 **중간 공손 레벨이 일본어에는 없다**. 일본어는 **ます vs 반말의 이분법**이 강하고, ます 안에서 **종조사(ね, よ, よね)** 로 친밀도를 조절한다.

## 8. 실전 예문집 — 장면별 마스형

### (1) 자기소개

```
初めまして、キムと申します。
韓国から来ました。
日本語を勉強しています。
よろしくお願いします。
```

- 「申します」 = 「言います」의 겸양형. 자기 이름 소개 필수.
- 「よろしくお願いします」 — 만남 끝 정형구.

### (2) 전화 응대

```
はい、〇〇株式会社でございます。
どちら様でしょうか?
少々お待ちください。
おつなぎいたします。
```

### (3) 비즈니스 메일

```
〇〇様

お世話になっております。
△△株式会社のキムです。

先日ご依頼いただいた件、進捗をご報告いたします。
...
ご確認のほど、よろしくお願い申し上げます。
```

- 「お世話になっております」 — 비즈니스 메일 시작의 정석
- 「ご報告いたします」 — お/ご + 한자어 + いたします 겸양 공식
- 「よろしくお願い申し上げます」 — 맺음말 최상급

### (4) 식당 / 상점

```
店員: いらっしゃいませ！
      ご注文はお決まりでしょうか?
손님: これをお願いします。
店員: かしこまりました。少々お待ちください。
      お待たせいたしました。
      ありがとうございました。
```

### (5) 호텔 / 료칸

```
チェックイン、お願いします。
承りました。お部屋までご案内いたします。
ごゆっくりお過ごしくださいませ。
```

## 9. 학습자가 자주 저지르는 5가지 실수

### ❌ 실수 1: 친해진 뒤에도 계속 ます

친한 일본인 친구에게 "行きます / 食べます"를 계속 쓰면 **"이 사람 거리를 두고 싶나?"** 오해를 산다. 친해지면 반말(行く / 食べる)로 전환하는 게 자연스럽다.

### ❌ 실수 2: 윗사람에게 ます만 쓰고 존경어 생략

「上司が来**ました**」 — 문법은 맞지만 상사에게 "왔습니다"는 존경 부족.
정답: 「部長が**いらっしゃいました**」 / 「**お見えになりました**」

### ❌ 실수 3: 자기 행동에 존경어 사용

「私は駅に**いらっしゃいます**」 (X) — 자기 행동에 존경어 금지
정답: 「私は駅に**参ります**」 (겸양) 또는 그냥 「行きます」

### ❌ 실수 4: 이중 경어(二重敬語)

「お召し上がりになられます」 (X) — 존경이 삼중으로 중첩
정답: 「召し上がります」 또는 「お召し上がりになります」까지.

일본인도 헷갈리는 영역이지만 **과도한 공손은 어색**하다.

### ❌ 실수 5: です・ます 섞어 쓰기

한 문단에서 「行き**ます**。食べ**た**。」처럼 정중체와 반말을 섞으면 부자연스럽다.
**문체 통일(文体統一)** 은 기본 원칙이다.

## 10. 한국인 학습자를 위한 팁

1. **마스형은 모든 경어의 뼈대** — 존경어·겸양어 변환표를 외우기 전에 마스형을 완벽히 체화할 것.
2. **함정 동사 11개는 통째로** — 帰る·切る·入る·走る·知る·要る·滑る·喋る·減る·焦る·蹴る. 이것만 외워도 기본 실력.
3. **ます를 떼는 타이밍을 관찰** — 일본인 친구가 먼저 반말로 말 걸 때 자연스럽게 따라가는 게 정답.
4. **종조사 ね/よ/よね** — ます 뒤에 붙이면 한국어의 "-요 / -죠 / -네요" 같은 풍부한 어감을 얻을 수 있다.
5. **비즈니스는 경어 세트로** — 「お世話になっております / 〜いたします / 〜くださいませ / よろしくお願い申し上げます」 를 패턴으로 습득.
6. **「まっせ」「まへん」은 교양** — 일상 학습에선 필요 없지만, 일본 드라마·여행에서 이해는 돼야 문화가 풍부해진다.

## 11. 마치며

「ます」 하나를 해부해 보면 일본어 전체가 보인다.

- **어원**은 헤이안 시대 겸양의 그림자
- **형태**는 동사 3그룹 분류 시스템의 관문
- **시제**는 비과거/과거 2시제 체계
- **파생**은 권유·제안·조건·요청의 허브
- **방언**은 각 지방의 공손 문화의 유산
- **경어**는 마스형 위에 쌓는 2층 구조

처음 일본어를 배울 때 "行きます"가 그저 "갑니다"의 번역이었다면, 이제는 **천 년에 걸쳐 다듬어진 일본인의 공손함 시스템의 입구**로 보일 것이다.

학습의 지름길은 없다. 하지만 **왜 그렇게 되는지 알고 외우는 것**과 **무작정 외우는 것**은 습득 속도에서 하늘과 땅 차이다. 이 글이 ます형을 뿌리부터 이해하는 데 도움이 됐기를 바란다.
