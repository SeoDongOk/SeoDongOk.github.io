일본어 초급에서 가장 많이 막히는 부분은 단어 뜻보다 **활용 변화**다.  
「行く」가 「行きます」, 「行って」, 「行かない」로 바뀌고, 「食べる」는 「食べます」, 「食べて」, 「食べない」로 바뀐다.

이 글은 JLPT N5~N4 초입에서 자주 쓰는 **동사·형용사 변동표**를 한 번에 볼 수 있도록 정리한 글이다.

## 먼저 외울 큰 구조

일본어 활용은 크게 아래 순서로 보면 된다.

1. 동사를 그룹으로 나눈다.
2. 그룹별 규칙에 따라 어미를 바꾼다.
3. 뒤에 붙는 문형을 연결한다.

| 품사     | 핵심 분류           | 예시               |
| -------- | ------------------- | ------------------ |
| 동사     | 1그룹, 2그룹, 3그룹 | 行く, 食べる, する |
| い형용사 | 어미가 い           | 高い, 新しい       |
| な형용사 | 명사처럼 활용       | 静か, 便利         |
| 명사     | です 활용           | 学生, 会社員       |

## 동사 1그룹 변동표

1그룹은 마지막 음이 `う단`에서 다른 단으로 바뀌는 것이 핵심이다.

| 사전형 끝 | ます형    | ない형    | て형 | 예시                                |
| --------- | --------- | --------- | ---- | ----------------------------------- |
| う        | い + ます | わ + ない | って | 買う → 買います / 買わない / 買って |
| く        | き + ます | か + ない | いて | 書く → 書きます / 書かない / 書いて |
| ぐ        | ぎ + ます | が + ない | いで | 泳ぐ → 泳ぎます / 泳がない / 泳いで |
| す        | し + ます | さ + ない | して | 話す → 話します / 話さない / 話して |
| つ        | ち + ます | た + ない | って | 待つ → 待ちます / 待たない / 待って |
| ぬ        | に + ます | な + ない | んで | 死ぬ → 死にます / 死なない / 死んで |
| ぶ        | び + ます | ば + ない | んで | 遊ぶ → 遊びます / 遊ばない / 遊んで |
| む        | み + ます | ま + ない | んで | 飲む → 飲みます / 飲まない / 飲んで |
| る        | り + ます | ら + ない | って | 作る → 作ります / 作らない / 作って |

> 예외: 行く → 行って / 行った

## 동사 2그룹 변동표

2그룹은 가장 쉽다. 마지막 `る`를 떼고 붙이면 된다.

| 형태   | 규칙           | 예시              |
| ------ | -------------- | ----------------- |
| ます형 | る 제거 + ます | 食べる → 食べます |
| て형   | る 제거 + て   | 食べる → 食べて   |
| ない형 | る 제거 + ない | 食べる → 食べない |
| た형   | る 제거 + た   | 食べる → 食べた   |

대표 동사:

- 見る → 見ます / 見て / 見ない / 見た
- 起きる → 起きます / 起きて / 起きない / 起きた
- 寝る → 寝ます / 寝て / 寝ない / 寝た
- 教える → 教えます / 教えて / 教えない / 教えた

## 3그룹 불규칙 동사

3그룹은 두 개뿐이다.

| 사전형       | ます형           | て형         | ない형           | た형         |
| ------------ | ---------------- | ------------ | ---------------- | ------------ |
| する         | します           | して         | しない           | した         |
| 来る（くる） | 来ます（きます） | 来て（きて） | 来ない（こない） | 来た（きた） |

## い형용사 변동표

| 형태      | 규칙            | 예시         |
| --------- | --------------- | ------------ |
| 현재 긍정 | 그대로          | 高い         |
| 현재 정중 | + です          | 高いです     |
| 현재 부정 | い → くない     | 高くない     |
| 과거 긍정 | い → かった     | 高かった     |
| 과거 부정 | い → くなかった | 高くなかった |
| て형      | い → くて       | 高くて       |

주의할 예외:

| 기본형 | 부정     | 과거     |
| ------ | -------- | -------- |
| いい   | よくない | よかった |

## な형용사 변동표

| 형태      | 예시                              | 뜻                    |
| --------- | --------------------------------- | --------------------- |
| 현재 긍정 | 静かだ / 静かです                 | 조용하다 / 조용합니다 |
| 현재 부정 | 静かじゃない / 静かではありません | 조용하지 않다         |
| 과거 긍정 | 静かだった / 静かでした           | 조용했다              |
| 과거 부정 | 静かじゃなかった                  | 조용하지 않았다       |
| 명사 수식 | 静かな部屋                        | 조용한 방             |
| 연결      | 静かで便利です                    | 조용하고 편리합니다   |

## 마무리

초급 활용은 결국 패턴 싸움이다.

- 1그룹: 어미가 단별로 바뀐다.
- 2그룹: る를 빼고 붙인다.
- 3그룹: する, 来る만 따로 외운다.
- い형용사: 마지막 い를 바꾼다.
- な형용사: 명사처럼 움직이고, 명사 앞에서는 な가 붙는다.

이 표를 보면서 직접 「書く, 読む, 食べる, 見る, する, 来る」를 소리 내어 변형해 보면 활용 감각이 빨리 붙는다.
