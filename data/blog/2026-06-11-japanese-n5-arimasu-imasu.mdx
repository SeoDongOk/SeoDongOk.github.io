일본어에서 "있다"를 표현하는 동사는 두 가지다.

- **あります** — 물건, 사물, 식물처럼 스스로 움직이지 않는 것
- **います** — 사람, 동물처럼 살아서 움직이는 것

한국어 "있어요" 하나로 다 되는데 일본어는 구분한다. 이걸 틀리면 바로 티가 나기 때문에 N5에서 가장 먼저 잡아야 할 포인트 중 하나다.

## あります vs います 한눈에 비교

| 구분           | 동사     | 예시                         |
| -------------- | -------- | ---------------------------- |
| 물건·사물·식물 | あります | 本があります (책이 있어요)   |
| 사람·동물      | います   | 猫がいます (고양이가 있어요) |
| 장소·건물·현상 | あります | 駅があります (역이 있어요)   |
| 곤충·어류      | います   | 魚がいます (물고기가 있어요) |

## 위치 표현: 〜に あります / います

위치를 말할 때는 **「장소 + に + あります/います」** 구조를 쓴다.

```
場所（ばしょ）に ＋ あります / います
```

### あります 예문

- 机（つくえ）の上（うえ）**に** 本（ほん）が**あります**。  
  (책상 위에 책이 있어요.)
- 駅（えき）の近（ちか）く**に** スーパーが**あります**。  
  (역 근처에 마트가 있어요.)
- 冷蔵庫（れいぞうこ）の中（なか）**に** りんごが**あります**。  
  (냉장고 안에 사과가 있어요.)

### います 예문

- 公園（こうえん）**に** 子（こ）どもたちが**います**。  
  (공원에 아이들이 있어요.)
- 部屋（へや）**に** 猫（ねこ）が**います**。  
  (방에 고양이가 있어요.)
- 学校（がっこう）**に** 先生（せんせい）が**います**。  
  (학교에 선생님이 계세요.)

## 부정형: ありません / いません

| 긍정     | 부정       |
| -------- | ---------- |
| あります | ありません |
| います   | いません   |

- 財布（さいふ）が**ありません**。 (지갑이 없어요.)
- 誰（だれ）も**いません**。 (아무도 없어요.)
- 今日（きょう）、授業（じゅぎょう）は**ありません**。 (오늘 수업은 없어요.)

## 위치를 나타내는 명사

위치 표현과 함께 쓰는 위치 명사를 외워두면 바로 문장을 만들 수 있다.

| 한국어 | 일본어  | 읽기          |
| ------ | ------- | ------------- |
| 위     | 上      | うえ          |
| 아래   | 下      | した          |
| 앞     | 前      | まえ          |
| 뒤     | 後ろ    | うしろ        |
| 옆     | 横 / 隣 | よこ / となり |
| 안     | 中      | なか          |
| 밖     | 外      | そと          |
| 오른쪽 | 右      | みぎ          |
| 왼쪽   | 左      | ひだり        |
| 근처   | 近く    | ちかく        |

**사용 예:**

- 椅子（いす）の**下に** バッグが あります。 (의자 아래에 가방이 있어요.)
- 銀行（ぎんこう）の**隣に** カフェが あります。 (은행 옆에 카페가 있어요.)

## 헷갈리는 케이스

### 식물은 あります

식물은 스스로 이동하지 않으므로 `あります`를 쓴다.

- 庭（にわ）に 桜（さくら）の木（き）が **あります**。 (정원에 벚나무가 있어요.)

### 사진 속 사람

사진이나 그림 속 사람도 **います**를 쓴다. 사람이 거기 "존재"하는 상황이기 때문이다.

- 写真（しゃしん）に 家族（かぞく）が **います**。 (사진에 가족이 있어요.)

### 로봇·인형

취급이 갈리지만 N5 수준에서는 보통 `あります`를 쓴다.

## 핵심 요약

|           | あります            | います      |
| --------- | ------------------- | ----------- |
| 대상      | 물건·사물·식물·장소 | 사람·동물   |
| 부정      | ありません          | いません    |
| 위치 구조 | 〜に あります       | 〜に います |

**판단 기준:** 스스로 걸어 다닐 수 있으면 `います`, 없으면 `あります`.
