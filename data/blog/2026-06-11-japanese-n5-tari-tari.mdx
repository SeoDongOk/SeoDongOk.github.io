"주말에는 게임도 하고 영화도 보고 그래요" — 이렇게 여러 동작을 예시처럼 나열할 때 쓰는 표현이 `〜たり〜たりする`다.

모든 것을 다 나열하는 게 아니라, **대표적인 몇 가지를 예시로 드는** 뉘앙스가 핵심이다.

---

## 만드는 법

```
동사 た형 + り、동사 た형 + りする
```

た형은 과거형이지만, 이 문형에서는 현재·미래에도 쓸 수 있다.

| 동사   | た형   | 〜たり형 |
| ------ | ------ | -------- |
| 見る   | 見た   | 見たり   |
| 聞く   | 聞いた | 聞いたり |
| 食べる | 食べた | 食べたり |
| 読む   | 読んだ | 読んだり |
| する   | した   | したり   |

---

## 기본 예문

- 週末（しゅうまつ）は音楽を聴い**たり**、映画を見**たりします**。  
  (주말에는 음악을 듣거나 영화를 보거나 해요.)

- 休みの日は家でゆっくりし**たり**、友達と遊ん**だりします**。  
  (쉬는 날에는 집에서 쉬거나 친구와 놀거나 해요.)

- 試験前は緊張（きんちょう）し**たり**、ドキドキし**たりします**。  
  (시험 전에는 긴장하거나 두근두근하거나 해요.)

---

## 형용사에도 쓸 수 있다

형용사도 た형으로 만든 후 `り`를 붙인다.

| 형용사          | た형       | 〜たり형     |
| --------------- | ---------- | ------------ |
| 高い (비싸다)   | 高かった   | 高かったり   |
| 静か (조용하다) | 静かだった | 静かだったり |

- この店は高かっ**たり**、安かっ**たりします**。  
  (이 가게는 비쌌다 싸졌다 해요. — 가격이 일정하지 않음)

- 天気が良かっ**たり**、悪かっ**たりします**。  
  (날씨가 좋았다 나빴다 해요.)

---

## 〜たりする vs 〜て 나열 차이

|        | 〜て 나열                            | 〜たり〜たりする                   |
| ------ | ------------------------------------ | ---------------------------------- |
| 의미   | 행동을 순서대로 모두 나열            | 대표적인 몇 가지만 예시            |
| 뉘앙스 | 완전한 목록                          | "이런 것들을 한다"                 |
| 예     | 起きて、顔を洗って、ご飯を食べました | 映画を見たり、音楽を聴いたりします |

- 朝は起き**て**、顔を洗**って**、朝ごはんを食べました。  
  (아침에 일어나서 세수하고 아침을 먹었어요.) — 전부 나열
- 休日は映画を見**たり**、ゲームをし**たりします**。  
  (쉬는 날에는 영화 보거나 게임하거나 해요.) — 예시만

---

## 과거·부정·공손 변형

마지막 `する` 부분을 변형하면 된다.

| 형태      | 표현           | 예               |
| --------- | -------------- | ---------------- |
| 현재      | 〜たりする     | 食べたりする     |
| 정중      | 〜たりします   | 食べたりします   |
| 과거      | 〜たりした     | 食べたりした     |
| 정중 과거 | 〜たりしました | 食べたりしました |
| 부정      | 〜たりしない   | 食べたりしない   |
| 부정 정중 | 〜たりしません | 食べたりしません |

- 昨日は映画を見**たり**、友達と話し**たりしました**。  
  (어제는 영화를 보거나 친구와 이야기하거나 했어요.)

---

## 한 가지만 쓰는 경우

두 가지 이상 나열하는 게 일반적이지만, 한 가지만 쓰는 것도 가능하다.  
이때는 "이런 것도 하고 그런 것도 한다"는 **여운 뉘앙스**가 생긴다.

- たまに泣い**たりします**。 (가끔 울거나 해요.) — 단독 사용, "울기도 하고..."

---

## 핵심 요약

1. **만드는 법:** 동사 た형 + り を 여러 개 나열 → 마지막에 する
2. **뉘앙스:** 모두 나열이 아닌 **대표 예시**
3. **형용사도 가능:** た형(かった・だった) + り
4. **する 활용:** 시제·공손도는 마지막 する를 바꿔서 조절
