"할 수 있어요", "할 수 없어요" — 가능·불가능 표현은 자기소개나 일상 대화에서 자주 쓴다.

N5 수준에서는 `〜ことができる`가 기본이고, N4에서 가능형 동사(`〜られる`)를 배운다. 두 가지를 비교해서 익혀두면 좋다.

---

## 〜ことができる — 〜하는 것이 가능하다

```
동사 사전형 + ことができる
```

가장 기본적인 가능 표현. 능력과 상황적 가능 모두 쓸 수 있다.

| 동사     | ことができる형       |
| -------- | -------------------- |
| 泳ぐ     | 泳ぐことができる     |
| 話す     | 話すことができる     |
| 運転する | 運転することができる |
| 来る     | 来ることができる     |

**예문:**

- 私は日本語を話す**ことができます**。 (저는 일본어를 말할 수 있어요.)
- ここで写真を撮る**ことができますか**。 (여기서 사진을 찍을 수 있어요?)
- 明日は来る**ことができません**。 (내일은 올 수 없어요.)
- 子どもの時、ピアノを弾く**ことができました**。  
  (어릴 때 피아노를 칠 수 있었어요.)

---

## 가능형 동사 (〜られる / N4)

동사 자체를 가능형으로 바꾸는 방법. 회화에서는 이 형태를 더 자주 쓴다.

### 2그룹 동사: る → られる

| 사전형 | 가능형     |
| ------ | ---------- |
| 食べる | 食べられる |
| 見る   | 見られる   |
| 起きる | 起きられる |

### 1그룹 동사: う단 → え단 + る

| 사전형 | 가능형 |
| ------ | ------ |
| 書く   | 書ける |
| 飲む   | 飲める |
| 話す   | 話せる |
| 帰る   | 帰れる |
| 買う   | 買える |

### 3그룹 동사 (불규칙)

| 사전형 | 가능형   |
| ------ | -------- |
| する   | できる   |
| くる   | こられる |

---

## ことができる vs 가능형 비교

|        | 〜ことができる     | 가능형 (〜られる/〜える) |
| ------ | ------------------ | ------------------------ |
| 공손도 | 약간 격식          | 자연스러운 회화          |
| 사용   | 공식 문서, 설명    | 일상 대화                |
| 예     | 泳ぐことができます | 泳げます                 |

실제로 같은 의미지만, 회화에서는 가능형 동사가 더 자연스럽다.

- 日本語が話せますか。 (일본어 말할 수 있어요?) ← 회화
- 日本語を話すことができますか。 (일본어를 말하는 것이 가능합니까?) ← 격식

---

## できる의 다양한 용법

`できる`는 "할 수 있다"뿐만 아니라 여러 뜻으로 쓰인다.

### 1. 할 수 있다 (능력)

- 英語が**できます**。 (영어를 할 수 있어요.) — ✨ が를 쓰는 것 주의!
- 料理が**できません**。 (요리를 못해요.)

### 2. 생기다·완성되다

- 新しいカフェが**できました**。 (새 카페가 생겼어요.)
- 友達が**できました**。 (친구가 생겼어요.)
- 報告書（ほうこくしょ）が**できました**。 (보고서가 완성됐어요.)

### 3. 상황이 가능하다

- 今日は会議に出ることが**できません**。 (오늘은 회의에 나갈 수가 없어요.)

---

## 가능·불가능 표현 정리

| 표현             | 의미                  |
| ---------------- | --------------------- |
| 〜ことができる   | 〜할 수 있다 (격식)   |
| 〜ことができない | 〜할 수 없다          |
| 가능형 동사      | 〜할 수 있다 (회화)   |
| 〜が下手/苦手    | 〜를 못한다/잘 못한다 |
| 〜が得意         | 〜를 잘한다           |

**관련 표현:**

- 水泳（すいえい）が**得意（とくい）**です。 (수영을 잘해요.)
- 数学（すうがく）が**苦手（にがて）**です。 (수학을 잘 못해요.)
- 英語は**まだできません**。 (영어는 아직 못해요.)

---

## 실전 자기소개에서 활용

- 私は韓国語と日本語が話せます。 (저는 한국어와 일본어를 말할 수 있어요.)
- ギターを弾くことができます。 (기타를 칠 수 있어요.)
- 料理はあまり得意じゃないです。 (요리는 별로 잘 못해요.)
- 運転ができません。 (운전을 못해요.)
