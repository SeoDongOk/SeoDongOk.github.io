가타카나는 처음 보면 전부 비슷한 기호처럼 보인다. 그런데 모양을 **그림**으로 바꿔서 외우면 훨씬 오래 간다. 이 글은 가타카나를 “쓰기”보다 **1초 안에 읽기** 위해 만든 매칭표다.

> 📌 기본 46자 표는 [일본어 가타카나 46자 표](/blog/2026-04-02-japanese-katakana-46)에서 먼저 확인하고, 이 글은 **암기 보조표**로 사용하면 좋다.

---

## 1. 히라가나랑 비슷한 가타카나

히라가나를 이미 외웠다면, 모양이 비슷한 가타카나부터 연결하는 게 가장 빠르다.

| 가타카나 | 읽기 | 히라가나 | 매칭 포인트             | 연상                                    |
| -------- | ---- | -------- | ----------------------- | --------------------------------------- |
| カ       | 카   | か       | 오른쪽 곡선이 사라진 か | 커피 방울이 떨어진 카푸치노             |
| キ       | 키   | き       | 곡선이 직선으로 바뀐 き | 열쇠 손잡이가 빠진 모양                 |
| セ       | 세   | せ       | 아래 곡선이 사라진 せ   | 엄마가 아기를 안고 있는데 아기만 사라짐 |
| モ       | 모   | も       | 아래 곡선이 사라진 も   | 머리카락, 모발이 올라오기 전            |
| ヤ       | 야   | や       | や를 단순화             | 히라가나 や를 직선화                    |
| リ       | 리   | り       | り를 직선화             | 리본 끈 두 줄                           |
| ヘ       | 헤   | へ       | 거의 동일               | 산처럼 꺾인 모양 그대로                 |
| ニ       | 니   | に       | 두 줄만 남음            | 위아래 두 줄, 숫자 2 느낌               |

### 빠른 매칭 퀴즈

| 가타카나 | 바로 떠올릴 것 |
| -------- | -------------- |
| カ       | か / 카        |
| キ       | き / 키        |
| セ       | せ / 세        |
| モ       | も / 모        |
| ヤ       | や / 야        |
| リ       | り / 리        |
| ヘ       | へ / 헤        |
| ニ       | に / 니        |

## 2. 사물로 외우기

모양이 사물처럼 보이는 글자는 억지로 획순을 외우기보다 이미지를 붙이면 빠르다.

| 가타카나 | 읽기 | 사물 연상              | 기억 문장                    |
| -------- | ---- | ---------------------- | ---------------------------- |
| テ       | 테   | 테이블                 | テ는 Table의 테이블 상판     |
| ロ       | 로   | 네모난 로봇 얼굴       | ロ봇 얼굴은 네모             |
| コ       | 코   | 돼지 코                | コ는 옆으로 열린 코          |
| ハ       | 하   | 하이파이브하는 두 팔   | ハ! 하고 양팔을 벌림         |
| ワ       | 와   | 와인잔                 | ワ인잔 윗부분                |
| レ       | 레   | 알파벳 L               | レ벨(Level)의 L 느낌         |
| ホ       | 호   | 주머니가 튀어나온 사람 | 호주머니가 양쪽으로 튀어나옴 |
| ミ       | 미   | 머리카락 3가닥         | 바닥에 떨어진 머리카락 3가닥 |
| ネ       | 네   | 내비게이션 화살표      | 네비가 방향을 꺾어 알려줌    |
| ム       | 무   | 무기, 칼날             | 무기처럼 뾰족한 모양         |

## 3. 동작으로 외우기

움직임이 느껴지는 글자는 “어떤 동작을 하는지”로 기억한다.

| 가타카나 | 읽기 | 동작 연상                 | 기억 문장                  |
| -------- | ---- | ------------------------- | -------------------------- |
| ウ       | 우   | 우회전                    | ウ는 꺾어서 우회전         |
| シ       | 시   | 낚싯바늘이 아래로 향함    | 시선이 아래로 흐름         |
| ツ       | 츠   | 위에서 아래로 처박는 모습 | 두 눈이 위에서 아래를 봄   |
| ノ       | 노   | 공을 던지는 궤적          | 노를 젓듯 한 획으로 지나감 |
| フ       | 후   | 훅 떨어지는 낚싯줄        | 후욱 떨어짐                |
| ク       | 쿠   | 구부러진 갈고리           | 쿠부러진 갈고리            |
| ケ       | 케   | 케이블이 매달린 모습      | 케이블이 아래로 늘어짐     |

## 4. 숫자·기호처럼 외우기

숫자나 기호처럼 보이는 글자는 모양 자체를 기준점으로 삼는다.

| 가타카나 | 읽기 | 숫자·기호 연상 | 기억 문장                         |
| -------- | ---- | -------------- | --------------------------------- |
| ニ       | 니   | 숫자 2         | 니는 2처럼 두 줄                  |
| エ       | 에   | 3층 구조       | 에스컬레이터 3층처럼 위·중간·아래 |
| ト       | 토   | 십자가 표시    | 토막 난 십자가                    |
| チ       | 치   | 천장 기둥      | 치켜든 기둥이 천장을 받침         |

## 5. 헷갈리는 글자 6개 집중 정리

가타카나에서 스트레스를 가장 많이 주는 글자는 보통 이 6개다.

```text
シ 시  = 세로눈
ツ 츠 = 가로눈

ソ 소 = 팔 높이 같음
ン 응 = 한쪽 팔 내려감

ナ 나 = +
メ 메 = X
```

이 6개만 확실히 구분해도 JLPT N5~N4 가타카나 읽기 부담이 크게 줄어든다.

### シ(시) vs ツ(츠)

| 글자 | 읽기 | 눈의 방향 | 연상         | 구분법                |
| ---- | ---- | --------- | ------------ | --------------------- |
| シ   | 시   | 세로      | `•` `•` `ノ` | **시 = 세로눈**       |
| ツ   | 츠   | 가로      | `• •` `ノ`   | **츠 = 두 눈이 가로** |

```text
シ
•
•
  ノ
```

```text
ツ
• •
 ノ
```

### ソ(소) vs ン(응)

| 글자 | 읽기 | 획 느낌                 | 연상          | 구분법                |
| ---- | ---- | ----------------------- | ------------- | --------------------- |
| ソ   | 소   | 양쪽 팔 높이가 비슷     | 소리친다      | 팔 높이가 비슷하면 ソ |
| ン   | 응   | 한쪽 팔이 아래로 내려감 | 응원하다 지침 | 한쪽이 축 내려가면 ン |

```text
ソ
＼ ＼
 ↓
```

```text
ン
＼
  ＼
   ↓
```

### ナ(나) vs メ(메)

| 글자 | 읽기 | 모양 | 연상               | 구분법             |
| ---- | ---- | ---- | ------------------ | ------------------ |
| ナ   | 나   | +    | NASA 로고의 나     | 더하기 모양이면 ナ |
| メ   | 메   | X    | 부메랑 던지기 직전 | X처럼 교차하면 メ  |

## 6. 생활 단어로 1초 읽기 훈련

JLPT N5를 준비 중이라면 가타카나를 예쁘게 쓰는 연습보다 **생활 단어를 바로 읽는 연습**이 더 직접적이다. 시험에서는 쓰기보다 읽기가 중요하기 때문이다.

| 가타카나 단어 | 읽기             | 뜻       | 체크 포인트               |
| ------------- | ---------------- | -------- | ------------------------- |
| コーヒー      | コー・ヒー       | 커피     | 장음 `ー`는 앞소리를 길게 |
| テレビ        | テ・レ・ビ       | 텔레비전 | テ와 レ 구분              |
| ホテル        | ホ・テ・ル       | 호텔     | ホ 모양 기억              |
| レストラン    | レ・ス・ト・ラン | 레스토랑 | レ, ト, ラン 읽기         |
| アイス        | ア・イ・ス       | 아이스   | ア와 イ 기본 모음         |
| パン          | パ・ン           | 빵       | ン과 ソ 구분              |
| バス          | バ・ス           | 버스     | 탁음 バ 확인              |
| タクシー      | タ・ク・シー     | 택시     | シ 장음 읽기              |

### 가타카나 → 한국어 매칭 퀴즈

| 문제 | 가타카나   | 힌트             | 답       |
| ---- | ---------- | ---------------- | -------- |
| 1    | コーヒー   | 마시는 것        | 커피     |
| 2    | テレビ     | 보는 것          | 텔레비전 |
| 3    | ホテル     | 여행 숙소        | 호텔     |
| 4    | レストラン | 밥 먹는 곳       | 레스토랑 |
| 5    | アイス     | 차가운 디저트    | 아이스   |
| 6    | パン       | 빵집에서 사는 것 | 빵       |
| 7    | バス       | 대중교통         | 버스     |
| 8    | タクシー   | 택시             | 택시     |

### 한국어 → 가타카나 매칭 퀴즈

| 문제 | 한국어   | 가타카나   |
| ---- | -------- | ---------- |
| 1    | 커피     | コーヒー   |
| 2    | 텔레비전 | テレビ     |
| 3    | 호텔     | ホテル     |
| 4    | 레스토랑 | レストラン |
| 5    | 아이스   | アイス     |
| 6    | 빵       | パン       |
| 7    | 버스     | バス       |
| 8    | 택시     | タクシー   |

## 7. 최종 암기표

마지막으로 이 표만 빠르게 반복한다.

| 묶음          | 외울 것                                                            |
| ------------- | ------------------------------------------------------------------ |
| 히라가나 매칭 | カ=か, キ=き, セ=せ, モ=も, ヤ=や, リ=り, ヘ=へ, ニ=に             |
| 사물 연상     | テ=테이블, ロ=로봇 얼굴, コ=코, ハ=하이파이브, ワ=와인잔, レ=L     |
| 동작 연상     | ウ=우회전, ノ=궤적, フ=훅, ク=갈고리, ケ=케이블                    |
| 헷갈리는 6개  | シ/ツ, ソ/ン, ナ/メ                                                |
| 생활 단어     | コーヒー, テレビ, ホテル, レストラン, アイス, パン, バス, タクシー |

가타카나는 “완벽하게 쓰기”보다 **보고 바로 읽기**가 먼저다. 오늘은 46자를 한 번에 다 외우려고 하기보다, 위 표에서 헷갈리는 글자만 골라 1초 안에 읽는 연습을 반복하자.
