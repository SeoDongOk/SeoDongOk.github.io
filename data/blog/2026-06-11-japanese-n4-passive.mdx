수동형은 "〜당하다", "〜해지다" 처럼 **누군가에 의해 행해지는 동작**을 나타낸다. 한국어 "비에 젖었다", "선생님께 혼났다" 등이 수동 표현에 해당한다.

---

## 수동형 만들기

### 2그룹 동사: る → られる

| 사전형 | 수동형     |
| ------ | ---------- |
| 食べる | 食べられる |
| 見る   | 見られる   |
| 起こす | 起こされる |
| 褒める | 褒められる |

### 1그룹 동사: あ단 + れる

| 사전형 | 수동형   |
| ------ | -------- |
| 書く   | 書かれる |
| 飲む   | 飲まれる |
| 話す   | 話される |
| 怒る   | 怒られる |
| 叩く   | 叩かれる |
| 盗む   | 盗まれる |
| 踏む   | 踏まれる |
| 呼ぶ   | 呼ばれる |

> **う로 끝나는 동사:** う → わ + れる  
> 使う → 使わ**れる**

### 3그룹 (불규칙)

| 사전형 | 수동형   |
| ------ | -------- |
| する   | される   |
| くる   | こられる |

---

## 직접 수동 — 피해자가 주어

가장 기본적인 수동. 누군가가 나에게 무언가를 했을 때.

```
[피해자]は [행위자]に [동사 수동형]
```

- 私（わたし）は先生（せんせい）に**褒められました**。  
  (저는 선생님께 칭찬 받았어요.)
- 財布（さいふ）を泥棒（どろぼう）に**盗まれました**。  
  (지갑을 도둑에게 도둑맞았어요.)
- 子どもの時、よく兄に**いじめられました**。  
  (어릴 때 자주 형에게 괴롭힘을 당했어요.)

---

## 간접 수동 (피해 수동) — 迷惑の受身

직접 행해지지 않았지만 **그 행동으로 피해를 입은** 경우. 일본어 수동의 독특한 용법.

- 雨に**降られて**、びしょびしょになった。  
  (비를 맞아서 흠뻑 젖었다. — 비가 내리는 것에 피해)
- 隣の人に夜中に**騒がれて**、眠れなかった。  
  (옆집 사람이 밤중에 떠들어서 잘 수 없었다.)
- 友達に急に**来られて**、困った。  
  (친구가 갑자기 와서 곤란했다.)

간접 수동은 대부분 **화자가 불편·피해를 입었음**을 나타낸다.

---

## 물건이 주어인 수동 (문어체·뉴스)

뉴스, 공식 문서에서 자주 나온다.

- この小説（しょうせつ）は夏目漱石（なつめそうせき）に**書かれました**。  
  (이 소설은 나쓰메 소세키가 썼습니다. / 써졌습니다.)
- この建物（たてもの）は100年前に**建てられました**。  
  (이 건물은 100년 전에 세워졌습니다.)
- 新しいルールが**決められました**。 (새 규칙이 정해졌습니다.)

---

## 수동형 정중 활용

수동형 동사는 る동사처럼 활용한다.

| 형태      | 예 (叱る → 叱られる) |
| --------- | -------------------- |
| 현재 정중 | 叱られます           |
| 과거 정중 | 叱られました         |
| 부정      | 叱られません         |
| て형      | 叱られて             |

---

## 수동형 vs 가능형 혼동 주의

2그룹 동사에서 수동형과 가능형 형태가 같다!

- 食べられる → **수동:** (누군가에게) 먹힌다
- 食べられる → **가능:** 먹을 수 있다

문맥으로 구분한다.

- 魚が猫に**食べられた**。 (생선이 고양이에게 먹혔다.) — 수동
- この魚は生で**食べられる**。 (이 생선은 날로 먹을 수 있다.) — 가능

## 핵심 요약

| 종류      | 뉘앙스      | 예               |
| --------- | ----------- | ---------------- |
| 직접 수동 | 내가 당함   | 先生に褒められた |
| 피해 수동 | 피해를 입음 | 雨に降られた     |
| 물건 주어 | 문어체      | 書かれた本       |
