た형은 동사의 **과거형 평어체**다. て형과 만드는 규칙이 거의 같아서, て형을 알면 た형은 바로 따라온다.

- て형: 書いて, 食べて, して
- た형: 書いた, 食べた, した

어미만 `て → た`, `で → だ`로 바꾸면 된다.

---

## 그룹별 た형 만들기

### 2그룹 동사 — る → た

| 사전형 | て형   | た형   |
| ------ | ------ | ------ |
| 食べる | 食べて | 食べた |
| 見る   | 見て   | 見た   |
| 起きる | 起きて | 起きた |
| 着る   | 着て   | 着た   |
| 教える | 教えて | 教えた |

### 1그룹 동사 — て형 규칙과 동일

て형 어미를 `て→た`, `で→だ`로 바꾼다.

| 어미     | て형 어미 | た형 어미 | 예                                            |
| -------- | --------- | --------- | --------------------------------------------- |
| く       | いて      | いた      | 書く → 書いた                                 |
| ぐ       | いで      | いだ      | 泳ぐ → 泳いだ                                 |
| す       | して      | した      | 話す → 話した                                 |
| つ/る/う | って      | った      | 待つ → 待った / 帰る → 帰った / 買う → 買った |
| ぬ/ぶ/む | んで      | んだ      | 死ぬ → 死んだ / 遊ぶ → 遊んだ / 飲む → 飲んだ |

> **예외:** 行く → 行って (行いて가 아님)

### 3그룹 동사 (불규칙)

| 사전형 | た형 |
| ------ | ---- |
| する   | した |
| くる   | きた |

## た형 기본 사용법

### 과거 사실 표현 (평어체)

공식적인 자리가 아닌 일상 대화, 일기, 메모 등에서 쓴다.

- 昨日、映画を**見た**。 (어제 영화 봤다.)
- 朝ごはんを**食べた**。 (아침밥 먹었다.)
- 友達と**会った**。 (친구와 만났다.)

**정중체로 바꾸면:** `〜た → 〜ました`

## た형 파생 문형

### 1. 〜たことがある — 〜한 적이 있다 (경험)

```
た형 + ことがある
```

- 富士山に**登ったことがあります**。 (후지산에 오른 적이 있어요.)
- お寿司を食べ**たことがありますか**。 (초밥을 먹어본 적 있어요?)
- 日本に行っ**たことがない**。 (일본에 가본 적 없어.)

**경험 없음:** `〜たことがない / ありません`

### 2. 〜たり〜たりする — 〜하거나 〜하거나 한다

여러 동작을 예시로 나열할 때 쓴다. 전부 나열하지 않고 "이런 것들을 한다"는 뉘앙스.

```
た형 + り、た형 + りする
```

- 週末は映画を見**たり**、音楽を聴い**たりします**。  
  (주말에는 영화 보거나 음악 듣거나 해요.)
- 泣い**たり**、笑っ**たり**、忙しい一日でした。  
  (울었다 웃었다, 바쁜 하루였어요.)

### 3. 〜たら — 만약 〜했으면 / 〜하면

가정·조건을 나타낸다. "그 일이 완료된 후"에 일어나는 일을 말할 때 특히 자연스럽다.

```
た형 + ら
```

- 家に帰っ**たら**、電話してください。 (집에 돌아가면 전화해 주세요.)
- 宿題が終わっ**たら**、遊んでいいです。 (숙제가 끝나면 놀아도 돼요.)
- もし宝くじが当たっ**たら**、旅行したい。 (만약 복권이 당첨되면 여행하고 싶다.)

### 4. 〜たあとで — 〜한 후에

```
た형 + あとで
```

- 食べ**たあとで**、散歩しましょう。 (먹은 후에 산책합시다.)
- 授業が終わっ**たあとで**、図書館に行きます。 (수업이 끝난 후에 도서관에 갈게요.)

> `〜てから`(て형+から)와 비슷하지만,  
> `〜てから` 는 순서 강조, `〜たあとで`는 완료 후 상황 강조.

## た형 전체 요약

| 문형             | 의미                   |
| ---------------- | ---------------------- |
| 〜た             | 〜했다 (과거)          |
| 〜たことがある   | 〜한 적이 있다         |
| 〜たり〜たりする | 〜하거나 〜하거나 한다 |
| 〜たら           | 〜하면 (가정·조건)     |
| 〜たあとで       | 〜한 후에              |

## て형 vs た형 비교

| 동사   | て형   | た형   |
| ------ | ------ | ------ |
| 書く   | 書いて | 書いた |
| 飲む   | 飲んで | 飲んだ |
| 食べる | 食べて | 食べた |
| する   | して   | した   |
| くる   | きて   | きた   |

て형에서 `て→た`, `で→だ`만 바꾸면 된다.
