일본어 지시대명사는 **こ·そ·あ·ど** 4계열로 이루어진다.

| 계열 | 거리                        | 의미    |
| ---- | --------------------------- | ------- |
| こ   | 화자 가까이                 | 이〜    |
| そ   | 청자 가까이 / 화자에서 멀리 | 그〜    |
| あ   | 화자·청자 모두 멀리         | 저〜    |
| ど   | 의문                        | 어느〜? |

한국어 이/그/저/어 와 거의 같은 구조다. 다만 일본어는 "그"(そ)가 상대방 쪽을 가리키기도 한다는 점이 차이다.

---

## 1. これ·それ·あれ·どれ — 물건

| 한국어   | 일본어 | 읽기 |
| -------- | ------ | ---- |
| 이것     | これ   | これ |
| 그것     | それ   | それ |
| 저것     | あれ   | あれ |
| 어느 것? | どれ   | どれ |

- **これ**は何（なん）ですか。 (이게 뭐예요?)
- **それ**は私（わたし）のです。 (그건 제 거예요.)
- **あれ**は富士山（ふじさん）ですか。 (저게 후지산이에요?)
- **どれ**が田中（たなか）さんのですか。 (어느 게 다나카 씨 거예요?)

---

## 2. この·その·あの·どの — 명사 수식

これ/それ/あれ에 명사가 뒤따를 때는 `この/その/あの/どの`로 바꾼다.

| 단독 | 수식형 | 예                               |
| ---- | ------ | -------------------------------- |
| これ | この   | この本（ほん） 이 책             |
| それ | その   | その鞄（かばん） 그 가방         |
| あれ | あの   | あの人（ひと） 저 사람           |
| どれ | どの   | どの電車（でんしゃ）? 어느 전철? |

- **この**映画（えいが）はおもしろいです。 (이 영화는 재미있어요.)
- **あの**レストランはどうですか。 (저 레스토랑은 어때요?)
- **どの**バスに乗（の）ればいいですか。 (어느 버스를 타면 돼요?)

---

## 3. ここ·そこ·あそこ·どこ — 장소

| 한국어 | 일본어 | 읽기   |
| ------ | ------ | ------ |
| 여기   | ここ   | ここ   |
| 거기   | そこ   | そこ   |
| 저기   | あそこ | あそこ |
| 어디?  | どこ   | どこ   |

- トイレは**どこ**ですか。 (화장실은 어디예요?)
- **ここ**に座（すわ）ってください。 (여기에 앉아 주세요.)
- **あそこ**に山（やま）が見（み）えます。 (저기에 산이 보여요.)
- 本（ほん）は**そこ**にあります。 (책은 거기에 있어요.)

---

## 4. こちら·そちら·あちら·どちら — 방향 / 공손한 쪽 지시

장소나 방향을 공손하게 말할 때, 또는 사람을 소개할 때 쓴다.

| 일본어 | 의미                |
| ------ | ------------------- |
| こちら | 이쪽 / 이분         |
| そちら | 그쪽 / 그분         |
| あちら | 저쪽 / 저분         |
| どちら | 어느 쪽? / 어느 분? |

- 入口（いりぐち）は**こちら**です。 (입구는 이쪽입니다.)
- **こちら**は鈴木（すずき）さんです。 (이분은 스즈키 씨입니다.) — 소개
- **どちら**からいらっしゃいましたか。 (어디서 오셨어요? — 공손)

> `どちら`는 "어느 쪽" 뜻도 있어서 두 가지 중 하나를 고를 때도 쓴다.
>
> - コーヒーと紅茶（こうちゃ）、**どちら**がいいですか。 (커피와 홍차 중 어느 쪽이 좋아요?)

---

## 5. こんな·そんな·あんな·どんな — 종류 / 성질

어떤 **종류나 성질**을 가리킬 때 쓴다.

| 일본어 | 의미  |
| ------ | ----- |
| こんな | 이런  |
| そんな | 그런  |
| あんな | 저런  |
| どんな | 어떤? |

- **こんな**問題（もんだい）は簡単（かんたん）です。 (이런 문제는 쉬워요.)
- **どんな**音楽（おんがく）が好（す）きですか。 (어떤 음악을 좋아해요?)
- **そんな**こと言（い）わないでください。 (그런 말 하지 마세요.)

## 전체 정리표

| 용도        | こ     | そ     | あ     | ど     |
| ----------- | ------ | ------ | ------ | ------ |
| 물건 (단독) | これ   | それ   | あれ   | どれ   |
| 물건 (수식) | この   | その   | あの   | どの   |
| 장소        | ここ   | そこ   | あそこ | どこ   |
| 방향/공손   | こちら | そちら | あちら | どちら |
| 종류/성질   | こんな | そんな | あんな | どんな |

---

## 한국인이 헷갈리는 포인트

### そ 계열 = 상대방 쪽

한국어 "그것"은 화자에서 먼 것이지만,  
일본어 `それ`는 **청자(상대방) 가까이**에 있는 것도 가리킨다.

- (상대방 손에 있는 걸 가리키며) **それ**を見せてください。 (그거 보여 주세요.)

한국어라면 "그거"도 되고 "이거"도 될 수 있지만, 일본어는 위치 기준을 엄격하게 쓴다.

### あ 계열 = 공유 지식

`あれ/あの/あそこ`는 물리적으로 먼 것뿐만 아니라,  
**대화 참여자 모두가 아는 것**을 가리킬 때도 쓴다.

- **あの**映画、見た? (그 영화 봤어? — 둘 다 아는 영화)
