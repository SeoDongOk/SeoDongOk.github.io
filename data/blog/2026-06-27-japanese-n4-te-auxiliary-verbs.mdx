て형 뒤에 동사를 하나 더 붙이면, 그 동사의 **원래 뜻은 옅어지고 문법적인 의미만 남는다**. 이걸 보조동사라고 한다. `〜てみる`(해보다)는 N5에서 다뤘으니, 여기서는 N4 빈출 4종 — `〜てしまう`·`〜ておく`·`〜てある`·`〜ていく/てくる`를 정리한다. 특히 **〜てある vs 〜ている** 구분은 N4 시험 단골이다.

---

## ① 〜てしまう — 해버리다 (완료 / 후회·유감)

두 가지 뉘앙스가 있다. 문맥으로 구분한다.

```
동사 て형 + しまう
(で형 동사는 + でしまう)
```

**(1) 완료 — 끝까지 다 했다:**

- 宿題（しゅくだい）を全部（ぜんぶ）やっ**てしまった**。  
  (숙제를 전부 해버렸다. — 완료의 후련함)
- この本（ほん）は一日（いちにち）で読（よ）ん**でしまった**。  
  (이 책은 하루 만에 다 읽어버렸다.)

**(2) 후회·유감 — 의도치 않게 그렇게 됐다:**

- 電車（でんしゃ）に傘（かさ）を忘（わす）れ**てしまった**。  
  (전철에 우산을 두고 내려버렸다. — 아차 싶은 유감)
- 大事（だいじ）なお皿（さら）を割（わ）っ**てしまった**。  
  (소중한 접시를 깨버렸다.)

> 💡 **회화체 축약**: `〜てしまう → 〜ちゃう` / `〜でしまう → 〜じゃう`  
> 食べてしまった → 食べ**ちゃった** / 飲んでしまった → 飲ん**じゃった**

---

## ② 〜ておく — 미리 해두다 (준비·조치)

어떤 목적을 위해 **미리 준비**하거나, 어떤 상태를 **그대로 유지**할 때.

```
동사 て형 + おく
```

**예문:**

- 旅行（りょこう）の前（まえ）に、ホテルを予約（よやく）し**ておく**。  
  (여행 전에 호텔을 예약해 둔다. — 준비)
- お客（きゃく）さんが来（く）る前（まえ）に、掃除（そうじ）し**ておきます**。  
  (손님이 오기 전에 청소해 둡니다.)
- 暑（あつ）いから、エアコンをつけ**ておいて**。  
  (더우니까 에어컨 켜 둬. — 상태 유지)

> 💡 **회화체 축약**: `〜ておく → 〜とく`  
> 予約しておく → 予約**しとく** / 買っておく → 買っ**とく**

---

## ③ 〜てある — 〜되어 있다 (의도적 결과 상태) ★시험 포인트

**타동사** て형 + ある. 누군가 **어떤 목적을 가지고 해 놓은 결과 상태**를 나타낸다.

```
타동사 て형 + ある
```

**예문:**

- 壁（かべ）に地図（ちず）が掛（か）け**てある**。  
  (벽에 지도가 걸려 있다. — 누군가 일부러 걸어 놓음)
- テーブルに花（はな）が飾（かざ）っ**てある**。  
  (테이블에 꽃이 장식되어 있다.)
- 会議（かいぎ）の資料（しりょう）はもう印刷（いんさつ）し**てあります**。  
  (회의 자료는 이미 인쇄해 두었습니다.)

### 〜てある vs 〜ている — 자동사/타동사가 갈림길

| 구분     | 형태                | 의미                       | 예                                      |
| -------- | ------------------- | -------------------------- | --------------------------------------- |
| 〜ている | **자동사** + ている | 자연히 그렇게 된 상태      | ドアが**開いている** (문이 열려 있다)   |
| 〜てある | **타동사** + てある | 누가 의도적으로 해 둔 상태 | ドアが**開けてある** (문을 열어 두었다) |

- 窓（まど）が**開（あ）いている**。 → 自動詞 `開く`. 그냥 열려 있는 상태 (바람에 열렸을 수도)
- 窓（まど）が**開（あ）けてある**。 → 他動詞 `開ける`. 환기 등 **목적이 있어 일부러 열어 둔** 상태

> 💡 **핵심**: `てある`에는 "한 사람의 의도·목적"이 숨어 있다. 그래서 항상 **타동사**와 짝을 이룬다.

---

## ④ 〜ていく / 〜てくる — 방향·변화

공간적 이동과 시간적 변화 두 축으로 쓴다.

```
동사 て형 + いく / くる
```

**(1) 공간 — 이동 방향:**

- お弁当（べんとう）を持（も）っ**ていく**。 (도시락을 가져가다. — 화자에게서 멀어짐)
- 友達（ともだち）が家（いえ）に遊（あそ）びに来（き）て、ケーキを持（も）っ**てきた**。 (친구가 케이크를 가져왔다. — 화자 쪽으로)

**(2) 시간 — 변화의 방향:**

- これから、寒（さむ）くなっ**ていく**でしょう。  
  (앞으로 추워져 갈 거예요. — 현재→미래)
- だんだん日本語（にほんご）が分（わ）かっ**てきた**。  
  (점점 일본어가 이해되기 시작했다. — 과거→현재)
- 急（きゅう）に雨（あめ）が降（ふ）っ**てきた**。  
  (갑자기 비가 오기 시작했다. — 출현)

> 💡 **방향 감각**: `ていく`는 "지금에서 저쪽(미래)으로", `てくる`는 "저쪽(과거)에서 지금으로". 변화가 어느 쪽을 향하는지로 고른다.

---

## 한눈에 비교

| 표현              | 의미        | 핵심 포인트                |
| ----------------- | ----------- | -------------------------- |
| 〜てしまう        | 해버리다    | 완료 or 후회 (회화 ちゃう) |
| 〜ておく          | 미리 해두다 | 준비·조치 (회화 とく)      |
| 〜てある          | 〜되어 있다 | **타동사** + 의도적 상태   |
| 〜ていく / てくる | 방향·변화   | 공간/시간 어느 쪽 향하나   |

---

## 핵심 요약

- **てしまう** = 완료의 후련함 or 의도치 않은 유감 → 회화에선 ちゃう/じゃう
- **ておく** = 목적을 위한 사전 준비 → 회화에선 とく
- **てある** = 누가 일부러 해 놓은 상태 → **반드시 타동사**, `ている`(자동사)와 구분
- **ていく/てくる** = 멀어지면 いく, 다가오면 くる (공간·시간 모두)

> **JLPT 시험 팁**: `窓が開いている`(자동사) vs `窓が開けてある`(타동사)처럼, **자동사+ている / 타동사+てある** 짝을 묻는 문제가 단골이다. 동사가 自動詞인지 他動詞인지부터 보자.
