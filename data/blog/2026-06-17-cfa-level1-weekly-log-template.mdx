> CFA 준비는 길다. 그래서 매주 기록하지 않으면 무엇을 했고, 무엇을 놓쳤는지 금방 흐려진다. 이 글은 앞으로 사용할 **CFA Level 1 주간 공부 로그 템플릿**이다.

## 주간 로그의 목적

주간 로그는 예쁜 회고가 아니라 실행 관리 도구다.

- 누적 공부시간 확인
- 과목별 진도 확인
- 오답 패턴 확인
- 다음 주 계획 수립
- 장기전에서 동기 유지

특히 CFA는 범위가 넓기 때문에, “열심히 했다”보다 “어떤 과목을 얼마나 했는지”가 더 중요하다.

## 주간 공부시간 기준

기본 목표는 주 10시간이다.

| 결과        | 판단                   |
| ----------- | ---------------------- |
| 12시간 이상 | 계획 초과 달성         |
| 9~11시간    | 정상 범위              |
| 6~8시간     | 보완 필요              |
| 5시간 이하  | 다음 주 복구 계획 필요 |

중요한 것은 한 주 실패했다고 전체 계획을 포기하지 않는 것이다. CFA 준비는 일간 목표보다 주간 목표로 관리하는 편이 현실적이다.

## 과목별 기록 방식

각 과목은 다음 기준으로 기록한다.

| 기록 항목 | 예시                     |
| --------- | ------------------------ |
| 진도      | FSA 현금흐름표 1회독     |
| 문제      | QBank 40문제             |
| 정답률    | 65%                      |
| 오답 유형 | CFO/CFI 분류 혼동        |
| 다음 조치 | 현금흐름표 분류표 재정리 |

정답률은 좋고 나쁨을 평가하기 위한 것이 아니라, 다음 주 시간을 어디에 쓸지 정하기 위한 데이터다.

## 오답 유형 태그

오답노트에는 다음 태그를 붙인다.

- `concept`: 개념 부족
- `formula`: 공식 암기 또는 적용 실패
- `calculator`: 계산기 입력 실수
- `reading`: 문제 문장 오독
- `memory`: 단순 암기 부족
- `speed`: 시간 부족

예시:

```text
[FSA][reading] IFRS 조건을 US GAAP처럼 읽음
[Quant][calculator] TVM 부호 입력 실수
[Ethics][concept] disclosure와 consent 구분 실패
```

## 월간 리뷰와 연결

주간 로그 4개가 모이면 월간 리뷰를 쓴다. 월간 리뷰에서는 다음을 확인한다.

1. 누적 공부시간
2. 과목별 시간 배분
3. 정답률 변화
4. 반복 오답 TOP 5
5. 다음 달 전략 수정

주간 로그는 원재료이고, 월간 리뷰는 전략 수정이다.

## 공개 기록의 장점

블로그에 공부 로그를 남기면 약간의 부담이 생긴다. 하지만 그 부담이 루틴을 유지하게 만든다.

- 공부를 미룬 주가 보인다.
- 약점 과목을 회피했는지 드러난다.
- 시간이 지나면 실제 성장 기록이 된다.
- 시험 후 어떤 방식이 효과적이었는지 복기할 수 있다.

## 첫 주 목표

첫 주는 거창하게 시작하지 않는다.

- 시험 구조 정리
- 과목별 비중표 정리
- Quant 시작
- FSA 시작
- 공부 기록표 만들기

목표는 “공부를 많이 하기”가 아니라 **공부 시스템을 시작하기**다.

## 결론

CFA Level 1 준비는 단기 암기가 아니라 장기 프로젝트다. 주간 로그는 그 프로젝트의 대시보드 역할을 한다. 앞으로 이 템플릿을 기준으로 실제 공부 기록을 남길 예정이다.
