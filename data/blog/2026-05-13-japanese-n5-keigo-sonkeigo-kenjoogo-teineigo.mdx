마스형과 て形을 잡았다면 이제 일본어의 **예절 시스템**과 마주칠 차례다.

일본어 경어(敬語／けいご)는 단순히 「말을 공손하게 하는 것」이 아니다. **누구의 행동인지**에 따라 쓰는 단어 자체가 달라진다. 이 원리를 모르면 아무리 정중하게 말하려 해도 오히려 실례가 되는 상황이 생긴다.

## 1. 경어(敬語)의 세 종류

일본어 경어는 크게 세 가지로 나뉜다.

| 경어 종류  | 일본어                 | 역할                 | 방향            |
| ---------- | ---------------------- | -------------------- | --------------- |
| **정중어** | 丁寧語（ていねいご）   | 말투 자체를 정중하게 | 말투를 높임     |
| **존경어** | 尊敬語（そんけいご）   | 상대방의 행동을 높임 | 상대방 ↑        |
| **겸양어** | 謙譲語（けんじょうご） | 나의 행동을 낮춤     | 나 ↓ → 상대방 ↑ |

한국어와 대응시키면 이해가 빠르다.

- 「먹어요」 → 정중어 (말투만 높임)
- 「드세요」 → 존경어 (상대방 행동을 높임)
- 「제가 먹겠습니다」 → 겸양어 (내 행동을 낮춰 상대를 높임)

핵심 원리는 딱 하나다.

> **"이 동작의 주인이 누구냐?"**
> → 상대방이면 존경어 / 나면 겸양어 / 말투만 높이면 정중어

## 2. 정중어(丁寧語) — です・ます

가장 기본적인 경어다. 상대방이나 나를 높이는 게 아니라, **말투 자체를 정중하게** 만든다. N5에서 배운 です・ます형이 바로 정중어다.

### 2-1. です・ます형

| 보통체       | 정중어     |
| ------------ | ---------- |
| 食（た）べる | 食べます   |
| 行（い）く   | 行きます   |
| きれいだ     | きれいです |

### 2-2. お〜 / ご〜 접두사

명사 앞에 「お」나 「ご」를 붙여 정중하게 만들 수 있다.

| 보통               | 정중어                 | 뜻   |
| ------------------ | ---------------------- | ---- |
| 名前（なまえ）     | お名前（おなまえ）     | 이름 |
| 電話（でんわ）     | お電話（おでんわ）     | 전화 |
| 水（みず）         | お水（おみず）         | 물   |
| 飯（めし）         | ご飯（ごはん）         | 밥   |
| 住所（じゅうしょ） | ご住所（ごじゅうしょ） | 주소 |
| 家族（かぞく）     | ご家族（ごかぞく）     | 가족 |

대체로 **和語(일본 고유어)** 에는 「お」, **漢語(한자어)** 에는 「ご」를 붙인다. 단, 예외도 많으므로 자주 쓰는 단어는 세트로 외우는 편이 낫다.

---

## 3. 존경어(尊敬語) — 상대방 행동 높이기

선생님, 사장님, 손님 등 **상대방이 하는 행동**을 높일 때 쓴다.

### 3-1. 특수 존경 동사

존경어에는 동사 자체를 바꾸는 특수 형태가 있다. 이 목록은 반드시 외워야 한다.

| 보통형       | 존경어                 | 뜻          |
| ------------ | ---------------------- | ----------- |
| いる         | いらっしゃる           | 계시다      |
| 行（い）く   | いらっしゃる           | 가시다      |
| 来（く）る   | いらっしゃる           | 오시다      |
| する         | なさる                 | 하시다      |
| 言（い）う   | おっしゃる             | 말씀하시다  |
| 食（た）べる | 召（め）し上（あ）がる | 드시다      |
| 飲（の）む   | 召し上がる             | 드시다      |
| くれる       | くださる               | 주시다      |
| 見（み）る   | ご覧（ごらん）になる   | 보시다      |
| 知（し）る   | ご存知（ごぞんじ）だ   | 알고 계시다 |

> 💡 「いらっしゃる」 하나가 いる・行く・来る 세 가지를 전담한다. 문맥으로 구분한다.

### 3-2. 일반 동사 존경어 공식

특수 동사 목록에 없는 동사는 다음 공식을 쓴다.

```
お + 동사(ます형 어간) + になる
```

| 보통형       | 존경어               | 뜻         |
| ------------ | -------------------- | ---------- |
| 読（よ）む   | お読（よ）みになる   | 읽으시다   |
| 使（つか）う | お使（つか）いになる | 사용하시다 |
| 待（ま）つ   | お待（ま）ちになる   | 기다리시다 |
| 帰（かえ）る | お帰（かえ）りになる | 돌아가시다 |

### 3-3. 존경어 실전 예문

1. 先生（せんせい）はいらっしゃいますか。 _(선생님은 계십니까?)_
2. 社長（しゃちょう）はもう帰宅（きたく）なさいました。 _(사장님은 이미 귀가하셨습니다.)_
3. 何（なに）をおっしゃいましたか。 _(무슨 말씀을 하셨습니까?)_
4. どうぞ、召し上がってください。 _(자, 드세요.)_
5. この映画（えいが）をご覧（ごらん）になりましたか。 _(이 영화를 보셨습니까?)_

---

## 4. 겸양어(謙譲語) — 나의 행동 낮추기

**내가 하는 행동**을 낮춰서 표현함으로써 상대방을 간접적으로 높인다. 비즈니스 일본어에서 특히 자주 쓰인다.

### 4-1. 특수 겸양 동사

| 보통형                     | 겸양어                 | 뜻                  |
| -------------------------- | ---------------------- | ------------------- |
| いる                       | おる                   | 있다 (저는~)        |
| 行（い）く                 | 参（まい）る           | 가다 (제가~)        |
| 来（く）る                 | 参（まい）る           | 오다 (제가~)        |
| する                       | いたす                 | 하다 (제가~)        |
| 言（い）う                 | 申（もう）す           | 말하다 (제가~)      |
| 食（た）べる・飲（の）む   | いただく               | 먹다/마시다 (제가~) |
| もらう                     | いただく               | 받다 (제가~)        |
| あげる                     | 差（さ）し上（あ）げる | 드리다              |
| 見（み）る                 | 拝見（はいけん）する   | 보다 (제가~)        |
| 聞（き）く・訪（たず）ねる | 伺（うかが）う         | 듣다/방문하다       |
| 会（あ）う                 | お目（め）にかかる     | 만나다              |
| 知（し）る                 | 存（ぞん）じる         | 알다                |

### 4-2. 일반 동사 겸양어 공식

```
お + 동사(ます형 어간) + する
```

| 보통형         | 겸양어             | 뜻             |
| -------------- | ------------------ | -------------- |
| 送（おく）る   | お送（おく）りする | 보내다 (제가~) |
| 持（も）つ     | お持（も）ちする   | 들다 (제가~)   |
| 伝（つた）える | お伝（つた）えする | 전하다 (제가~) |

### 4-3. 겸양어 실전 예문

1. 私（わたし）は田中（たなか）と申（もう）します。 _(저는 다나카라고 합니다.)_
2. 資料（しりょう）を拝見（はいけん）しました。 _(자료를 확인했습니다.)_
3. 明日（あした）、そちらに参（まい）ります。 _(내일 그쪽으로 가겠습니다.)_
4. お荷物（にもつ）をお持（も）ちしましょうか。 _(짐을 들어 드릴까요?)_
5. 先生（せんせい）にお目（め）にかかれて光栄（こうえい）です。 _(선생님을 뵙게 되어 영광입니다.)_

## 5. 3가지 경어 나란히 비교

같은 동사를 세 경어로 표현하면 차이가 확실하게 보인다.

### 来る (오다)

| 경어 종류 | 표현                   | 설명              |
| --------- | ---------------------- | ----------------- |
| 보통체    | 先生が来る             | 선생님이 오다     |
| 정중어    | 先生が来ます           | 선생님이 옵니다   |
| 존경어    | 先生がいらっしゃいます | 선생님이 오십니다 |

### 行く (가다) — 나의 행동

| 경어 종류 | 표현                 | 설명            |
| --------- | -------------------- | --------------- |
| 보통체    | 私が行く             | 내가 가다       |
| 정중어    | 私が行きます         | 제가 갑니다     |
| 겸양어    | 私が参（まい）ります | 제가 가겠습니다 |

### 食べる (먹다) — 주어에 따라 달라짐

| 경어 종류       | 표현                       | 설명       |
| --------------- | -------------------------- | ---------- |
| 보통체          | 食べる                     | 먹다       |
| 정중어          | 食べます                   | 먹습니다   |
| 존경어 (상대방) | 召（め）し上（あ）がります | 드십니다   |
| 겸양어 (나)     | いただきます               | 먹겠습니다 |

## 6. 한국인이 자주 틀리는 포인트

### ① 자기 자신에게 존경어 사용

```
❌ 私はいらっしゃいます。 (나는 계십니다?? → 자신에게 존경어)
⭕ 私はおります。 (나는 있습니다 — 겸양어 おる)
```

### ② 상대방에게 겸양어 사용

```
❌ 先生が参ります。 (선생님이 가겠습니다?? → 상대에게 겸양어)
⭕ 先生がいらっしゃいます。 (선생님이 가십니다 — 존경어)
```

### ③ いただく vs 召し上がる 혼동

| 표현                 | 주어   | 의미                       |
| -------------------- | ------ | -------------------------- |
| 私がいただきます     | 나     | 제가 먹겠습니다 (겸양어)   |
| 先生が召し上がります | 선생님 | 선생님이 드십니다 (존경어) |

```
❌ 先生がいただきます。
⭕ 先生が召し上がります。
```

### ④ 申す vs おっしゃる 혼동

| 표현                 | 주어   | 의미                           |
| -------------------- | ------ | ------------------------------ |
| 私が申します         | 나     | 제가 말씀드립니다 (겸양어)     |
| 先生がおっしゃいます | 선생님 | 선생님이 말씀하십니다 (존경어) |

### ⑤ いただきます의 두 가지 쓰임

| 상황            | 뜻                        |
| --------------- | ------------------------- |
| 밥 먹기 전 인사 | 잘 먹겠습니다 (관용 표현) |
| 동사로 사용     | 받다 / 먹다 (겸양어)      |

## 7. 핵심 요약 카드

| 경어 종류  | 공식                        | 대표 동사                                                      |
| ---------- | --------------------------- | -------------------------------------------------------------- |
| **정중어** | 〜です / 〜ます + お/ご〜   | —                                                              |
| **존경어** | お〜になる / 특수 존경 동사 | いらっしゃる, なさる, おっしゃる, 召し上がる, くださる         |
| **겸양어** | お〜する / 특수 겸양 동사   | おる, 参る, いたす, 申す, いただく, 差し上げる, 拝見する, 伺う |

최소한 아래 10쌍만 확실히 구분하면 일상 회화와 N4 시험에서 크게 막히지 않는다.

| 동사               | 존경어       | 겸양어      |
| ------------------ | ------------ | ----------- |
| いる / 行く / 来る | いらっしゃる | おる / 参る |
| する               | なさる       | いたす      |
| 言う               | おっしゃる   | 申す        |
| 食べる / 飲む      | 召し上がる   | いただく    |
| もらう             | —            | いただく    |
| くれる / あげる    | くださる     | 差し上げる  |
| 見る               | ご覧になる   | 拝見する    |
