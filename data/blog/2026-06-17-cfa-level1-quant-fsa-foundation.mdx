> CFA Level 1 준비 초반에는 어떤 과목부터 볼지가 중요하다. 내 기준에서는 **Quant와 FSA를 먼저 잡는 것**이 전체 공부 효율을 높인다.

## 왜 Quant인가

Quantitative Methods는 단독 과목으로도 중요하지만, 다른 과목의 언어가 된다.

- 수익률 계산
- 현재가치와 미래가치
- 확률과 통계
- 분산과 표준편차
- 공분산과 상관계수
- 가설검정
- 회귀분석
- CAPM과 포트폴리오 이론

이 개념들은 Portfolio Management, Equity, Fixed Income, Derivatives로 계속 이어진다. Quant를 모르면 뒤 과목의 수식이 갑자기 낯설어진다.

## Quant 학습 순서

내가 잡은 Quant 순서는 다음이다.

1. 수익률의 종류
2. Time Value of Money
3. 통계량: 평균, 분산, 표준편차
4. 확률분포와 기대값
5. 가설검정
6. 포트폴리오 수익과 위험
7. 단순회귀분석
8. 금융 데이터 사이언스 개요

초반에는 이론을 깊게 파기보다, 계산기 입력과 공식 적용을 같이 훈련한다.

## Quant 오답 포인트

| 오답 원인          | 대응                      |
| ------------------ | ------------------------- |
| 공식 혼동          | 공식 카드 작성            |
| 계산기 입력 실수   | 입력 순서 고정            |
| 단위 혼동          | 연율화 여부 표시          |
| 통계 용어 혼동     | 정의와 해석을 같이 기록   |
| 회귀분석 해석 실수 | 계수, 설명력, 잔차를 구분 |

Quant는 “이해했다”보다 “문제에서 바로 적용된다”가 중요하다.

## 왜 FSA인가

Financial Statement Analysis는 Level I에서 가장 시간이 많이 들어갈 가능성이 높은 과목이다. 재무제표를 읽는 기본 체계뿐 아니라, 회계처리와 재무비율 분석까지 다룬다.

핵심 범위는 다음과 같다.

- 손익계산서 분석
- 재무상태표 분석
- 현금흐름표 분석
- 재고자산
- 장기자산
- 리스와 부채
- 법인세
- 재무보고 품질
- 재무비율
- 재무제표 모델링 기초

FSA는 단순히 외울 내용이 많아서 어려운 것이 아니다. 하나의 회계처리가 세 재무제표와 비율에 어떤 영향을 주는지 연결해야 한다.

## FSA 학습 순서

내 FSA 학습 순서는 다음이다.

1. 세 재무제표의 연결 구조
2. 손익계산서와 수익·비용 인식
3. 재무상태표 주요 계정
4. 현금흐름표 직접법·간접법
5. 재고와 장기자산
6. 리스, 부채, 세금
7. 재무비율과 DuPont 분석
8. 보고 품질과 조작 신호
9. 간단한 재무 모델링

처음부터 세부 회계처리에 들어가기보다, 세 재무제표가 어떻게 연결되는지 먼저 잡는 것이 좋다.

## FSA 오답 포인트

| 오답 원인                | 대응                              |
| ------------------------ | --------------------------------- |
| IFRS와 US GAAP 차이 혼동 | 차이표 작성                       |
| CFO, CFI, CFF 분류 실수  | 현금흐름표 분류표 반복            |
| 재고 평가 영향 혼동      | 인플레이션/디플레이션 케이스 비교 |
| 리스 회계처리 혼동       | 재무제표별 영향 기록              |
| 비율 공식 암기 부족      | 같은 기업 예제로 반복 계산        |

FSA는 반드시 표로 비교해야 한다. 문장으로만 외우면 시험장에서 헷갈린다.

## Quant와 FSA를 연결해서 공부하기

두 과목은 성격이 다르지만, 공통점이 있다.

- 계산이 나온다.
- 정의를 정확히 알아야 한다.
- 숫자를 해석해야 한다.
- 문제풀이 반복이 중요하다.

그래서 나는 초반 6주 동안 Quant와 FSA를 병렬로 진행할 예정이다.

```text
월: Quant 개념
화: FSA 개념
수: Quant 문제
목: FSA 문제
금: 오답 정리
주말: 누적 복습
```

## 초반 목표

| 과목  | 6주 목표                                     |
| ----- | -------------------------------------------- |
| Quant | TVM, 통계, 가설검정까지 1회독                |
| FSA   | 세 재무제표, 현금흐름표, 주요 회계처리 1회독 |
| 공통  | 계산기 입력 루틴 만들기                      |
| 공통  | 오답노트 양식 고정                           |

초반부터 완벽하게 맞히려 하지 않는다. 대신 “어디서 틀리는지”를 빠르게 발견하는 것을 목표로 한다.

## 결론

CFA Level 1에서 Quant는 수학적 언어를 제공하고, FSA는 기업 분석의 기초를 제공한다. 두 과목을 초반에 잡아두면 뒤 과목을 공부할 때 훨씬 덜 흔들린다.

나의 첫 6주는 Quant와 FSA 중심으로 간다.
