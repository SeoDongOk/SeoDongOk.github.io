> CFA Level 1은 “어려운 한 과목” 시험이 아니라 **넓은 10개 과목을 끝까지 놓치지 않는 시험**에 가깝다. 그래서 과목별 비중을 보고 시간 배분을 먼저 정해야 한다.

## 공식 과목 비중

CFA Institute Level I 안내에 공개된 과목별 비중은 다음과 같다.

| 과목                               | 시험 비중 |
| ---------------------------------- | --------: |
| Ethical and Professional Standards |    15-20% |
| Financial Statement Analysis       |    11-14% |
| Equity Investments                 |    11-14% |
| Fixed Income                       |    11-14% |
| Portfolio Management               |     8-12% |
| Alternative Investments            |     7-10% |
| Quantitative Methods               |      6-9% |
| Economics                          |      6-9% |
| Corporate Issuers                  |      6-9% |
| Derivatives                        |      5-8% |

이 표만 보면 단순히 비중이 높은 순서대로 공부하면 될 것 같지만, 실제로는 세 가지 요소를 같이 봐야 한다.

1. 시험 비중
2. 누적 이해 필요성
3. 다른 과목과의 연결성

## 1순위 과목: Ethics, FSA, Fixed Income

### Ethics

Ethics는 비중이 가장 크다. 동시에 마지막까지 감각을 유지해야 하는 과목이다. 단순 암기보다 상황 판단형 문제가 많으므로, 초반에 한 번 보고 끝내면 감이 떨어진다.

공부 전략은 다음과 같다.

- 초반: 기준과 용어 파악
- 중반: 사례 문제 반복
- 후반: 매일 짧게 유지

### Financial Statement Analysis

FSA는 재무제표를 읽고 해석하는 과목이다. 범위가 넓고, 계산·회계처리·비율분석이 함께 나온다.

핵심은 다음이다.

- 손익계산서, 재무상태표, 현금흐름표 연결
- IFRS와 US GAAP 차이
- 재고, 장기자산, 리스, 세금, 부채
- 재무비율과 DuPont 분석

FSA는 뒤로 미루면 회복이 어렵다. 초반부터 꾸준히 봐야 한다.

### Fixed Income

Fixed Income은 채권 구조, 수익률, 듀레이션, 신용위험 등 투자 실무와 연결성이 크다. 공식도 있지만, 단순 공식 암기만으로는 부족하다.

공부할 때는 “가격과 수익률은 반대로 움직인다” 같은 직관에서 출발해, 점점 계산으로 확장하는 것이 좋다.

## 2순위 과목: Equity, Quant, Portfolio Management

### Equity Investments

Equity는 비교적 직관적이다. 주식, 시장, 지수, 산업분석, 기업분석, 밸류에이션이 중심이다. 하지만 DDM 같은 기본 밸류에이션 모델은 확실히 손에 익혀야 한다.

### Quantitative Methods

Quant는 단독 비중은 중간 정도지만, 여러 과목의 기반이 된다.

- 수익률
- 확률과 통계
- 가설검정
- 회귀분석
- 포트폴리오 위험과 수익

초반에 Quant를 정리해두면 Portfolio Management와 자산가격결정 개념이 훨씬 편해진다.

### Portfolio Management

Portfolio Management는 Level I에서는 기초 포트폴리오 이론, 위험과 수익, CAPM, 투자자 니즈가 중심이다. Quant와 연결해서 공부하면 좋다.

## 3순위 과목: Economics, Corporate Issuers, Derivatives, Alternatives

3순위라고 덜 중요하다는 뜻은 아니다. 다만 시간 투입 전략이 다르다.

### Economics

범위가 넓다. 미시, 거시, 통화정책, 재정정책, 환율, 국제무역, 지정학까지 다룬다. 깊게 파고들기보다 시험에 나오는 개념의 구분을 명확히 해야 한다.

### Corporate Issuers

기업지배구조, 운전자본, 자본예산, 자본구조 등이 나온다. NPV, IRR, WACC 같은 계산은 반복해서 손에 익힌다.

### Derivatives

비중은 낮지만 처음 접하면 낯설다. Forward, Futures, Options, Swaps의 구조와 기본 가치평가 논리를 이해해야 한다.

### Alternative Investments

대체투자는 헤지펀드, 사모펀드, 부동산, 원자재, 인프라 등 다양한 자산군을 비교하는 방식으로 접근한다. 암기할 것이 있지만, 비교표로 정리하면 효율이 올라간다.

## 시간 배분 초안

300시간을 기준으로 과목별 시간을 대략 배분하면 다음과 같다.

| 과목                    | 목표 시간 |
| ----------------------- | --------: |
| Ethics                  |    45시간 |
| FSA                     |    45시간 |
| Fixed Income            |    40시간 |
| Equity                  |    35시간 |
| Quant                   |    30시간 |
| Portfolio Management    |    25시간 |
| Economics               |    25시간 |
| Corporate Issuers       |    20시간 |
| Alternative Investments |    20시간 |
| Derivatives             |    15시간 |
| 총합                    |   300시간 |

실제 공부에서는 과목별 난이도에 따라 조정해야 한다. 특히 FSA와 Fixed Income은 계획보다 시간이 더 들어갈 가능성이 크다.

## 나의 과목별 학습 순서

1. Quant
2. FSA
3. Ethics
4. Fixed Income
5. Equity
6. Corporate Issuers
7. Portfolio Management
8. Economics
9. Derivatives
10. Alternative Investments

이 순서는 “공식 비중순”이 아니라 “기초 → 회계 → 윤리 → 자산군” 흐름이다. 처음부터 Ethics만 오래 붙잡기보다, 기본기를 잡고 다시 Ethics를 반복하는 식으로 갈 예정이다.

## 결론

CFA Level 1에서 중요한 것은 모든 과목을 같은 깊이로 파는 것이 아니다. **비중이 높고 회복이 어려운 과목에 먼저 시간을 쓰고, 비중이 낮은 과목은 비교표와 반복 문제로 효율화하는 것**이다.

과목별 비중표는 앞으로 매달 다시 확인하면서 시간 배분을 조정할 계획이다.
