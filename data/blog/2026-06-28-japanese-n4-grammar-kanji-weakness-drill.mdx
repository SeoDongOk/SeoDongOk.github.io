N4 문법을 틀리는 숨은 원인은 두 가지다. **① 비슷한 표현의 의미 차이를 못 잡는 것**, 그리고 **② 문장 속 한자(漢字)를 못 읽어서 뜻 자체를 놓치는 것**. 이 글은 그 둘을 한 번에 잡는 약점 공략 드릴이다. 먼저 헷갈림 포인트를 압축한 뒤, 한자 읽기를 정리하고, 실전 18문제로 마무리한다.

---

## 1. 유사표현 "한 줄 구분" 압축표

| 헷갈리는 짝                    | 구분 키                                              | 예                                                      |
| ------------------------------ | ---------------------------------------------------- | ------------------------------------------------------- |
| **ので / のに**                | 당연한 결과=**ので** / 의외·불만=**のに**            | 雨なので中止 / 雨なのに行く                             |
| **ために / ように**            | 의지동사=**ために** / 가능·무의지=**ように**         | 合格するために / 合格できるように                       |
| **らしい / ようだ**            | 정황·소문 근거=**らしい** / 직접 관찰=**ようだ**     | 来ないらしい / 来ないようだ                             |
| **そうだ(전문) / つもり**      | 들은 것 전달=**そうだ** / 의도(추측 아님)=**つもり** | 来るそうだ / 来るつもり                                 |
| **てしまう / ておく / てある** | 완료·후회 / 준비 / (타동사)의도된 상태               | 食べてしまった / 買っておく / 書いてある                |
| **と / ば / たら / なら**      | 자연결과 / 일반조건 / 순서·완료 / 화제전제           | 押すと開く / 安ければ買う / 着いたら電話 / 行くなら京都 |

> ⚠️ **`のに` 주의**: "~인데도(의외·불만)"일 때만! 뒤가 당연한 결과면 `ので`, 목적이면 `ために`다. (가장 많이 과용하는 함정)

## 2. 문장에 자주 나오는 N4 한자 읽기

문법 문제 지문에 단골로 등장하는 한자들이다. 읽기가 막히면 문제 자체가 안 풀린다.

| 한자 | 읽기       | 뜻        |
| ---- | ---------- | --------- |
| 経験 | けいけん   | 경험      |
| 予約 | よやく     | 예약      |
| 留学 | りゅうがく | 유학      |
| 練習 | れんしゅう | 연습      |
| 故障 | こしょう   | 고장      |
| 残業 | ざんぎょう | 야근      |
| 会議 | かいぎ     | 회의      |
| 資料 | しりょう   | 자료      |
| 都合 | つごう     | 사정·형편 |
| 用意 | ようい     | 준비      |
| 約束 | やくそく   | 약속      |
| 説明 | せつめい   | 설명      |

> 💡 **읽기 팁**: 음독 한자어는 **장음(ー)·촉음(っ)·탁음(゛)**에서 자주 틀린다. 経験=けい**けん**(장음 X 주의), 残業=ざん**ぎょう**(탁음+장음), 約束=やく**そく**(촉음 아님).

## 4. 틀리기 쉬운 포인트 최종 정리

- **のに 과용 금지**: 뒤가 "당연한 결과"면 ので, "목적"이면 ために. のに는 오직 "의외·불만".
- **추량 4종**: 정황·소문=らしい / 직접 관찰=ようだ / 전달=そうだ(보통형+) / 곧 일어날 듯=そうだ(ます형+, 양태). つもり는 추측이 아니라 **의도**.
- **て형 보조동사**: 미리=ておく / 의도된 상태(타동사)=てある / 완료·후회=てしまう.
- **조건**: 기계·자연결과=と / 일반=ば / 순서=たら / 화제전제·조언=なら.
- **한자 읽기**: 음독은 장음·촉음·탁음에서 갈린다. 눈으로만 읽지 말고 소리 내어 외우면 청해에도 도움.
