한국어 "A보다 B가 더 〜하다"를 일본어로 어떻게 말할까?

일본어 비교 표현은 기본 틀이 하나라서, 한 번 잡아두면 응용이 쉽다.

---

## 기본 비교: AよりBの方が〜

```
A より B の方が [형용사/형용동사]
      ↑              ↑
    "A보다"       "B쪽이 더"
```

- 電車（でんしゃ）**より** バスの方が安（やす）い。  
  (전철보다 버스가 더 싸다.)
- 夏（なつ）**より** 冬（ふゆ）の方が好（す）きです。  
  (여름보다 겨울이 더 좋아요.)
- コーヒー**より** お茶の方が健康（けんこう）にいいです。  
  (커피보다 차가 건강에 더 좋아요.)

> **어순 주의:** 한국어 "B가 A보다 더 〜"와 어순이 반대. 일본어는 `A より B の方が` 순서.

---

## 질문: AとBとどちらが〜

둘 중 어느 것이 더 〜한지 물을 때.

```
A と B と どちらが [형용사]ですか
```

- 英語（えいご）と日本語、**どちらが**難（むずか）しいですか。  
  (영어와 일본어, 어느 쪽이 더 어려워요?)
- 山（やま）と海（うみ）、**どちらが**好きですか。  
  (산과 바다, 어느 쪽이 더 좋아요?)

**대답:**

- 日本語の方が難しいと思います。 (일본어 쪽이 더 어렵다고 생각해요.)
- どちらも好きです。 (둘 다 좋아요.)
- どちらも好きじゃないです。 (둘 다 별로예요.)

---

## 최상급: 〜が一番〜

셋 이상에서 "가장 〜하다"를 말할 때.

```
[그룹] の中で [것]が 一番 [형용사]
```

- クラスの中で田中（たなか）さんが**一番**背（せ）が高（たか）いです。  
  (반에서 다나카 씨가 키가 가장 커요.)
- 季節（きせつ）の中で春（はる）が**一番**好きです。  
  (계절 중에서 봄이 가장 좋아요.)
- この店で何が**一番**おいしいですか。 (이 가게에서 뭐가 가장 맛있어요?)

---

## 비교급 질문과 대답 패턴

### 2개 비교

| 상황   | 표현                                |
| ------ | ----------------------------------- |
| 질문   | 〜と〜とどちらが〜ですか            |
| 대답   | 〜の方が〜です                      |
| 비슷함 | どちらも〜です / あまり変わりません |

### 3개 이상 비교

| 상황 | 표현                            |
| ---- | ------------------------------- |
| 질문 | 〜の中で何が/どれが一番〜ですか |
| 대답 | 〜が一番〜です                  |

---

## 실전 예문 모음

- 私（わたし）は犬（いぬ）**より** 猫の方が好きです。 (개보다 고양이가 더 좋아요.)
- 昨日**より** 今日の方が寒いです。 (어제보다 오늘이 더 추워요.)
- この映画は思ったより面白（おもしろ）かった。 (이 영화는 생각보다 재미있었어.)
- 世界（せかい）で一番高い山はエベレストです。 (세계에서 가장 높은 산은 에베레스트예요.)

## 한 줄 요약

| 표현         | 한국어          | 구조                     |
| ------------ | --------------- | ------------------------ |
| AよりBの方が | A보다 B가 더    | A より B の方が + 형용사 |
| AとBどちらが | A와 B 어느 쪽이 | と〜とどちらが           |
| 〜が一番     | 〜가 가장       | の中で〜が一番 + 형용사  |
