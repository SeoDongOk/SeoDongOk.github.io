て형이 "모든 응용 문법의 출발점"이라면, ない형은 **부정 표현의 출발점**이다.

ない형만 제대로 만들 수 있으면 "하면 안 돼", "해야 해", "안 해도 돼" 같은 중요한 문형을 전부 만들 수 있다.

---

## 그룹별 ない형 만들기

### 2그룹 동사 (る동사) — 가장 쉬움

어미 `る`를 `ない`로 바꾼다.

| 사전형 | ない형   | 뜻            |
| ------ | -------- | ------------- |
| 食べる | 食べない | 먹지 않다     |
| 見る   | 見ない   | 보지 않다     |
| 起きる | 起きない | 일어나지 않다 |
| 寝る   | 寝ない   | 자지 않다     |
| 教える | 教えない | 가르치지 않다 |

### 1그룹 동사 (う동사) — 어미를 あ단으로

어미의 **う단 → あ단**으로 바꾸고 `ない`를 붙인다.

| 어미 | あ단 전환 | 예                              |
| ---- | --------- | ------------------------------- |
| く   | か        | 書く → 書かない                 |
| ぐ   | が        | 泳ぐ → 泳がない                 |
| す   | さ        | 話す → 話さない                 |
| つ   | た        | 待つ → 待たない                 |
| ぬ   | な        | 死ぬ → 死なない                 |
| ぶ   | ば        | 遊ぶ → 遊ばない                 |
| む   | ま        | 飲む → 飲まない                 |
| る   | ら        | 乗る → 乗らない                 |
| う   | わ        | 買う → **買わない** ← う만 예외 |

> **う로 끝나는 동사 주의:** `買う → 買わない` (うわ, `買あない`가 아님)

### 3그룹 동사 (불규칙) — 외우기

| 사전형 | ない형 |
| ------ | ------ |
| する   | しない |
| くる   | こない |

## ない형 + よく 쓰이는 파생 문형

### 1. 〜ない + です = 정중 부정

- 今日は学校に**行かないです**。 (오늘은 학교에 가지 않아요.)
- あまり好きじゃないです → 자연스러운 회화체

### 2. 〜なくて — 부정 연결 (사실·상태)

어미 `ない → なくて`

- お金が**なくて**、買えません。 (돈이 없어서 살 수 없어요.)
- 時間が**なくて**、困っています。 (시간이 없어서 곤란해요.)

### 3. 〜ないで — 하지 않은 채로 / 하지 말고

어미 `ない → ないで`

- 朝ごはんを食べ**ないで**学校に行きました。 (아침을 먹지 않고 학교에 갔어요.)
- 心配し**ないで**ください。 (걱정하지 마세요.)

> **なくて vs ないで 차이:**
>
> - `なくて` → 이유·원인 (〜 때문에)
> - `ないで` → 동작 없이 다음 행동 / 하지 말기 요청

### 4. 〜なければならない — 해야 한다

어미 `ない → なければならない`

- 宿題をし**なければなりません**。 (숙제를 해야 합니다.)
- 早く起き**なければなりません**。 (일찍 일어나야 합니다.)

일상 회화에선 줄여서 **〜なきゃ** (나키야) 를 자주 쓴다.

- 行かなきゃ。 (가야 해.)

### 5. 〜なくてもいい — 안 해도 돼

어미 `ない → なくてもいい`

- 来**なくてもいいです**よ。 (안 와도 돼요.)
- 全部食べ**なくてもいいです**。 (다 먹지 않아도 돼요.)

## ない형 활용 총정리

| 문형               | 만들기                | 의미                 |
| ------------------ | --------------------- | -------------------- |
| 〜ない             | ない형                | 〜하지 않다          |
| 〜なくて           | ない→なくて           | 〜하지 않아서        |
| 〜ないで           | ない→ないで           | 〜하지 않고 / 마세요 |
| 〜なければならない | ない→なければならない | 해야 한다            |
| 〜なくてもいい     | ない→なくてもいい     | 안 해도 돼           |
