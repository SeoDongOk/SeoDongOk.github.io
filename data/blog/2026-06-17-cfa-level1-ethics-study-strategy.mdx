> CFA Level 1에서 Ethics는 단순 암기 과목처럼 보이지만, 실제로는 **문장 해석과 판단 기준을 반복해서 훈련해야 하는 과목**이다.

## Ethics를 먼저 봐야 하는 이유

Ethics는 Level I에서 가장 높은 비중 구간을 가진 과목이다. 하지만 더 중요한 이유는, Ethics가 CFA 시험 전체의 언어를 보여준다는 점이다.

- 행동 기준을 묻는다.
- 이해상충을 구분하게 한다.
- 고객, 고용주, 시장, 전문가 책임을 비교한다.
- “가장 적절한 행동”을 판단하게 한다.

즉, Ethics는 정답 문장을 외우는 과목이 아니라 **CFA가 기대하는 사고방식에 익숙해지는 과목**이다.

## 초반 전략: 용어와 기준 잡기

초반에는 문제를 많이 맞히려 하기보다 용어와 기준을 먼저 잡는다.

정리할 항목은 다음과 같다.

| 구분                           | 질문                              |
| ------------------------------ | --------------------------------- |
| Duty                           | 누구에게 어떤 의무가 있는가       |
| Disclosure                     | 무엇을 공개해야 하는가            |
| Loyalty                        | 고객과 고용주 중 어떤 관계인가    |
| Fair Dealing                   | 누구에게 불공정한 이익이 생기는가 |
| Misrepresentation              | 어떤 표현이 오해를 만든 것인가    |
| Material Nonpublic Information | 정보가 중요하고 비공개인가        |

Ethics는 같은 단어가 반복해서 나오기 때문에, 초반에 용어를 잡아두면 이후 문제 풀이가 빨라진다.

## 중반 전략: 사례 문제 반복

중반부터는 사례를 많이 풀어야 한다. Ethics는 “규정을 아는가”보다 “이 상황에서 어떤 행동이 적절한가”를 묻기 때문이다.

문제를 풀 때는 다음 순서로 읽는다.

1. 등장인물 확인
2. 이해관계자 확인
3. 문제가 되는 행동 표시
4. 공개 또는 동의 여부 확인
5. 가장 보수적인 행동 선택

특히 “이미 공개했는가”, “고객에게 동의를 받았는가”, “고용주에게 알렸는가” 같은 디테일이 중요하다.

## 후반 전략: 매일 짧게 유지

Ethics는 한 번 몰아서 공부하고 끝낼 과목이 아니다. 시험 직전까지 매일 조금씩 보는 것이 낫다.

나의 후반 루틴은 다음과 같다.

```text
매일 Ethics 20분
- 기준 1개 읽기
- 관련 문제 5개 풀기
- 틀린 문장 1개 기록
```

이 루틴은 부담이 작고, 시험 직전 감각 유지에 좋다.

## 오답노트 방식

Ethics 오답노트는 길게 쓰지 않는다. 대신 “내가 왜 그 선택지를 맞다고 착각했는지”를 기록한다.

```text
상황:
내 선택:
정답:
착각한 지점:
다음 판단 기준:
```

예시:

```text
상황: 애널리스트가 고객에게 리포트를 먼저 공유함
내 선택: 회사 내부 승인만 있으면 가능하다고 판단
정답: 공정한 배포 원칙 위반 가능성
착각한 지점: 내부 승인과 고객 간 공정 배포를 혼동
다음 판단 기준: 특정 고객에게 먼저 이익이 갔는가
```

## 자주 헷갈릴 포인트

| 포인트          | 체크 질문                               |
| --------------- | --------------------------------------- |
| 이해상충        | 공개만 하면 충분한가, 회피가 필요한가   |
| 추천과 보장     | 수익 가능성을 보장처럼 표현했는가       |
| 성과 표시       | 대표성 없는 성과를 제시했는가           |
| 고객 우선       | 본인 또는 회사 이익이 고객보다 앞섰는가 |
| 내부정보        | 정보가 중요하고 아직 공개되지 않았는가  |
| 독립성과 객관성 | 보상이나 접대가 판단을 흔들 수 있는가   |

Ethics에서 중요한 것은 “상식적으로 괜찮아 보이는 행동”이 아니라 CFA 기준에서 허용되는 행동인지다.

## 결론

Ethics는 암기 과목처럼 시작하지만, 마지막에는 판단 훈련 과목이 된다. 고득점을 위해서는 초반에 기준을 잡고, 중반에 사례를 많이 풀고, 후반에는 매일 짧게 유지해야 한다.

나에게 Ethics의 목표는 단순히 비중을 방어하는 것이 아니라, Level I 전체 합격 안정성을 높이는 것이다.
