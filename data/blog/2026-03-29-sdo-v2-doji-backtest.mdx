## 발단 — 왜 SOL을 못 샀나?

라즈베리파이에서 `sdo_v2` 전략을 실거래 중이다. 어제(3/28) SOLUSDT가 1시간 가까이 1% 이상 상승 추세를 유지하고 있었는데, 봇이 계속 진입하지 않았다. 로그를 보니 이유가 있었다.

```
SOLUSDT 진입 대기: 연속 구간에 보합 캔들 포함 (현황: [1, 1, 1, 0])
SOLUSDT 진입 대기: 연속 구간에 보합 캔들 포함 (현황: [1, 1, 0, 1])
SOLUSDT 진입 대기: 연속 구간에 보합 캔들 포함 (현황: [1, 0, 1, -1])
```

SDO v2는 패턴 검사 시 **4틱 연속 구간에 보합(0) 캔들이 하나라도 있으면 HOLD**를 반환한다. 약 1시간 동안 15분봉마다 보합 캔들이 끼어들어서 패턴이 계속 깨졌다.

## 두 가지 방식 정의

| 방식                   | 설명                                              |
| ---------------------- | ------------------------------------------------- |
| **보합 차단 (현재)**   | streak 구간에 보합(0) 캔들 존재 → 즉시 HOLD       |
| **보합 허용 (수정안)** | 보합(0)을 직전 캔들 방향값으로 채워서 연속성 유지 |

수정안의 핵심 로직:

```python
# allow_doji=True: 보합(0)을 직전 방향으로 채움
if self._allow_doji:
    filled = directions.copy()
    for i in range(1, len(filled)):
        if filled.iloc[i] == 0:
            filled.iloc[i] = filled.iloc[i - 1]
    directions = filled
```

`[1, 1, 1, 0]` → `[1, 1, 1, 1]` 로 처리되어 패턴 완성으로 인정.

## 분석

보합 허용 방식은 거래 횟수가 6회 더 많아졌지만, 수익률·승률·MDD 세 지표 모두 나빠졌다.

**이유를 추정하면:**

- 보합 캔들이 끼인 구간 = 추세가 불명확한 횡보 구간
- 그런 구간에서 진입 → 방향성 없이 SL에 걸리는 경우 증가
- 결과적으로 필터 역할을 잘 수행하고 있었던 것

어제 SOL 진입 못 한 건 아쉽지만, 전략 로직을 건드리지 않는 게 맞다는 결론이다.

## 결론

> **보합 캔들 차단은 의도된 노이즈 필터다. 현재 방식 유지.**

단기적으로 신호를 놓치는 것처럼 보여도, 90일 누적으로는 더 나은 리스크/리워드 비율을 만들어 준다. MDD 차이(-25.8% vs -35.3%)가 특히 의미 있다 — 레버리지 x30 환경에서 낙폭 관리는 생존과 직결된다.
