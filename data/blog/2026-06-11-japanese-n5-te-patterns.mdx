て형 자체는 EP.2에서 다뤘다. 이 글은 그 て형에 문형을 붙여서 실제로 쓰는 방법을 정리한다.

て형을 알아도 응용 문형을 모르면 회화에서 막힌다. 아래 7가지만 익혀도 N5 회화의 절반 이상을 커버할 수 있다.

---

## 1. 〜ています — 진행 / 상태

**의미:** 지금 〜하고 있어요 / (상태가) 〜해 있어요

```
동사 て형 + います
```

### 진행 (동작 진행 중)

- 今（いま）、ご飯（はん）を **食べています**。 (지금 밥을 먹고 있어요.)
- 彼（かれ）は音楽（おんがく）を **聞いています**。 (그는 음악을 듣고 있어요.)

### 상태 (결과 상태 지속)

- 窓（まど）が **開いています**。 (창문이 열려 있어요. — 누군가 열어 놓은 상태)
- 電気（でんき）が **ついています**。 (불이 켜져 있어요.)
- 結婚（けっこん）して **います**。 (결혼해 있어요. = 기혼이에요.)

> **진행 vs 상태 구분:** 「走っています」는 지금 달리는 중(진행), 「壊れています」는 고장 난 상태(상태).

---

## 2. 〜てから — 〜하고 나서

**의미:** 먼저 A를 하고, 그 다음에 B를 한다

```
동사 て형 + から + 다음 행동
```

- 手（て）を洗（あら）っ**てから**、食べてください。 (손을 씻고 나서 드세요.)
- 宿題（しゅくだい）をし**てから**、ゲームをします。 (숙제를 하고 나서 게임을 해요.)
- シャワーを浴（あ）び**てから**、寝（ね）ます。 (샤워하고 나서 잘 거예요.)

> **〜てから vs 〜て:** 단순 연결 「〜て」는 행동을 나열, 「〜てから」는 순서를 강조할 때.

---

## 3. 〜てもいいです — 〜해도 돼요

**의미:** 허가 / 〜해도 괜찮아요

```
동사 て형 + もいいです
```

- ここで写真（しゃしん）を撮（と）っ**てもいいです**か。 (여기서 사진 찍어도 돼요?)
- もう帰（かえ）っ**てもいいですよ**。 (이제 가도 돼요.)
- このペンを使（つか）っ**てもいい**ですか。 (이 펜 써도 돼요?)

**부정(안 돼요):**

- ここで食べ**てはいけません**。 (여기서 먹으면 안 돼요.) → 다음 문형 참고

---

## 4. 〜てはいけません — 〜하면 안 돼요

**의미:** 금지 / 〜하면 안 됩니다

```
동사 て형 + はいけません
```

- ここでタバコを吸（す）っ**てはいけません**。 (여기서 담배를 피우면 안 됩니다.)
  | 표현 | 공손도 |
  |------|--------|
  | 〜てはいけません | 공식 금지 |
  | 〜てはだめです | 일상 금지 |
  | 〜ないでください | "〜하지 마세요" |

- 廊下（ろうか）を走（はし）っ**てはいけません**。 (복도에서 뛰면 안 됩니다.)
- 試験（しけん）中（ちゅう）に話し**てはいけません**。 (시험 중에 말하면 안 됩니다.)

---

## 5. 〜てください — 〜해 주세요

**의미:** 부탁 / 지시

```
동사 て형 + ください
```

- ここに名前（なまえ）を書い**てください**。 (여기에 이름을 써 주세요.)
- もう少（すこ）しゆっくり話し**てください**。 (조금 더 천천히 말해 주세요.)
- ドアを閉（し）め**てください**。 (문을 닫아 주세요.)

**부드럽게 하면:**

- 〜てくださいませんか (더 공손한 부탁)
- 〜てくれる? (친구 사이 부탁)

---

## 6. 〜てみます — 〜해 볼게요

**의미:** 시도 / 한번 해보다

```
동사 て형 + みます
```

- この料理（りょうり）を食べ**てみます**。 (이 요리를 먹어 볼게요.)
- 日本語（にほんご）で話し**てみましょう**。 (일본어로 말해 봅시다.)
- 一度（いちど）着（き）**てみてください**。 (한번 입어 보세요.)

---

## 7. 〜たいです — 〜하고 싶어요

**의미:** 희망 / 욕구 표현

> 엄밀히는 ます형 어간에 붙지만, て형과 함께 배우는 게 흐름상 자연스럽다.

```
동사 ます형 어간 + たいです
```

| 동사   | ます형 어간 | 〜たい형 |
| ------ | ----------- | -------- |
| 食べる | 食べ        | 食べたい |
| 行く   | 行き        | 行きたい |
| 見る   | 見          | 見たい   |
| 飲む   | 飲み        | 飲みたい |

- 日本（にほん）に行き**たい**です。 (일본에 가고 싶어요.)
- 映画（えいが）を見**たい**です。 (영화를 보고 싶어요.)
- 何（なに）が食べ**たい**ですか。 (뭐가 먹고 싶어요?)

**부정(〜하고 싶지 않아요):**

- 行き**たくない**です。 (가고 싶지 않아요.) — い형용사 부정 규칙 그대로

---

## 문형 7가지 빠른 요약

| 문형           | 의미               | 예               |
| -------------- | ------------------ | ---------------- |
| 〜ています     | 진행 / 상태 지속   | 食べています     |
| 〜てから       | 〜하고 나서        | 洗ってから食べる |
| 〜てもいい     | 허가 (~해도 돼)    | 帰ってもいいです |
| 〜てはいけない | 금지 (~하면 안 돼) | 走ってはいけない |
| 〜てください   | 부탁 (~해 주세요)  | 書いてください   |
| 〜てみる       | 시도 (~해 보다)    | 食べてみます     |
| 〜たい         | 희망 (~하고 싶다)  | 行きたいです     |

て형 하나에서 이 7가지가 모두 나온다. て형을 확실히 만들 수 있으면 이 문형들은 자연스럽게 따라온다.
