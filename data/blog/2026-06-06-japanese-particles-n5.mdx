일본어 문장의 뼈대를 잡아주는 것이 조사(助詞, じょし)다.  
단어 뒤에 붙어서 그 단어가 문장에서 무슨 역할을 하는지 정해주는 작은 글자로, 한국어의 "은/는, 이/가, 을/를, 에, 에서" 같은 토씨와 같다.

한국어에 이미 조사가 있기 때문에 한국인 학습자에게는 비교적 유리한 부분이다. 다만 `は`와 `が`, `に`와 `で`처럼 비슷해 보이는 조사를 구분하는 것이 관건이다. 이 글에서는 N5 핵심 조사 15가지를 용법과 예문 중심으로 정리한다.

## 조사가 중요한 이유

일본어는 어순이 비교적 자유롭다. 대신 조사가 각 단어의 역할을 표시한다.

- 私（わたし）**は** 田中（たなか）さん**に** 本（ほん）**を** あげました。
- 田中さん**に** 私**は** 本**を** あげました。

순서를 바꿔도 `は`(주제), `に`(대상), `を`(목적어)의 역할이 그대로이기 때문에 의미가 흔들리지 않는다. 조사를 정확히 쓰는 것이 곧 문장 정확도로 이어진다.

## N5 핵심 조사 15가지

### ① は (wa) — 주제 "~은/는"

문장의 주제(화제)를 제시한다. 조사로 쓸 때는 "하"가 아니라 **"와(wa)"**로 읽는다.

- 私（わたし）**は** 韓国人（かんこくじん）です。 (저는 한국인입니다.)
- コーヒー**は** 好（す）きですが、お茶（ちゃ）**は** 好きじゃないです。 (커피는 좋아하는데, 차는 안 좋아해요. — 대비)

### ② が (ga) — 주격 "~이/가"

동작·상태의 주어를 나타낸다. 새로 등장하는 정보나 존재, 자연현상에 자주 쓴다.

- 猫（ねこ）**が** います。 (고양이가 있어요.)
- 雨（あめ）**が** 降（ふ）っています。 (비가 내리고 있어요.)
- 私は 日本語（にほんご）**が** 好（す）きです。 (저는 일본어를 좋아해요. — 好き 앞은 を가 아니라 が)

### ③ を (o) — 목적격 "~을/를"

타동사의 목적어를 표시한다. 이동·통과하는 장소에도 쓴다.

- パン**を** 食（た）べます。 (빵을 먹어요.)
- 公園（こうえん）**を** 散歩（さんぽ）します。 (공원을 산책해요. — 통과 장소)

### ④ に — 시간·도착점·존재·대상

N5에서 가장 다재다능한 조사다. "콕 찍는 지점"으로 기억하면 편하다.

| 용법              | 예               |
| ----------------- | ---------------- |
| 시간(정확한 시각) | 7時**に**起きる  |
| 도착점            | 学校**に**行く   |
| 존재 장소         | 部屋**に**いる   |
| 대상(~에게)       | 友達**に**あげる |

- 毎朝（まいあさ）6時**に** 起（お）きます。 (매일 아침 6시에 일어나요.)
- 机（つくえ）の上（うえ）**に** 本があります。 (책상 위에 책이 있어요.)

> 「今日・明日・毎日」처럼 상대적 시간에는 `に`를 붙이지 않는다. (❌ 今日に → ⭕ 今日)

### ⑤ で — 장소·수단·범위·이유

`に`가 "지점"이라면 `で`는 **"동작이 일어나는 무대"**가 핵심이다.

| 용법      | 예                     |
| --------- | ---------------------- |
| 행동 장소 | 家**で**勉強する       |
| 수단·도구 | バス**で**行く         |
| 범위·한정 | クラス**で**一番       |
| 이유      | 風邪（かぜ）**で**休む |

- 図書館（としょかん）**で** 勉強します。 (도서관에서 공부해요.)
- 電車（でんしゃ）**で** 会社（かいしゃ）に行きます。 (전철로 회사에 가요.)

### ⑥ へ (e) — 방향 "~로"

이동의 방향을 나타낸다. 조사로 쓸 때는 "헤"가 아니라 **"에(e)"**로 읽는다. N5 단계에서는 `に`와 거의 바꿔 써도 된다.

- 友達（ともだち）の家（いえ）**へ** 行きます。 (친구 집으로 가요.)

### ⑦ と — 열거·동반·인용

- りんご**と** バナナを買（か）いました。 (사과와 바나나를 샀어요. — 열거)
- 先生（せんせい）**と** 話（はな）しました。 (선생님과 이야기했어요. — 동반)
- 田中さん**と** 言います。 (다나카라고 합니다. — 인용)

### ⑧ も — "~도"

추가·동조를 나타낸다. `は`·`が`·`を` 자리를 대신 차지한다.

- 私**も** 学生（がくせい）です。 (저도 학생이에요.)
- 1時間（じかん）**も** 待（ま）ちました。 (한 시간이나 기다렸어요. — 수량 강조)

> `私はも`(❌) → `私も`(⭕). `は`·`が`·`を`는 `も`와 함께 쓸 수 없다.

### ⑨ の — 소유·수식·주격대용

명사와 명사를 잇는 접착제 역할을 한다.

- これは 私（わたし）**の** かばんです。 (이건 제 가방이에요.)
- 日本語**の** 本を 読みます。 (일본어 책을 읽어요.)
- 母（はは）**の** 作（つく）った ケーキ。 (엄마가 만든 케이크. — が 대용)

### ⑩ から — 시작점·이유

- 授業（じゅぎょう）は 9時**から** です。 (수업은 9시부터예요.)
- 今日は 忙（いそが）しい**から**、行きません。 (오늘은 바쁘니까 안 가요. — 이유)

### ⑪ まで — 도착점·한계

`から`와 짝을 이룬다.

- 9時から 5時**まで** 働（はたら）きます。 (9시부터 5시까지 일해요.)

### ⑫ や — 예시 열거

여러 개 중 대표 몇 개만 예로 들 때 쓴다. 흔히 `〜や〜など` 형태로 쓴다.

- 机の上に 本**や** ノート**など**があります。 (책상 위에 책이랑 노트 등이 있어요.)

### ⑬ か — 의문·선택

- お元気（げんき）です**か**。 (잘 지내세요?)
- コーヒー**か** 紅茶（こうちゃ）、どちらにしますか。 (커피 아니면 홍차, 어느 걸로 할래요?)

### ⑭ ね — 확인·공감

문장 끝에 붙어 상대의 동의를 구하거나 공감할 때 쓴다.

- いい天気（てんき）です**ね**。 (날씨 좋네요.)

### ⑮ よ — 알림·강조

상대가 모르는 새 정보를 알려주거나 의견을 강조할 때 쓴다.

- その店（みせ）、おいしいです**よ**。 (그 가게 맛있어요!)

> `ね`는 "우리 둘 다 알죠?"(공감), `よ`는 "당신이 모르는 걸 알려줄게요"(정보 전달). 합친 `〜よね`(~죠?)도 자주 쓴다.

## 헷갈리는 조사 비교

### は vs が

|      | は (주제)    | が (주격)  |
| ---- | ------------ | ---------- |
| 역할 | 화제 제시    | 주어 표시  |
| 정보 | 이미 아는 것 | 새로 등장  |
| 강조 | 뒤쪽         | 앞쪽(주어) |

의문사(誰·何)가 주어이면 답도 `が`로 받는다.

- Q: 誰**が** 来ましたか。 (누가 왔어요?)
- A: 田中さん**が** 来ました。 (다나카씨가 왔어요.)

### に vs で

- `に` = 존재·도착하는 지점 (있다)
- `で` = 동작이 벌어지는 무대 (한다)

| 문장                | 조사      |
| ------------------- | --------- |
| 部屋**に** いる     | に (존재) |
| 部屋**で** 勉強する | で (동작) |

### を vs が

`好き`·`嫌い`·`上手`·`下手`·`できる` 앞은 `を`가 아니라 `が`를 쓴다.

- ❌ 日本語**を** 好きです → ⭕ 日本語**が** 好きです

### と vs や

- `と` = 나열한 것이 전부 (사과와 바나나, 둘뿐)
- `や` = 예시일 뿐 더 있음 (사과·바나나 등)

## 한국인이 자주 틀리는 포인트

1. **は를 "하"로 읽기** — 조사 `は`는 wa, `へ`는 e로 읽는다.
2. **〜を好き** — 좋아함·능력 대상은 `が`. (映画**が**好き)
3. **시간에 무조건 に 붙이기** — 今日·明日·毎日에는 붙이지 않는다.
4. **友達と vs 友達に** — `と`는 함께(쌍방), `に`는 ~에게(일방향).
5. **も를 は/が/を와 같이 쓰기** — `私はも`(❌) → `私も`(⭕).

## 마무리

조사는 양이 많아 보이지만, 용법을 따로 외우기보다 「7時**に**起きて、電車**で**会社**に**行く」 같은 통문장을 통째로 익히는 편이 빠르다. 하루에 예문 몇 개씩 소리 내어 읽으면 자연스럽게 감각이 붙는다.

핵심만 한 줄로 정리하면 이렇다.

> は는 주제, が는 주어 / に는 지점, で는 무대 / 好き 앞엔 が
